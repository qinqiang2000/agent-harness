# ELK 日志搜索指南

根据问题场景选择对应的关键字模板。**必需信息缺失时，必须用 `AskUserQuestion` 反问用户，不得跳过。**

---

## A. 发票云公有云（ELK）

| 问题场景 | 必需信息（缺失时反问） | ELK 关键字模板 |
|---|---|---|
| 查询开票结果（已知流水号） | 流水号 | `"流水号" AND "查询发票"` |
| 查询开票请求参数（未知流水号） | 税号 | `"税号" AND "账号" AND "第三方" AND "/fpdk/etax/bill/invoice/create"` |
| 查询开票请求参数（已知发票号码） | 发票号码 | 先查：`"发票号码" AND "serialNo"`，再查：`"流水号" AND "第三方" AND "/fpdk/etax/bill/invoice/create"` |
| 查询发票交付邮件状态 | 发票号码 | `"发票号码" AND "阿里邮箱发送参数"` |
| 查询报销单保存接口 | bxd_key | `"bxd_key" AND "EAS调用保存报销单"` |
| 查询附件入库记录 | bxd_key | `"附件全量入库并清空缓存" AND "bxd_key"` |
| 查询合规性校验方案 | 无 | `"com.kingdee.service.expense.ExpenseVerifyService - config"` |
| 查询电子凭证解析结果 | 发票号码 | `"发票号码" AND "xbrl解析"` |
| 账号登录认证记录 | 税号、账号 | `"账号" AND "登录记录上传"` |
| 查询登录成功记录 | 税号、账号 | `"税号" AND "账号" AND "第三方" AND "/newEra/issueLogin" AND "成功"` |
| 短信验证码问题 | 税号 | `"税号" AND "短信验证码"` |

## B. 星瀚/星空旗舰版（天梯/monitor）

> 这类问题需先在 ERP 系统日志中获取 traceId，再到 ELK 查详情。
> 提示用户：进入 ERP → 相关模块日志 → 找到对应记录 → 复制 traceId。

| 问题场景 | 必需信息 | ELK 关键字模板 |
|---|---|---|
| 开票申请单推送 | 开票申请单编号 | `"开票申请单编号" AND "openApi解密后数据"` |
| 开票结果 | 发票流水号 | `"发票流水号"` |
| AWS 识别结果 | 无 | `"aws识别接口返回结果"` |
| PDF 解析结果 | 无 | `"pdf解析结果耗时"` |
| 报销单接收发票 | bxd_key | `"bxd_key" AND "接收发票"` |

---

## C. 星空企业版/EAS

> 需先在 ERP 日志系统（财务会计 → 发票管理 → 金税连接设置 → 请求日志）获取 traceId，再到 ELK 查。

| 问题场景 | 必需信息 | 查询步骤 |
|---|---|---|
| 开票失败 | 报错信息 | ERP 日志筛选"请求返回" → 获取 traceId → ELK 查询 |
| 收票单无数据 | 发票号码 | ELK：`"发票号码" AND "insert"` 或 `"发票号码" AND "新版保存发票返回"` |
| 接收发票下载失败 | 任务批次号 | ELK：`"任务批次号" AND "单收票消息处理完成"` |
| 获取 bxd_key | 无（需远程操作） | F12 → Network → upload 接口返回值 |

---

## 使用规则

1. **匹配场景**：根据用户描述的问题，从上表找到最匹配的场景
2. **检查必需信息**：
   - 已知 → 将信息填入模板，直接执行 ELK 查询
   - 缺失 → 用 `AskUserQuestion` 反问，例如：
     - "请提供税号，以便查询短信验证码记录"
     - "请提供 bxd_key，以便查询报销单附件入库记录"
     - "请提供发票号码或流水号，以便定位开票请求"
3. **B/C 类系统**：若用户未提供 traceId，先引导用户去 ERP 系统获取 traceId，再执行 ELK 查询
