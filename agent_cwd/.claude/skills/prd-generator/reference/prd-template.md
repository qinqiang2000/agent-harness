## 模板使用指南

### 何时使用完整模板

- 大型功能开发
- 涉及多个团队协作
- 需要详细的技术评审
- 高优先级（P0/P1）需求

### 何时使用简化模板

- 小型功能优化
- 单一团队开发
- 快速迭代需求
- 低优先级（P2/P3）需求

### 模板定制

根据项目特点，可以：
- 增加特定章节（如国际化需求、合规要求）
- 删除不适用章节
- 调整章节顺序
- 修改表格结构
