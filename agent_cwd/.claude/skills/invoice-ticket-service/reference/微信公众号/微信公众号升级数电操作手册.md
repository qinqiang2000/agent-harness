## 公众号申请升级数电
初次使用需要关注发票云公众号（若已关注，可忽略此步骤），操作步骤如下：

①扫描下方服务号二维码或在微信搜索【金蝶发票云】公众号，关注公众号。

![](https://cdn.nlark.com/yuque/0/2022/png/12881570/1671076535497-0a8fb2fe-b9ec-45f1-90d8-3ab53ae0b938.png?x-oss-process=image%2Fformat%2Cwebp)

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685586542805-c46d0c65-95de-463c-85eb-a013a7639a4a.png)

②点击【我的服务】—【数电升级】。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1687772091222-c8143e39-4680-4cab-88be-18e594a3761e.png)

③点击搜索相关企业名称，完成后点击下一步。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685587188612-55bb8199-9f08-4f75-b159-717cd4806c53.png)



![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685587188579-99d8e39f-b563-45c1-ad73-a54ce4f486b2.png)

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685587188613-97d02c10-0f74-44b3-81c4-d337136fb610.png)

④选择企业下需要升级的公司，点击下一步。

 ![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685587189565-5da27b22-0a1e-4e5d-b63e-6078295ee3e4.png)

⑤输入个人相关信息：收邮件地址（用于接收含升级数电必备内容的邮件）、选择使用的产品、姓名及联系方式。输入并确认无误后点击提交申请即可。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685587189551-7e5d7789-dee5-49b1-adff-225f27b058e0.png)

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685590209220-b6b47154-5cf8-4d6b-b9f2-6e843f55609b.png)

## 邮箱接收邮件，获取数电配置信息
在公众号申请成功后，申请升级时填写的邮箱地址会收到数电升级邮件。

①在公众号申请成功后，进入邮箱找到相关邮件。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685590883415-893f0656-f3a4-47e7-bd74-32d2b9b32805.png)

②进入邮件，复制“操作密钥”，点击“操作链接”进入下一界面。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685590895766-c4155ac1-78a0-44f7-ba82-9ec804193533.png)

③粘贴邮件中的“操作密钥”。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685590958964-5d1bdf13-140d-427b-abd4-8ce3c129dd4f.png)

⑤提交成功后，使用微信扫描二维码，进入发票云公众号。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685590986536-2f900f5f-2ca3-4e73-809d-ebb649fb2aae.png)

## 公众号添加企业账号
### 添加电子税局账号
①扫码后，公众号中有如下消息弹窗，点击进入。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1684979174145-4560b85e-11d8-4c7c-9a80-3de40c85b69e.png?x-oss-process=image%2Fresize%2Cw_260%2Climit_0)

②点击进入待配置企业列表后，选择需要配置的企业。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685599651937-9584e26d-ea6d-443e-a307-675a19ef7bf2.png)

③点击新增账号。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685599709162-5951d296-b46e-4c10-b739-d3229b3061b5.png)

④进入新增页面，核对地区与税号无误之后，输入用户、密码、角色、登陆类型（默认选择人工登录），点击登录按钮。

登录类型详细说明：

人工登录：<font style="color:rgba(0, 0, 0, 0.85) !important;"></font>需手动验证的常规账号默认选择人工登录，输入用户名密码并点击“登录”后，需要继续完成后续获取验证码等配置，才可新增账号成功。

华盟托管号码：仅限账号是“华盟托管号码”类的账号使用，输入用户名密码并点击“登录”后，等待系统自动处理完成，即可新增账号成功。

~~【已下架】发票云登录App：适用于账号形式是手机号的账号，需要根据文档验证码登录APP使用说明下载APP并进行初始化配置后才可使用，否则无法添加账号。~~

企享云办税小号：仅限账号是“企享云办税小号”类的账号使用，输入用户名密码并点击“登录”后，等待系统自动处理完成，账号即可新增成功。使用此登录类型的，配置详见：[企享云办税小号使用说明](https://jdpiaozone.yuque.com/nbklz3/dn5ehb/qwtvytgyhe2236dm)。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1765434835325-04d19e04-a6a2-488f-ad28-baac440b31ac.png?x-oss-process=image%2Fformat%2Cwebp)

⑤点击“登录”之后，输入正确的验证码，点击“登录”。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1765435810140-8dc61349-39a6-4d5d-824b-3baa327f42ec.png?x-oss-process=image%2Fformat%2Cwebp)
