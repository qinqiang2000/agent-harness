# 功能一：更换地址头
环境部署后，初始访问地址为：http://ip:port/；客户实际需要使用域名或公网地址、负载均衡地址访问，则需要对地址头进行修改，可使用工具实现，操作步骤如下：

```shell
# 切换到工具所在目录
cd /opt/image-tools/
# 执行脚本
sh image-tools.sh
# 根据提示输入信息
#选择功能，输入0；再输入新的地址头信息
```

![](https://cdn.nlark.com/yuque/0/2025/png/21778712/1760063484883-fbbd31a2-6ee8-4be5-9435-360edff4491b.png)

# 功能二：更换zk连接信息
Zookeeper服务重装或其他原因导致需要更改zk连接信息的，可使用运维工具修改连接地址。

操作步骤如下：

```shell
# 切换到工具所在目录
cd /opt/image-tools
# 执行脚本
sh image-tools.sh
# 根据提示输入信息
#选择功能，输入1；再根据提示输入新的zk连接信息
```

![](https://cdn.nlark.com/yuque/0/2025/png/21778712/1760063484895-8bc60a40-6ebf-47a4-98ac-9d04e68edf33.png)

# 功能三：替换许可
产品许可存在有效期，如用户许可到期续费后，需要更换许可，可使用本工具进行替换。

操作如下：

```shell
# 切换到工具所在目录
cd /opt/image-tools
# 上传新的许可到工具所在目录，并修改文件名为： license.lic
# 执行脚本
sh image-tools.sh
# 根据提示输入信息
# 选择功能，输入2；
```

![](https://cdn.nlark.com/yuque/0/2025/png/21778712/1760063484789-2dadf8d2-c45b-4bc9-ab4c-eaf095606902.png)

# 功能四：私包更新
操作步骤：

```shell
# 切换到工具所在目录
cd /opt/image-tools
# 将私包上传到服务器上
# 存放目录为：/opt/image-tools/services
# 执行脚本
sh image-tools.sh
# 根据提示输入信息
# 选择功能，输入3；
```

# 功能五：日志打包
如需提取后台日志，可使用本工具将所需日志打包后，提取

操作步骤：

```shell
# 切换到工具所在目录
cd /opt/image-tools
# 执行脚本
sh image-tools.sh
# 根据提示输入信息
# 选择功能，输入4；
# 输入日志时间：格式 YYYY-MM-DD
# 选择具体服务
# 日志打包后，默认存放在/opt/image-tools目录下，可使用ftp工具进行下载
```
