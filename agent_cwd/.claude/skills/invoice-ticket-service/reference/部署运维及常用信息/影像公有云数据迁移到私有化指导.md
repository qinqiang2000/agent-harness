# 一、背景
有客户原先使用的是公有云影像，后需要将影像**系统数据迁移**到私有化环境。

# 二、确保前提
1. 私有化影像系统环境已经安装（[安装指导](https://www.yuque.com/piaozone/implement/cx7fs10dhe2w0vka)）且数据存储有足够的空间（大于该租户在公有云已使用的存储空间）
2. 私有化影像系统许可已经导入
3. 迁移前必须确保私有化影像系统版本和公有云一致（[升级指导](https://www.yuque.com/piaozone/implement/ob8smaa77gevhvg8)）
4. 私有化影像系统的所有app应用机器需要通公网的：api.piaozone.com 443 （测试环境：api-dev.piaozone.com 443）

# 三、迁移配置
## 添加zk配置
通过zkui添加配置：**配置路径api-archive-scan/pri**

登录zkui导入或者手工添加如下配置（_zkui地址：登录地址头/zkui/  默认用户名密码：admin/Kd@ArchiveZkui2023_）：

dataSyncTaskPubImgsysStatus=2

dataSyncTaskPubImgsysUrl=https://api.piaozone.com/

dataSyncTaskPubImgsysClientId=公有云clientid（_可以从激活邮件中获取_）

dataSyncTaskPubImgsysClientSecret=公有云clientSecret（_可以从激活邮件中获取_）

+ xxl_job地址：登录地址头/archive-job-admin/toLogin  
+ 默认用户名：admin
+ 密码：123456

## 新建同步任务。
用管理员账号登录影像系统：

_影像地址：登录地址头/archive-job-admin/imgsys-web/login?domain=kingdee_

_默认管理员用户名：注册许可手机号_

忘记手机号可以数据库查询：<font style="color:#117CEE;">select fdomain from tenant_cms.t_user limit 2</font>

    1. 选择菜单下载管理->下拉框点击异构影响系统文件同步列表->选择手工创建同步任务

![](https://cdn.nlark.com/yuque/0/2025/png/21778712/1737539231279-f7dd7305-02be-4b5c-a83a-713f1844b4a4.png)

    2. 选择同步时间区间->点击确定

![](https://cdn.nlark.com/yuque/0/2025/png/21778712/1737539372994-d7d201e6-db3d-4a34-9b70-84db1048ccc7.png)

    3. 等待任务执行。当同步单据数和同步成功单据数的数量一样表示那个月数据同步成功，如过程有失败需要查看日志定位原因后重新同步。

![](https://cdn.nlark.com/yuque/0/2025/png/21778712/1737539549032-4e20288b-e6a5-43c5-b278-e55a3eeb726c.png)
