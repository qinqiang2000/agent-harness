# 2、角色管理
角色管理分通用角色及业务角色，详细说明见下述链接。

**<font style="color:rgb(26, 26, 26);">通用角色：</font>**<font style="color:rgb(26, 26, 26);">企业按照业务职能定义通用角色，对多个实体表单的功能权限、字段权限、数据规则进行打包，为具体相同业务职能的用户批量授权，授权过程和后续的权限调整都非常便捷。</font>

**<font style="color:rgb(26, 26, 26);">业务角色：</font>**<font style="color:rgb(26, 26, 26);">业务角色可以继承通用角色，增加控制的隔离维度范围，如组织范围、渠道范围，即用户分配业务角色后就在隔离维度范围内拥有相同的权限。</font>
