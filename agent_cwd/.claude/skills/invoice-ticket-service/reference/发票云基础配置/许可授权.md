## 许可授权操作路径及所属模块编码
操作路径：【发票服务】→【基础资料】→【许可授权】

所属模块编码：F17030

所属模块名称：许可授权

## 许可授权功能设计理念概述
本功能是对已在【企业管理】页面引入成功的发票服务组织，进行星空发票服务的许可授权，需要根据实际使用情况自行配置。

旧版：除在本功能完成星空许可授权外，还需通过激活邮件的链接操作权益分配并成功激活（激活邮件见下图），才能成功使用发票服务。

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1723721379344-8e5a5522-91bc-4962-a91e-1cfb370225c9.png?x-oss-process=image%2Fformat%2Cwebp)

新版（公有云已为新版）：无需通过激活邮件的链接操作权益分配并激活，仅需在本功能完成星空许可授权，并将星空许可授权情况同步至激活邮件（即发票服务客户权益查询平台），成功同步方可使用发票服务。授权成功后支持取消授权，但<font style="background-color:rgb(255,255,255);">取消授权后60天内不</font><font style="background-color:rgb(255,255,255);">允许对</font><font style="color:rgb(0,0,0);background-color:rgb(255,255,255);">该税号</font><font style="background-color:rgb(255,255,255);">重新授权</font>。

### 许可授权主要操作说明
##### 维护产品序列号
维护发票服务产品序列号，用于后续将星空许可授权信息同步至发票服务客户权益查询平台（即原激活邮件）。

产品序列号允许为空，系统会根据根据所用许可类型及环境判断是否需维护产品序列号，具体规则如下表所示，若无法明确许可类型或环境，建议直接维护产品序列号。

| 许可及环境类型<br/>（若无法明确，直接填写产品序列号即可） | | 是否需维护产品序列号 |
| :--- | --- | :--- |
| 正式许可 | 非IDC | 否 |
| | IDC | 是 |
| 临时许可 | 全部 | 是 |
| 无许可文件 | 全部 | 是 |


**步骤1**：打开维护序列号界面，填写开票管理以及收票管理的产品序列号，点击确定即可。  
![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751353072859-888ad7c0-75b1-48ab-bf24-613749a790f5.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751353176611-c3deb81f-8dcd-4f99-a87e-b6fd6f62dbc1.png)

##### 许可授权
**步骤1**：选择需要分配的服务。在左侧列表中可分别对开票服务、收票服务进行授权。

其中许可括号内的（A/AA）：A为已授权数，AA为总授权数。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751354742037-5d03db85-17bf-4fca-84d4-ccd2b5032803.png)

**步骤2：**选择需要进行授权的企业。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751354755617-bd00d361-4038-42f3-b92e-79410046d759.png)  
**步骤3：**点击许可授权按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751354777593-30aca1f8-329f-49dc-8bac-b7667dcbb06d.png)

**步骤4：**确定税号无误后，点击确认，完成许可授权。

旧版：此功能仅用于对企业进行星空许可授权。<font style="color:#DF2A3F;">授权后将无法修改纳税人识别号且无法撤销， 请确保税号信息的准确性。如果在发票服务中税号配置错误，请在基础资料-企业管理进行修改。</font>

新版：此功能用于对企业进行星空许可授权，同时将星空许可授权情况同步至发票服务客户权益查询平台（即原激活邮件）。<font style="color:#DF2A3F;">虽此版本允许取消授权，</font><font style="color:#DF2A3F;background-color:rgb(255,255,255);">但同一税号取消授权后60天内不允许重新授权，取消授权期间该税号将无法进行开票/收票业务。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751354439823-d9bd2705-f0d3-456d-9564-c7da83ddf575.png)

##### 取消授权
**步骤1**：选择需要取消授权的服务。在左侧列表中可分别选择开票服务、收票服务。

其中许可括号内的（A/AA）：A为已授权数，AA为总授权数。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751354742037-5d03db85-17bf-4fca-84d4-ccd2b5032803.png)

**步骤2：**选择需要进行取消授权的企业。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751355846382-1d768833-dcce-43ae-82b3-ed083c1e5f38.png)  
**步骤3：**点击取消授权按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751355862581-5439821d-e1b9-4b9d-9f4c-f128fdb71bc8.png)

**步骤4：**确定需要进行取消授权后，点击确认，完成取消授权。

旧版：不支持取消授权。

新版：此功能用于取消星空许可授权，同时将取消的星空许可同步至发票服务客户权益查询平台（即原激活邮件）。<font style="color:#DF2A3F;">虽此版本允许取消授权，</font><font style="color:#DF2A3F;background-color:rgb(255,255,255);">但同一税号取消授权后60天内不允许重新授权，取消授权期间该税号将无法进行开票/收票业务。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751355903666-35a32bde-168f-413a-a8b6-2da43a22744d.png)

##### 同步许可授权/同步取消授权
操作许可授权/取消授权时，系统会自动将星空许可授权信息同步至发票服务客户权益查询平台（即原激活邮件），但仍存在同步失败情况，例如<font style="background-color:rgb(255,255,255);">网络问题导致同步失败、临时许可没有配置序列号导致同步失败等，此时可以用此功能手工触发同步。</font>

**<font style="background-color:rgb(255,255,255);">步骤1</font>**<font style="background-color:rgb(255,255,255);">：选择需要进行同步的服务。在左侧列表中可分别选择开票服务、收票服务。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751354742037-5d03db85-17bf-4fca-84d4-ccd2b5032803.png?x-oss-process=image%2Fformat%2Cwebp)

**步骤2：**选择需要进行同步的企业。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751355846382-1d768833-dcce-43ae-82b3-ed083c1e5f38.png)  
**步骤3：**点击同步许可授权/同步取消授权按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1751357113065-dc56977c-b267-443a-a704-967c89d4c13a.png)
