+ 所属模块名称：系统配置
+ 页面编码：bds_bill_sys_config

## 业务系统配置功能设计理念概述
业务系统配置是在发票云影像系统与各类业务系统对接的初始化配置之一。

1. **<font style="color:rgb(0, 0, 0) !important;">降低技术对接成本</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">提供测试工具验证接口连通性，避免 “配置完成但功能不可用” 的风险。</font>
2. **<font style="color:rgb(0, 0, 0) !important;">提升数据同步效率</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">通过第三方插件实现影像状态实时回调（如 “影像就绪” 通知），业务系统可即时触发下一步流程。</font>
3. **<font style="color:rgb(0, 0, 0) !important;">增强系统扩展性</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">预留 “自定义业务系统” 配置入口，支持通过插件市场快速获取新系统对接模板，无需定制开发。</font>

## 业务系统配置功能详解
### 业务系统配置关键字段说明
| **<font style="color:rgb(26,26,26);">字段名称</font>** | **<font style="color:rgb(26,26,26);">业务含义</font>** | **<font style="color:rgb(26,26,26);">字段规则说明</font>** | **<font style="color:rgb(26,26,26);">是否必填</font>** | **<font style="color:rgb(26,26,26);">其他说明</font>** |

| 业务系统 | 单据和发票上传提交的系统以及接收影像系统就绪通知 | 可选值：<br/>星瀚：不用额外配置其他信息<br/>EAS：需配置 IP 地址、端口、数据中心等信息<br/>第三方：需配置第三方自定义回调插件<br/>K2：需配置业务系统地址 | 是 |  |

### 业务系统配置主要操作说明
##### 新增并维护业务系统配置信息  
1. 打开业务系统配置窗口

在业务系统配置列表页面点击【新增】，打开业务系统配置弹窗，配置业务系统相关信息。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1747378493805-f1fcfacc-ac5a-4ef1-a5ff-77ca37a2c772.png)

2. 按业务系统类型配置明细  
+ 当业务系统为“星瀚”时：
    - 无需额外配置其他信息；选择对应的收票系统后，将状态设置为启用即可。
+ 当业务系统为“EAS”时：以下信息均需配置：

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1747378976102-e17fcd66-e829-425f-a713-76489823d843.png)

    - 需配置 IP 地址、端口、数据中心id等业务系统信息；
    - 需额外配置第三方自定义回调插件：
        * 实现接口：kd.imc.bds.common.service.ImageCallBackErpService
        * 接口常量：kd.imc.bds.common.constant.ImageCallBackErpConstant

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1747379366581-2bceb35e-eff9-4436-9127-0b08707e0049.png)

+ 当业务系统为“K2”时：
    - 需额外配置业务系统地址和编码示例；
        * 业务系统地址：用于接收影像就绪通知的业务系统地址；
        * 编码示例：从封面编码规则列表复制，多个编码示例以英文分号分隔。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1747379410923-54ee69f5-3b91-4316-8be7-4aafa49036f9.png)
