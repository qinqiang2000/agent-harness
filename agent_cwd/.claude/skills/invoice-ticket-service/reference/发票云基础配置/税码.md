## 税码功能设计理念概述
### 功能定位
“税码”用于集中维护各国家/地区在开票过程中使用的 Tax Category（税码主数据）。  
通过在此页面统一配置税制、国家/地区、分类编码、分类名称、税率及标识信息，为后续开具全球电子发票时的税码选择与税率回填提供标准来源，避免手工维护和不合规使用。  
主要使用角色包括：系统管理员（Admin）、税务管理员/合规专员（Tax Admin / Tax Specialist）、实施顾问。  

### 设计理念与使用价值
+ **标准化与一致性**：通过标准字段（税制、国家、编码、名称、标识、税率等）记录各国税码，杜绝手工维护导致的不一致。
+ **减少人工错误**：提供必填校验、范围与税率逻辑检查、日期合法性检查等规则，避免录入不合规数据。
+ **自动化协同**： 支持统一维护“分类编码 + 分类名称 + 标识 + 税率”，为发票行税码选择组件提供标准化数据来源，提高开票效率和准确率。  
+ **审计可追溯**： 通过列表展示“创建时间”等审计信息，便于后续追溯变更记录。

## 税码功能详解
### 税码关键字段说明
| **字段名称** | **业务含义** | **字段规则说明** | **其他说明** |
| --- | --- | --- | --- |
| 税码 | 税码唯一编码，用于在系统内标识与引用该 Tax Category。 | 一般由字母、数字和下划线组成，如 NL_VAT_Z_0、CN_VAT_13、TX 等；编码规则可由实施/税务团队约定。 | 与“新建税码”表单中的“税码”字段保持一致。 |
| 税制和国家/地区 | 展示该税码所属的税制与国家/地区组合，例如 VAT / NL、VAT / CN、VAT / US。 | 由表单中的“税制代码”和“国家/地区”组合生成；列表中仅展示，不可直接编辑。 | 用于快速识别税码适用的税制和区域。 |
| 分类编码 | 税码分类的短编码。用于为税码指定一个简短的分类代码，用来区分不同类型或层级的税收分类。 |  |  |
| 分类名称 |  税码分类的可读名称，例如 “Standard rate”“Zero rated goods”。   |  |  |
| 税率 | 税码对应的税率。 | 不同“标识”值对税率填写有额外校验。例如标识为STANDARD RATE / REDUCED RATE 时必须填写正数税率；ZERO RATE 标识下应填写 0。  |  |
| 标识 | 税种类型标签 | 如标准税率、减征税率、零税率、免税、不在征税范围。 |  |
| 创建时间 | 此条税码在系统中的创建时间。 | 系统自动生成，不可编辑；用于审计和追溯。 |  |

### 税码主要操作说明
#### 新建税码
**操作目标**：为某一税制和国家/地区新增一条可用于开票的税码（Tax Category）。

**操作步骤：**

1. 在列表上方点击绿色按钮 **「新建税码」**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763360830501-cc29cda8-6202-46f0-9ce3-cc49e7de6774.png)

2. 在右侧弹出的“新建税码”表单中，依次填写：
    1. 在 **「税制代码」** 下拉框中选择目标税制（如 增值税 / 商品及服务税 等）。
    2. 在 **「国家/地区」** 下拉框中选择该税码适用的国家/地区。
    3. 在 **「税码」** 输入框中输入税码编码（例如 `NL_VAT_Z_0`），确保符合企业约定的编码规范。
    4. 在 **「分类编码」** 输入框中输入短编码（如 `SR`、`ZR` 等）。
    5. 在 **「分类名称」** 输入该税码的名称。
    6. 根据需要在 **「分类描述」** 中补充详细说明。
    7. 在 **「标识」** 下拉框中选择税码标识类型（如标准税率、减征税率等）。
    8. 在 **「税率」** 输入框中填写税率数值（例如 0、9、13 等），系统会自动在右侧展示 `%`。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763360864871-ec52fbcf-2dc6-4169-a283-5fe3b9b2ec61.png)

3. 完成填写后，点击表单底部 **「保存」** 按钮，即可在“税码管理”列表中新增一行记录，显示刚才录入的税码及其税率、标识等信息。点击点击表单底部 **「取消」** 按钮，即可取消新增。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763361082650-dbcffb57-8aa1-4512-b481-6e9abea453a2.png)

**结果说明**：

+ 新建的税码可以在后续的发票业务中被选择并回填税率。
+ 创建时间会自动记录在“创建时间”列中。

#### 编辑税码
**操作目标**：修改已有税码的名称、税率或标识等信息，以适应最新合规要求。

**操作步骤：**

1. 在“税码管理”列表中，通过筛选条件或搜索框定位到目标税码。
2. 在该行的“操作”列中，点击 **“编辑”（铅笔图标）**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763360886148-934a9638-29b5-46d4-a071-d3d3b48ba442.png)

3. 右侧打开“编辑税码”表单，字段会预填当前记录信息。
4. 根据需要修改字段，例如调整 **「分类名称」**、**「标识」** 或 **「税率」** 等。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763360905559-b1eea8f1-a086-4bc5-ac4a-62938ad46fe0.png)

5. 修改完成后点击 **「保存」**，即可将此税码的数据更新为新值。点击 **「取消」**，即可取消本次修改。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763361130866-b0336886-8abc-4711-8e57-36ab7b0154bd.png)

**结果说明**：

+ 编辑后的税码将影响后续开票时的税率回填与合规判断。

#### 过滤与搜索税码
+ **操作目标**：在大量税码中快速定位符合条件的记录。

**操作步骤：**

1. 在“税码管理”页面顶部筛选区域：
    1. 如需按国家过滤，在 **「国家/地区」** 下拉中选择目标国家/地区。
    2. 如需按税制过滤，在 **「选择税制」** 下拉中选择 VAT/GST 等税制。
    3. 如需按关键字搜索，在 **「按税码或分类名称搜索」** 输入框中输入关键字（如分类编码、名称的一部分）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763360954159-8074ce15-2819-4f1c-989c-b9a637067baf.png)

2. 点击 **「查询」** 按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763360964876-5b3f0d33-8c70-454f-b85f-919d8784a4d1.png)

3. 列表刷新后，仅显示符合所有筛选条件的记录。
4. 若想取消筛选并返回默认视图，点击 **「重置」** 按钮，清空筛选条件后列表重新加载。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763360981004-538008cc-0773-4033-a77b-3dd8474518c0.png)

**结果说明**：

+ 筛选不会修改任何数据，仅影响当前页面展示。
+ 当筛选条件过于严格可能导致列表为空（查询不到记录）。

#### 删除税码（单条及批量）
**操作目标**：清理不再使用或错误创建的税码记录。

**操作步骤（单条删除）：**

1. 在列表中找到目标税码。
2. 在该行“操作”列点击 **“删除”（垃圾桶图标）**。
3. 在弹出的确认对话框中确认删除操作。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763361004977-9c27716d-a302-481b-bdf9-6a8772c6f96c.png)

**操作步骤（批量删除）：**

1. 在列表左侧勾选多条需要删除的税码记录。
2. 确认顶部按钮 **「批量删除(×)」** 中的数字已大于 0。
3. 点击 **「批量删除(×)」** 按钮。
4. 在确认对话框中确认删除。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763361050614-809aa365-e71f-4eeb-9861-afcf93173d76.png)

**结果说明**：

+ 删除成功后，列表中相应记录消失。
