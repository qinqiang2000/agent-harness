# 功能说明
为更好的管理客户的权益数据，并方便发票云租户可以自助查看订单、权益等数据，发票云推出了全新的租户权益查看平台。

我们将通过发送邮件的方式将查询平台链接和客户登录账号信息发送到您的邮箱地址，邮件内容如下：

# 登录租户查询平台
1. 租户用户可以点击邮件中连接打开租户查询登录平台，登录平台地址链接为：

[http://cosmic-pro.piaozone.com/ierp/login.html](http://cosmic-pro.piaozone.com/ierp/login.html)

2. 在登录页面输入邮件中的账号、密码登录系统，进行租户权益数据查询

<font style="color:#585A5A;">备注：首次登录，会要求用户修改密码，请确保系统账号显示的手机号能正常使用并能接收短信，如果邮件中发送的账号手机已经不使用了，请联系发票云售后进行用户信息修改。</font>

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699237804575-741ed912-2f2b-4915-ac23-2165a352d5dc.png)

3. 登录成功后，点击 应用-运营管理 即可根据要求查询相关权益数据

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699237891826-87889139-e53d-44ef-ba2c-b5cd28b35811.png)

# 租户权益看板
### 进入运营管理应用后在哪个模块查看租户看板？
进入运营管理应用后，点击左边菜单栏的租户模块，点击租户看板，即可查看当前租户下的权益数据汇总

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699238094280-db97b3ca-7879-4fa4-902c-9a52c93ca3fa.png)

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699238126090-c9ed5e71-00c4-4064-acd2-8478f16f4b44.png)

### 租户权益看板页面各个区域信息说明
看板信息包含：

+ 租户整体基础信息；
+ 租户管理员列表，并对管理员账号进行编辑修改；
+ 租户已购买在有效内且还没有分配使用企业的订单，支持用户查看并进行分配；
+ 许可即将到期的产品和已经过期的产品，可以根据到期时间即时进行续费；
+ 以及分配了产品使用许可的企业授权信息和租户流量产品的耗用情况和扣减规则。

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699238873907-aa2cc7a5-a473-4e0e-9bba-99b0936d6369.png)

### 查看租户详情
进入租户权益看板后，点击”查看租户详情“即可查询到租户名称等信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1758091503704-835042a8-f28d-40d1-abb8-8367e89a7eb2.png)

### 查看修改租户管理员信息
点击看板上的【查看租户管理员】-弹出管理员管理操作弹窗，在操作弹窗可操作管理员账号的启用/禁用，只有启用状态的管理员可以登录到权益查看系统。

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699238944134-f3efe516-4439-42f4-988f-91dece03d36e.png)

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699239035415-277754ac-1a3e-4899-af24-7235de406f54.png)

### 查看租户下已购买有效期内但是没有分配完成的订单
点击看板页面上的交易订单数量-弹窗查看当

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699239145379-e319c3ba-21c3-4f17-b83d-51fca4b014e6.png)

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699239349062-3784842c-4b48-4402-9622-8d7c9b0b7655.png)

### 查看租户下即将到期的企业和产品的分配详情
用户可以通过租户看板上产品到期续费提醒模块，查看不同时间段内即将到期的产品和分配情况。可获取到续费所需的产品序列号等信息，从而联系机构销售人员协助进行续费操作

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699239543261-f8d1406d-70d0-4e2f-9e26-9b02bccfc21c.png)

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699239749914-4c3f9259-771d-4e87-ac13-c4f3bf7d7c0f.png)

### 查看租户下绑定了产品权益业务配置信息
用户可以通过租户看板上产品使用配置信息模块，点击对应的产品查看获取租户下对应产品的业务配置信息，例如企业名称、企业税号、权益类型、授权信息等。

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699239889911-148a20b2-ded0-4e27-a047-ec1d7969910f.png)

### 租户流量耗用查询
租户如果想查询租户下流量产品的耗用情况，可通过看板上的票量耗用明细和短信耗用明细查看。

<font style="color:#ED740C;">备注：流量产品是税号共享的权益。流量权益包含：识别查验票量、短信、存储服务等（后续持续更新补充）；没有特殊说明耗用规则的，都是使用1次扣减一个流量，比如发送一条短信，则扣减一条数量。</font>

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699240131102-b0f31e4c-3f3b-4375-8fe4-6536ea993e1a.png)

![](https://cdn.nlark.com/yuque/0/2023/png/25465214/1699240398178-47fbb44a-95ed-40a3-8c89-34dc485385ac.png)

### 租户数电信息查询
用户可以通过租户看板上租户数电信息模块，查看对应的数电激活码以及二维码。若数电信息已过期，可以点击”更新数电信息“按钮进行更新。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1759978676498-b964ab8d-7a93-4e99-ad55-67144b9ced66.png)
