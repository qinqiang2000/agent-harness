## 商品操作路径
【发票云】-【开票管理】-【海外发票】-【基础资料】-【商品】

## 商品功能设计理念概述
### 功能定位
 【商品】功能位于：【发票云】-【开票管理】-【海外发票】-【基础资料】-【商品】。  
用于集中维护开票所需的商品/服务主数据（物料编号、名称、税码、税率、分类编码等），为后续海外发票开具时自动带出行项目信息提供基础。  
典型使用角色包括：实施顾问、系统管理员、财务/税务人员等。  

### 设计理念与使用价值
+ **统一商品主数据**：在系统内集中维护商品/服务信息（物料编号、名称、规格、计量单位等），避免在各业务流程中重复录入。
+ **支撑合规开票**：通过配置税码、税率、分类编码及税收优惠/免税原因，支持根据当地法规选择合适的商品税务信息。
+ **提升开票效率与准确性**：开票时直接选择已维护好的商品，实现发票行信息自动带出，降低人工录入错误。
+ **支持多维查询与管理**：列表提供按物料编号、物料名称搜索及多列展示，便于快速定位、维护指定商品。
+ **灵活属性扩展**：通过“新增属性”，可扩展商品的属性维度（如颜色、规格等），应对多样化业务需求。

## 商品功能详解
### 商品关键字段说明
| **字段名称** | **业务含义** | **字段规则说明** | **其他说明** |
| --- | --- | --- | --- |
| 物料编号 | 商品/服务在系统中的唯一编码 | 在“新建商品”中为必填项，格式可为数字/字母/符号组合，建议与ERP一致 | 此列可筛选 |
| 物料名称 | 商品/服务的名称 | 在“新建商品”中为必填项 | 此列可筛选 |
| 开票名称 | 发票行上展示的名称，可与物料名称不同 | 用于满足业务上“内部名称”和“对外票面名称”不一致的场景 | 此列可筛选 |
| 规格型号 | 商品的规格或型号描述 | 用于补充说明商品规格信息 |  |
| 计量单位 | 开票行使用的计量单位（如：个、箱、小时等） | 从预设的计量单位列表中选择 |  |
| 商品描述 | 面向业务/客户的商品说明，便于开票选择和理解 | 一般为长文本，可选填，用于补充描述 |  |
| 分类编码类型 | 商品分类编码所使用的编码体系类型   | 下拉选择分类编码的类型，目前可选值包括：UNSPSC、HSCode、ServiceCode、Other。用于指定“分类编码”字段所采用的编码体系。 |  与“分类编码”字段配合使用 |
| 分类编码 | 商品对应的分类编码 | 如 26111600 或 HS 编码；用于记录商品在某一分类体系下的编码 |  |
| 税码 | 主要标识商品对应的增值税税率类型 |  下拉选择税码，当前下拉选项包括：VAT_STD、VAT_SERVICE、VAT_REDUCED、VAT_ZERO、VAT_EXEMPT 等。不同税码代表不同类型的增值税/税务处理方式。 |  |
| 税率 | 商品适用的税率 | 按小数形式录入税率，例如填写“0.13“ 代表“ 13%”。 |  |
| 税收优惠/免税原因 | 当商品享受税收优惠或免税时的原因说明 | 用于记录政策依据/原因说明，以满足当地合规留痕 |  |
| 商品属性 | 对商品进行扩展属性维护的区域 | 通过“+ 新增属性”添加新的属性项；属性名称、类型、值等结构。适用于对商品进行多维度描述，如颜色、尺寸、材质等 |  |
| 启用 | 控制商品是否在业务场景中可被选用 | 在“新建商品”底部显示为开关控件；打开表示该商品为启用状态，可在开票等场景中选择；关闭则视为停用 |  |
| 状态 | 商品当前的可用状态 | 列表列头为“状态”；展示商品是否启用、停用 | 与“启用”开关和业务规则联动 |

### 商品主要操作说明
#### 新增商品
**操作目标**  
在系统中新增一条商品主数据，用于后续开具海外发票时选择。

**操作步骤**

1. 进入【发票云】-【开票管理】-【海外发票】-【基础资料】-【商品】页面，打开“商品管理”列表。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373358546-ae2abeb9-a9f6-4b1c-88f2-93fed2a49afa.png)

2. 点击列表上方绿色按钮 **「+ 新建商品」**，右侧弹出“新建商品”编辑面板。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373372083-089f41bf-3515-4d77-80f2-3f645a71312d.png)

3. 在“新建商品”面板中依次填写各字段：
+ **物料编号**：必填，建议与 ERP 物料编码保持一致。
+ **物料名称***：必填，作为商品的主名称。
+ **开票名称**：如发票行上需要展示不同于物料名称的名称，在此填写。
+ **规格型号**：填写商品的规格/型号信息。
+ **计量单位**：从下拉中选择合适的计量单位。
+ **商品描述**：填写面向业务/客户的商品说明，便于开票选择。
+ **分类编码类型**：从下拉中选择 **UNSPSC / HSCode / ServiceCode / Other** 等类型。
+ **分类编码**：在选定类型下填写对应编码，如 “26111600” 或 HS 编码。
+ **税码**：从下拉中选择 **VAT_STD / VAT_SERVICE / VAT_REDUCED / VAT_ZERO / VAT_EXEMPT** 等税码。
+ **税率**：按小数形式录入税率，如 13% 录入为 `0.13`。
+ **税收优惠/免税原因**：如商品享受优惠税率或免税，在此填写原因说明。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373407900-621a03d0-15d0-4529-884c-fc6ea051527e.png)

+ **商品属性（可选）**：在“商品属性”区域点击 **「+ 新增属性」**，按界面要求维护属性信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373420219-b8acf1de-2af6-42a9-b1ae-264406d54c9e.png)

4. 确认底部 **「启用」** 开关状态：
+ 打开：新建后即为启用状态，可在开票等场景中选择。
+ 关闭：新建为非启用状态，仅用于预配置，不支持使用。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373455511-e70f3b42-79ca-4a25-990d-35f2212f0cb4.png)

5. 点击右下角 **「保存」** 按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373468244-f7866bd5-6df7-471e-b9fa-0146f9ad6856.png)

**结果说明**

+ 保存成功后，商品出现在“商品管理”列表中，对应列展示其物料编号、物料名称、开票名称、分类编码、类型、税码、税率、状态等信息。
+ 若必填项缺失或格式不符合校验要求，系统会提示错误，需按提示修改后再次保存。

#### 按条件查询商品
**操作目标**  
通过物料编号或物料名称快速定位目标商品。

**操作步骤**

1. 在“商品管理”列表顶部搜索框中输入待查询的 **物料编号** 或 **物料名称** 关键字。
2. 点击搜索框右侧蓝色按钮 **「查询」**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373506854-95c06ee8-b0cc-4421-bc52-33fd1dc86705.png)

3. 查看下方列表中返回的商品记录。
4. 如需清空条件并查看全部商品，点击 **「重置」** 按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763373524340-0bf0fb1b-7905-4ac2-82c2-84d453d4c1df.png)

**结果说明**

+ 列表将根据输入条件进行过滤，仅展示匹配的商品。
+ 重置后恢复默认查询条件和列表数据。

#### 编辑商品
**操作目标**  
对已有商品的基础信息（名称、规格、税码、税率等）进行修改。

**操作步骤**

1. 通过查询或翻页，在“商品管理”列表中找到目标商品。
2. 在该行的 **「操作」** 列中，点击 **「编辑」** 按钮。
3. 右侧打开“编辑商品”面板，根据需要调整各字段信息：
    - 可修改的字段范围以系统配置为准，一般包括：开票名称、规格型号、计量单位、分类编码类型与分类编码、税码、税率、税收优惠/免税原因、商品描述、商品属性等。
    - 如涉及调整税码/税率，应根据当地法规及企业内部政策进行确认。
4. 修改完成后，点击面板底部 **「保存」**。

**结果说明**

+ 保存成功后，列表中该商品的相关字段会更新为最新值。
+ 若修改内容不符合校验规则（例如税率格式不正确等），系统会弹出错误提示，需根据提示调整。

#### 删除商品
**操作目标**  
将不再需要、且尚未正式使用的商品从系统中完全删除。

**操作步骤**

1. 在“商品管理”列表中找到目标商品。
2. 在该行的 **「操作」** 列中点击 **「删除」** 按钮。
3. 系统弹出删除确认弹窗，确认无误后，点击弹窗中的确认按钮。

**结果说明**

+ 删除成功后，该商品从列表中消失。
