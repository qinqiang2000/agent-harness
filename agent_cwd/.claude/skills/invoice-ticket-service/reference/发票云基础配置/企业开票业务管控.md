## 企业开票业务管控设计理念概述
### 企业开票业务管控设计理念
1. 以核算组织与开票业务的组合为基础，用于配置选用核算组织所属税号的开票通道。这样可以确保开票信息与企业的核算组织和税务信息紧密关联，提高开票的准确性和税务合规性。
2. 通过对开票业务类型、通道类型、通道调度策略、授信额度等关键要素的设置和管理，实现对企业开票业务的精细化管控。用户可以根据自身业务需求和实际情况，灵活配置这些参数，以满足不同业务场景下的开票需求。
3. 强调用户在配置过程中的自主性和灵活性。例如，用户可以从已配置的开票业务类型和通道调度策略中进行选择，也可以自行调整每日授信额度限额、自然月授信额度限额和预警阈值等参数，使开票业务管控更符合企业的个性化需求。

### 目标客户以及客户解决核心痛点问题
1. <font style="color:rgba(0, 0, 0, 0.85);">开票业务类型复杂多样：</font>企业可能涉及多种不同的开票业务类型，每种业务类型的开票规则和要求可能存在差异。如何准确区分和管理这些不同的业务类型，确保开票的准确性和合规性，是用户面临的一个挑战。
2. <font style="color:rgba(0, 0, 0, 0.85);">开票通道选择与调度问题：</font>存在多个开票通道时，用户需要考虑如何选择最合适的通道以及通道之间的优先级问题。不同的通道可能在稳定性、速度、成本等方面存在差异，如何根据实际业务需求进行合理的选择和调度是一个难题。

### 企业开票业务管控主要操作说明
##### 新增企业开票业务管控
**步骤1：**<font style="color:#000000;">点击“新增”按钮，新增企业开票业务管控。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366296054-0e94386d-ff12-48a4-bdff-240d80feb373.png)

  
**步骤2：**<font style="color:#000000;">在详情页编辑基本信息。</font>

<font style="color:#000000;">开票时，业务方传入新参数“开票业务类型”，开票申请单处理后生成待开发票，待开发票下推到税局通道前，通过组织对待开发票的开票业务类型进行业务校验，并根据开票业务管控匹配出可用的通道和通道调度策略再将二者推送至税局开票。</font>

<font style="color:#000000;">开票基本类型业务方已配置完成，此处作为参数传入。待开发票通过开票业务类型校验，通过后，授信额度预扣，在进入通道校验过程。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366313376-bd976a7e-4420-4908-85d7-7fe55bb219e7.png)

**步骤3：**配置开票通道

此处推送可用通道数组包括乐企、RPA以及税控。用户也可设置全部通道均可用，系统根据调度策略自动选择通道。同个业务可能有多个可用通道，多种通道类型间有固定优先级：乐企>电子发票服务平台账号>税控；并且支持配置多种通道调用规则，如上游指定电子发票服务平台账号不在开票业务管控允许范围内则不允许开票，或是优先匹配上游指定的账号，否则降级取管控允许的其他账号开票等。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366337970-4467823a-a0fc-4481-8250-557610ca9205.png)

**步骤4：**配置通道调度策略

在通道调用策略中已配置完成，此处推送可用的通道策略。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366360453-94ea3bba-4b94-4969-9d3a-2b2228ec4316.png)

**步骤5：**配置开票业务类型的授信额度

自然月授信额度需大于或等于每日授信额度。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366381636-98280295-a955-4c8b-a91e-e6652786a2b2.png)

**步骤6：**配置开票角色策略

其中，<font style="color:#000000;background-color:#FAFAFA;">角色日授信额度必须小于或等于业务类型管控日授信额度，同理自然月授信额度需大于或等于日授信额度。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366403935-f2b313d3-4ff4-4ba6-a6f0-139a3cde3140.png)
