【发票云】-【开票管理】-【海外发票】-【公共数据】-【身份类型】

## 身份类型功能设计理念概述
### 功能定位
“身份类型”功能用于维护 Party/企业身份标识的类型方案（Scheme）主数据，例如 `NL:VAT`、`GLN` 等。  
系统通过为每个 Scheme 配置 Description、Country/Region、Type 等属性，定义不同国家/地区在录入企业身份信息（如税号、登记号等）时可选的身份类型。

### 设计理念与使用价值
+ **统一维护企业身份类型**：集中维护 Scheme、Description、Country/Region、Type 等信息，避免在各业务流程中重复维护或口径不一致。
+ **支持多国家/地区配置**：通过 Country/Region 字段区分不同国家/地区的身份类型。
+ **区分身份类型用途**：通过 Type 字段（如 TAX、LEGAL、OTHER）标识不同用途的身份类型，方便业务规则按类型进行过滤或校验。
+ **列表化管理与批量维护**：支持查询、筛选、导入、导出 CSV 和批量删除，提升主数据管理员日常维护效率。
+ **与 Party 主数据联动使用**：该身份类型列表通常在 Party/企业信息维护时被引用，确保在开票等业务中选择正确、规范的类型。

### 身份类型主要操作说明
#### 新增身份类型
**操作目标**：在系统中新增一条企业身份类型（PartyID），供后续 Party/企业信息引用。

**操作步骤**：

1. 在页面右上方点击【新增】按钮，打开“新增 PartyID”弹窗。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379079285-29acd646-d5ad-40e8-ba37-fb35e2a9a250.png)

2. 在弹窗中依次填写：
+ *** Scheme**：输入身份类型编码，例如“NL:VAT”或“GLN”等，需符合企业编码规范。
+ *** Description**：填写该身份类型的描述，如“HK BR No. (Legal Company ID)”等。
+ **国家/地区**：从下拉框中选择该身份类型适用的国家/地区，如 HK、CN 等。
+ *** Type**：从下拉框选择身份类型分类，如 TAX、LEGAL、OTHER 等。默认值为“TAX”。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379105687-659ccfee-c8d8-49e3-ac67-6dfffd1e3122.png)

3. 点击弹窗右下角【确认】按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379114443-ff51a72c-8d65-4d76-9023-3f84ce43f895.png)

**结果说明**：

+ 校验通过后，系统保存该记录并关闭弹窗，新记录出现在“数据列表”中。

#### 编辑身份类型
**操作目标**：修改已存在的身份类型信息（例如修正文案、调整适用国家/地区或类型）。

**操作步骤**：

1. 在列表中，找到需要修改的记录。
2. 在该行的【操作】列中点击**铅笔图标（编辑）**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379160171-293bc66d-b005-497c-a6f4-c21b15e4c5fe.png)

3. 在弹出的编辑弹窗中，调整需要修改的字段，例如：Description、Country/Region、Type 等。
4. 修改完成后点击【确认】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379191396-83155f3e-7613-4e2a-93a1-2e1340b84c43.png)

**结果说明**：

+ 保存成功后，列表中对应记录被更新。

#### 删除身份类型（单条与批量）
**操作目标**：删除不再使用或误建的身份类型记录。

**操作步骤（单条删除）**：

1. 在列表中定位目标记录。
2. 在该行【操作】列点击**垃圾桶图标（删除）**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379233274-75e0105e-6191-4c14-84c3-9e411a612030.png)

3. 在弹出的确认提示（<font style="color:rgba(0, 0, 0, 0.88);">确定要删除这条记录吗?</font>）中确认删除。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379259809-48f91307-fbfb-430d-a86e-8c41e758e814.png)

**操作步骤（批量删除）**：

1. 在列表左侧，勾选需要删除的多条记录（通过行首复选框）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379274065-121f2872-b2e0-41a3-9c3e-c08a6df01602.png)

2. 点击工具栏中的【批量删除】按钮（实际显示数字会随勾选数量变化，如“批量删除(3)”为选中的需删除数据有3条）

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379311550-948cbe3c-3d22-42bb-b46f-d7e31c782141.png)

3. 在系统弹出的确认提示中确认删除操作。

**结果说明**：

+ 删除成功后，对应记录从列表中消失。

#### 查询与筛选身份类型
**操作目标**：在大量身份类型记录中快速定位目标记录。

**操作步骤**：

1. 在页面顶部的搜索区域：
+ 在 **“搜索Scheme/Description/CountryCo...”** 输入框中，输入关键字（如 “VAT”“HK”“Tax ID”等），或
+ 在 **“全部类型”** 下拉中选择某一类型（如 TAX/LEGAL/OTHER），或
+ 在 **“选择国家/地区”** 下拉中选择特定国家/地区。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379375312-a4df7519-caf0-4a73-9b3a-4245d85b5d0d.png)

2. 输入/选择完成后，点击【查询】按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379387275-587055f4-a395-4215-9846-84cca6adcb3c.png)

3. 查看“数据列表”中的结果；如结果较多，可通过分页区域的页码按钮、`10条/页` 下拉和 `跳至` 输入框继续翻页查找。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763379405909-c139b298-451f-40f7-aa16-e29d36c52f64.png)

**结果说明**：

+ 列表将根据设置的条件展示匹配的身份类型。
+ 若条件过于严格导致无记录，列表可能显示为空。
