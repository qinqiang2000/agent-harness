## 企业信息操作路径
【发票云】-【开票管理】-【海外发票】-【基础资料】-【企业信息】

## 企业信息功能设计理念概述
### 功能定位
“企业信息”用于集中维护跨国家/地区的开票企业主数据，包括企业的基础信息、默认税务标识、法律标识，以及多套税务 ID、地址、联系人、交货地址和付款方式等信息。  
该功能主要服务于实施顾问、财务/税务人员和系统管理员，用于在全球电子发票流程中统一管理企业档案，为后续开票、收票与对账等业务提供基础数据。  

### 设计理念与使用价值
+ **统一企业主数据**：将不同国家/地区的企业信息集中维护，避免在各个开票流程中重复录入。
+ **支持多国家/多身份标识**：通过“默认税务标识”、“其他税务ID”、“法律标识”等，支持企业在不同国家/地区下的税务与法律标识配置。
+ **多维度联系信息管理**：通过“地址信息”、“联系人”、“交货地址”、“付款方式”等子表，覆盖企业日常业务所需的地址、收货、付款账户等信息，便于业务调用。
+ **默认配置能力**：列表及子表中提供“默认”控制，用于标记系统优先使用的税号、地址、联系人或付款方式，减少业务操作时的选择成本。
+ **状态可控**：通过“启用状态”和行级开关，控制企业档案是否参与后续业务，支持按需启用/禁用。

### 企业信息主要操作说明
#### 新增企业主数据
**操作目标**：在系统中创建一条新的企业档案，并维护基本信息及常用标识信息。

**操作步骤：**

1. **进入企业信息页面**  
1）登录系统后，依次进入：  
【发票云】→【开票管理】→【海外发票】→【公共数据】→【企业信息】。  
2）进入页面后看到企业列表及筛选区。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763364796208-dc3791d5-6997-43e1-9716-e103d8bf3dba.png)

2. **打开新增弹窗**  
点击列表上方的绿色按钮 **“+ 新增企业”**，弹出“新增企业”弹窗。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763364809383-edcaf68c-3194-4b95-afe3-e4227c4c389b.png)

3. **填写“基本信息”**  
1）在“国家/地区”中选择企业所属国家/地区；  
2）在“名称”中输入企业全称（必填）；  
3）根据需要填写：
    - “别名（Trading Name）”：企业的业务简称/门店名等；
    - “Org ID”：集团或组织标识；
    - “公司联系邮箱”：企业统一联系邮箱。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763364933510-e273962d-fa30-46e1-bf8a-97764839a71b.png)

4. **维护“默认税务标识”**  
1）在“税种”下拉框中选择该企业在本国家/地区的税种；  
2）在“税号”输入框中填写对应税号。

具体填写规则须根据当地法规及企业实际注册信息确认。

5. **维护“法律标识”**  
1）在“标识类型”中选择法律标识类型（如工商注册号、BRN 等）；  
2）在“法律标识”中填写具体编号。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763364965459-ee95c478-08c7-44f9-adb6-d051122bfb44.png)

6. **按需维护各类附属信息**  
可根据业务需要，在弹窗下方 Tab 中维护附属信息：
+ **其他标识**：  
1）切换到“其他标识”Tab；  
2）点击“添加行”，为每条其他标识选择“类型”，填写“值”。
+ **其他税务ID**：  
1）切换到“其他税务ID”Tab；  
2）为每个国家/地区填写“国家/地区”、“税号”和“方案”；  
3）根据需要点击某一行的“默认”按钮，标记为默认税务 ID。
+ **地址信息**：  
1）在“地址信息”Tab 中，为企业维护多个地址（国家/地区、街道、门牌号、城市、邮编、州/省）；  
2）如需设置主要地址，点击对应行的“默认”。
+ **联系人**：  
1）在“联系人”Tab 中维护企业的联系人：国家/地区、联系人姓名、职位、部门、邮箱、电话；  
2）可为主要联系人点击“默认”。
+ **交货地址**：  
1）在“交货地址”Tab 中维护交货用地址：地址、城市、邮编、州/省、国家/地区；  
2）按需设置默认交货地址。
+ **付款方式**：  
1）在“付款方式”Tab 中维护收款账户信息：方式代码、账号、账户名称、银行名称；  
2）设置默认付款方式（如仅一条记录，通常应标记为默认）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365006504-51e0fee2-d959-4dbf-9f31-5145a587ee40.png)

7. **保存企业信息**  
1）确认所有必填项已填写；  
2）点击弹窗右下角 **“保存”**；  
3）保存成功后，弹窗关闭，新建企业出现在企业列表中；

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365022348-c12268c7-70e5-4851-801a-e7101f580276.png)

#### 编辑企业信息
**操作目标**：修改已有企业的基础信息或标识信息。

**操作步骤：**

1. 在企业列表中，通过“搜索名称/税号/注册号/GLN”或“选择国家/地区”定位到目标企业。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365116061-f911352b-ac27-49ff-b9ee-bd82ce3bb13c.png)

2. 在目标行的“操作”列，点击 **“编辑（铅笔图标）”**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365131220-18e0ab47-4994-44d5-90bb-84bdf47a0ed6.png)

3. 在弹出的编辑弹窗中，根据需要修改以下内容：
    - “基本信息”：如名称、别名、公司联系邮箱等；
    - “默认税务标识”、“法律标识”；
    - 各 Tab 中的附属信息（可添加/修改/删除子表行，或重新设置“默认”）。
4. 修改完成后点击 **“保存”**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365493810-b6855322-4e34-482f-8f1b-94a5d1abd2af.png)

5. 返回列表确认修改结果是否生效。

#### 启用 / 停用企业
**操作目标**：控制企业档案是否参与后续业务，后续是否还继续使用。

**操作步骤：**

1. 在企业列表中找到目标企业。
2. 查看“启用状态”列当前状态标签（如“已启用”）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365546724-9d17999f-fcb7-4a64-b39f-56319acb807e.png)

3. 在同一行“操作”列点击 **“启用/禁用状态开关”**：
    - 若当前为启用状态（绿色），点击后将切换为禁用状态（灰色）。
    - 若当前为禁用状态（灰色），点击后将切换为启用状态（绿色）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365622976-a0f93618-a5c8-457b-86f5-32f455660d93.png)

4. 状态切换成功后，列表中的“启用状态”标签随之更新。

#### 删除企业
**操作目标**：删除不再使用或误建的企业档案。

**操作步骤：**

1. 在企业列表中找到目标企业。
2. 在“操作”列点击 **“删除（垃圾桶图标）”**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365669544-049e7dd4-cbae-4645-abd3-6f7e62c65f8c.png)

3. 系统应弹出确认提示：<font style="color:rgba(0, 0, 0, 0.88);">确定要删除这条企业信息吗？点击“确认”，即可从列表中删除此条企业数据。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763365710756-2b9d02ab-c04c-4347-9b16-696950355581.png)
