## 商品统计功能设计理念概述
1. **功能概述**

<font style="color:rgb(51, 51, 51);">发票统计报表提供多种不同维度的发票数据统计，为主要服务于企业内部运营、管理团队，为企业内部了解已开票数据、分析业务趋势、监测开票状态提供有力依据。</font>

<font style="color:rgb(51, 51, 51);">主要提供商品、购方、税率、状态、资料、对账、业务类型等统计维度；</font>

商品统计：从发票的明细行层面对发票信息进行汇总统计，方便用户查看热销商品的税额、金额。

支持查询统计商品对应的开票数量、发票数量、金额、税额和价税合计等。

2. ******<font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>**

<font style="color:rgba(0, 0, 0, 0.85);">①内部操作使用人员：财务或相关统计岗位，解决了查询和统计商品开票信息的痛点，能够快速获取商品的开票数量、金额、税额等信息，提高了数据查询和统计的效率。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">②部门管理人员：财务部门管理人员，解决了对商品开票数据进行分析和监控的痛点，能够通过商品统计报表了解业务趋势，监测开票状态，为决策提供数据支持。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">③公司管理层：公司高层管理岗位，解决了对公司开票数据进行整体把握和决策的痛点，能够通过商品统计报表了解公司的业务情况，及时发现问题，制定合理的经营策略。</font>

### 商品统计主要操作说明
#### 商品统计查询
**步骤1：**选择查询的组织和日期，可进一步填写商品名称以设置过滤条件，点击“查询”按钮，即可看到该条件下的商品统计详情。

注：旧版本菜单中还可以通过中，填写已选中组织的名称和税号添加过滤条件。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1750821474759-c6c12b6f-eacc-4fac-a777-c17252c5d529.png)

**步骤2：**点击发票数量的数字，可以穿透到发票查询页面。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1750821490718-e6b5c6e9-f743-4b66-8666-091a53f3da43.png)
