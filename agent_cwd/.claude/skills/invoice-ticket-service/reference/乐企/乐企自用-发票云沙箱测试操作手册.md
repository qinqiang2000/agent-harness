# 一、测试说明
**企业使用乐企沙箱，需要满足以下条件： **

1、直连单位需完成乐企直连单位接入，将系统的互联网出口 IP 报税务机关备案，开通 IP 白名单。 

2、直连单位需完成乐企数字能力的订阅。 

3、直连单位需要获取自身及虚拟使用单位平台编号（乐企 ID）。 

4、直连单位需要获取沙箱密钥用于报文请求加密。 

5、在沙箱环境，<font style="color:#DF2A3F;">一个直连单位默认分配一个虚拟使用单位</font>。

_除保险行业企业外销售方纳税人识别号和购买方纳税人识别号不能相同，__<font style="color:#DF2A3F;">直连单位只能给虚拟使用单位开票，或接收虚拟使用单位开票</font>__。保险行业企业允许购销方纳税人识别号一致，需调用企业基础信息更新接口设置行业代码。_

更多信息请查看：[乐企数字开放平台-沙箱操作指引1（开票及基础）-V4.000.pdf](https://jdpiaozone.yuque.com/attachments/yuque/0/2025/pdf/39256605/1760598488884-f963df3c-8662-404b-9079-a7a0e5b856ed.pdf)

<font style="color:rgb(0,0,0);">4、</font>**<font style="color:rgb(0,0,0);">接入纳入识别号：</font>**<font style="color:rgb(0,0,0);">必填，选择接入单位税号，需先在发票云→基础资料→企业信息中，将接入单位的税号绑定发票云组织中，并完成销项许可授权。</font>

<font style="color:rgb(0,0,0);">5、</font>**<font style="color:rgb(0,0,0);">省市级税务机关代码：</font>**<font style="color:rgb(0,0,0);">必填，选择接入单位的机关代码。</font>
