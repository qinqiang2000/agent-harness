# 1.概述       
为了使企业纳税人能够使用测试中心进行沙箱、能力测试，本文对测试中心

的能力测试管理做了详细介绍。 

 

# 2.能力中心 
## 2.1.我的能力 
### 2.1.1.操作步骤 
1.纳税人点击某一项能力开通状态为“未开通”的非公用基础能力，进入能力详情页面，能力详情页面展示“申请开通”按钮，界面如图所示。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902592059-58b2655d-1b1c-4342-ae66-6ab79a83614d.jpeg)<font style="color:#004080;"> </font>我的能力初始界面 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902592426-2c5a0dd0-e970-45e9-8cd4-bbda036ce6d6.jpeg)<font style="color:#004080;"> </font>

能力详情<font style="color:#004080;"> </font>

2.点击“申请开通”按钮，系统跳转能力开通页面，界面如图所示。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902592657-3932479e-393c-42a9-ad8f-ca66f9ec4c3b.jpeg)<font style="color:#004080;"> </font>

能力开通申请 

3.纳税人完成能力开通页面必录项录入，点击“提交”按钮，界面如图所示。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902592848-b1cd3a8d-f93f-41a2-958d-48ab384519c3.jpeg)<font style="color:#004080;"> </font>

录入申请信息<font style="color:#004080;"> </font>

4.提交成功，页面回到【我的能力】初始页面，能力开通状态变更为待测试，

界面如图所示。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902593081-9971f252-070d-4a38-b65a-79ebe862d6a5.jpeg)<font style="color:#004080;"> </font>我的能力初始界面 

 

# 3.测试中心 
## 3.1.能力测试管理 
### 3.1.1.操作步骤 
1.纳税人提交能力订阅申请后，点击【测试中心】按钮跳转测试中心-能力测试管理页面，按照订阅批次查看能力测试情况。界面如图所示。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902593381-6da67acc-4b3a-4ecb-a853-f0bee45766c5.jpeg)<font style="color:#004080;"> </font>能力测试管理初始界面<font style="color:#004080;"> </font>

2.点击【查看详情】查看该订阅批次下能力对应测试任务。进入能力测试详情页面，在能力测试模块查看需要测试的所有能力及其对应的测试用例，点击【查看接入指引】可查看能力测试指引页面，在测试用例列表中勾选需要测试的测试用例后点击【启动测试】按钮启动能力测试。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902593605-a98ffe5c-453f-41c3-8194-b190fafa440e.jpeg) 

 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902593805-a8f3d336-a2a5-40cd-8c1d-fe6d9846eebf.jpeg)<font style="color:#004080;"> </font>测试详情 

3.能力启动测试后，页面将展示测试倒计时，用户需操作自有业务系统按照测试用例调用仿真测试能力。调用完成点击“验证结果”按钮，判断此次测试是否通过。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902593990-c2cb2ddc-d5ba-4803-b82e-1baa5a34e366.jpeg)<font style="color:#004080;"> </font>启动测试成功 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902594209-f196d97e-35a7-41e6-8c92-2293f1605976.jpeg)<font style="color:#004080;"> </font>

测试中<font style="color:#004080;"> </font>

4.当能力测试通过时，点击“上传附件资料”按钮进行附件资料上传。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902594405-287b59f9-8f33-42c4-b824-4b55ff23590a.jpeg)<font style="color:#004080;"> </font>测试通过<font style="color:#004080;"> </font>

当能力测试不通过时，在测试倒计时结束之前，纳税人可以继续操作本单位业务系统进行测试，之后点击“验证结果”按钮重新验证测试结果。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902594607-955f098c-d065-49f0-b784-4a66507df76c.jpeg)<font style="color:#004080;"> </font>

测试不通过<font style="color:#004080;"> </font>

4.纳税人可在测试用例列表中点击“测试结果”按钮查看测试用例下所有接口测试结果信息。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902594824-ff42e3b6-2faa-4965-ba87-cbc9efa2c062.jpeg) 能力接口测试结果 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902595069-49bb8690-8dd1-4541-af55-140ed71289d5.jpeg)<font style="color:#004080;"> </font>接口详情测试信息<font style="color:#004080;"> </font>

6.测试用例测试通过后可以在附件资料上传模块，选择已测试通过的用例点击“上传”按钮，上传资料，并填写资料说明后点击“保存”按钮。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902595285-d972a109-1369-4f93-b56f-f4ab6092f37c.jpeg)<font style="color:#004080;"> </font>待上传资料列表<font style="color:#004080;"> </font>

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902595501-ad8b8fea-e6be-4bac-8167-4276ca67e418.jpeg) 

资料上传 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902595715-4840ef34-5fb1-426c-aa83-b255dbe7d601.jpeg)<font style="color:#004080;"> </font>上传资料并填写说明<font style="color:#004080;"> </font>

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902595919-7a3d318a-9dce-4297-8ac9-c9558ccd5c28.jpeg)<font style="color:#004080;"> </font>

上传成功 

7.当批次下所有能力全部测试通过且已上传附件资料后，点击“生成测试报告”按钮，生成测试报告，测试报告生成成功后点击“查看测试报告”可查看测试报告并提交。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902596154-e09b4bf4-34d9-42ba-bdbf-3301cc059111.jpeg)<font style="color:#004080;"> </font>

生成测试报告 

8.测试报告生成后可直接预览，也可点击“下载报告”下载已生成测试报告，

点击“查看报告”在线预览报告；点击“提交审核”按钮提交至税局端进行受理审核。 

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902596340-c7efbffc-d994-4f64-951f-ae1deb95f1a9.jpeg)<font style="color:#004080;"> </font>查看测试报告<font style="color:#004080;"> </font>

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902596525-c5c2e031-f778-41cd-bf3c-90aa3c36652e.jpeg)<font style="color:#004080;"> </font>提交审核成功<font style="color:#004080;"> </font>

![](https://cdn.nlark.com/yuque/0/2024/jpeg/38481329/1724902596793-bc95b417-7a51-4e52-8eb4-5f42ab561691.jpeg)<font style="color:#004080;"> </font>能力测试管理初始界面<font style="color:#004080;"> </font>
