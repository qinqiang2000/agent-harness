# 三、乐企平台操作说明
本文档主要分为三个部分，乐企资质申请、乐企沙箱调试、乐企开通使用。

**（1）乐企资质申请**为线下准备资料的线上化过程；

**（2）乐企沙箱调试**为乐企沙箱环境的能力调试过程；

**（3）乐企开通使用**为乐企邀请使用和能力授权确认过程。

## 1、乐企接入单位申请
根据线下准备好的资料清单，通过乐企平台线上提交给税务局。

**路径：乐企平台-接入管理-接入申请**

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434103092-0eed5e63-e72d-43f7-8f33-40e9cc594fca.png)![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720433956985-179bc7c3-ff9c-4444-a0f3-250df93959f5.png)![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720433962463-1578e26d-091f-4583-93cc-782d280af602.png)![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720433966823-cac34022-9ef9-4fc3-bcec-127ff888b28b.png)

提交后，可以和税务局联系沟通，等待审核通过。

## 2、乐企调试
开通乐企能力，然后进入调试阶段，这个过程需要技术人员在乐企前置服务配置好相关的环境。

**乐企平台操作如下：**

### 2.1 获取乐企ID和密钥
（1）路径：乐企平台—接入管理—接入申请，如果审核通过后，可以获取乐企ID

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720433973345-671895f0-01de-4a69-b6c5-ea1c20b9a736.png)



**（2）路径：乐企平台—接入管理—秘钥管理**，获取乐企沙箱环境秘钥和正式环境秘钥，发给实施人员。

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720433979835-406e91d0-7c94-47a0-8e8c-c8d7a1efe3aa.png)

### 2.2 开通能力
**路径：乐企平台—能力中心—我的能力**，选择需要开通的能力，默认开通<font style="background-color:#FBDE28;">乐企开票能力（基础版）</font>和<font style="background-color:#FBDE28;">乐企归集能力</font>、<font style="background-color:#FBDE28;">查验能力</font>，其他能力可根据企业实际业务选择开通。

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720433986455-3fb91ff5-ea85-447a-adc1-da8a4e74c7e9.png)

开通时，输入订阅申请相关信息：

**能力开票有效期止：**选择一个较大的日期。

**拟嵌入自有业务系统名称：**发票管理系统（或XXX管理系统）

**每项能力日使用量（次/天）：**默认最大99999

**能力使用业务范围：**发票开具和管理（根据实际情况填写）

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720433996022-a7a1e28f-c4c0-480c-adeb-576bbb866d7f.png)

### 2.3 乐企调试
**路径：乐企平台—测试中心-待开通能力测试**，选择订阅的能力（根据前期调研客户的业务情况，选择对应的能力），点击查看详情，进入启动测试

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434002158-38c75c08-e943-4970-ba4b-deb395d5663d.png)![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434007817-157846c0-e0e9-4f80-80c3-9687be3771d2.png)

然后等待技术人员调试，调试完成后，点击下图**<font style="background-color:#FBDE28;">验证结果</font>**。

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434014035-7b37d65b-2d4a-4484-8612-f63e90a13d4f.png)

验证成功后，需要上传发票管理系统的界面，并进行说明，保存后，生成测试报告

<font style="background-color:#FBDE28;">（当前每个能力测试结果最多上传</font>**<font style="background-color:#FBDE28;">5</font>**<font style="background-color:#FBDE28;">张图片，每张图片大小在</font>**<font style="background-color:#FBDE28;">1M</font>**<font style="background-color:#FBDE28;">以内，以电子税局平台操作为准，）</font>

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434018867-306087b9-0708-40cf-a5d3-5bb310c2bda6.png)

生成报告后，在线提交报告，等待审核。

审核通过后，界面如下

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434026390-d98c5a49-6039-4d28-9c44-3dc937df6c21.png)



## 3、乐企使用单位邀请及授权
完成调试后，进行使用单位的配置。

### 3.1 使用单位接入邀请
邀请使用单位，可单个可批量，批量时选择导入邀请

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434032895-b91de8ab-11eb-4375-80c9-ddb4ea66e0c2.png)![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434041340-886efb84-4783-4950-bafb-16d587a8c8e1.png)

### 3.2 邀请待审核
邀请后，等待审核，审核通过后可查看到审核状态

**路径：乐企平台—接入管理—接入邀请**

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434047582-41f5a5e7-115a-4c14-a34e-66251ea12d14.png)

### 3.3 能力授权
针对邀请审核通过的企业，进行能力授权

**路径：乐企平台—能力中心—授权我的能力**

根据该企业需要申请的能力，进行授权

<font style="color:#DF2A3F;">注意：接入单位在进行能力授权时，不仅需要给使用单位授权，接入单位本身也是使用单位之一，一定需要给接入单位自己本身也进行能力授权，不然接入单位乐企能力无法正常使用。</font>

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434057139-3d92320e-851c-4064-a997-380f41e671ff.png)![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434065245-5321d265-d799-4121-9fa4-3f0bfcd620be.png)

### 3.4 接入单位能力受理管理
邀请后在能力受理管理中进行查看后续状态

**路径：乐企平台—能力中心—能力受理管理**

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434071752-093660d5-4b3c-42df-93c5-5552ff1143fe.png)

### 3.5 使用单位能力受理管理
使用单位需要登录乐企平台，在能力中心—能力受理管理下，点击详情

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434079082-2cdb9e05-2e5d-4aeb-a216-aba2a43db89e.png)

点击详情，进入确认界面

![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1720434086593-cbf0b73c-efb7-4157-add5-b02e14351bdf.png)



以上过程在乐企平台的操作完成。
