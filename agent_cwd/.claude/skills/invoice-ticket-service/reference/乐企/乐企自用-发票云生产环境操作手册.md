# 一、星瀚生产环境配置
## 指定乐企生产地址
**（1）新版本：**

1. **路径：发票云->基础资料->企业管理->企业信息**
2. **开票通道和收票通道，其中之⼀选择【乐企平台】**
3. **企业⻆⾊选择为接⼊单位**
4. **乐企接⼊地址：**[**https://lqpt.chinatax.gov.cn:8443/access/v2/invoke/**](https://lqpt.chinatax.gov.cn:8443/access/v2/invoke/)**  
****【提示】若有使⽤代理服务器转发，需要⽤代理服务器地址和端⼝号做替换**

**填写示例为****https://116.116.10.1:8443****/access/v2/invoke/**

**<font style="background-color:#FBDE28;">（2）老版本（建议都升级到新版本）：</font>**

**路径：**开发服务云->开发平台->发票云->系统管理->云应用参数配置->参数配置单据->列表（TAB）->预览 -> 新增 一项配置：(如果已经存在该项配置，则点击进行编辑操作即可)

# 二、汇总确认
**路径：发票云 -> 开票管理 -> 发票数据上报税局 -> 乐企汇总确认**

<font style="color:#117CEE;">切换乐企开票后，每个月进行增值税纳税申报前必须先操作乐企汇总确认！！！</font>

<font style="color:#DF2A3F;">注意：</font>首次切换乐企正式环境，<font style="color:rgba(245,31,53,1);">如能力授权与正式开票存在跨月</font>，即6月进行能力授权，7月切换乐企正式环境并开票，<font style="color:rgba(245,31,53,1);">需要6月切换乐企并手动操作汇总确认</font>；如能力授权与正式开票在当月，则无需手动操作新增。

## 新增汇总确认数据（首次上线时存在跨月情况需操作）
1. 选择组织、汇总⽉份、接⼊⾝份
2. 直接点击【保存】按钮

![](https://cdn.nlark.com/yuque/0/2025/png/39256605/1739244074060-687460d4-cd04-4217-b14c-f8fa61555f85.png?x-oss-process=image%2Fformat%2Cwebp)

## 月度汇总确认
正式环境通过乐企开具发票后，次月1日汇总确认列表会**<font style="background-color:#FBDE28;">自动生成</font>**一条记录，点击【汇总确认】，乐企会返回汇总确认的结果，确认成功后，【汇总状态】会更新为【已确认】，确认后也可进行【取消确认】；

如返回确认失败，则根据具体汇总失败原因调整即可。

如未手动点击汇总确认，乐企后台会在每个月13日自动进行汇总确认操作。![](https://cdn.nlark.com/yuque/0/2024/png/39256605/1721120044936-0efaad3d-3164-42b7-a473-d057b756f83c.png?x-oss-process=image%2Fformat%2Cwebp)

# 三、开票额度
乐企开票为预赋码制，开票前需要保证系统内存在可开票剩余额度

<font style="color:#117CEE;">路径：发票云->基础资料->乐企相关->授信额度管理</font>

每个月月初系统默认自动下载电子税局80%的额度，可以在系统参数界面调整自动下载比例

也可以在授信额度管理界面手动调整或退回额度

具体可参考：[授信额度管理](https://jdpiaozone.yuque.com/nbklz3/tadboa/ycmpscsw816l1ekc?singleDoc#)
