## 影像文件明细列表功能设计理念概述
影像文件明细是一张可检索、可筛选、可导出的影像文件明细台账，便于业务与运维人员对影像文件数据进行全局查看与追踪。  

**用户痛点：**

1. <font style="color:rgba(0, 0, 0, 0.85) !important;">传统方式下，影像文件预览需跳转多个页面，源文件下载流程繁琐，无法快速获取原始文件。</font>
2. <font style="color:rgba(0, 0, 0, 0.85) !important;">单文件逐一导出耗时耗力，面对大量影像文件时，缺乏批量导出和统计功能，难以满足数据整理需求。</font>
3. <font style="color:rgba(0, 0, 0, 0.85) !important;">影像文件信息展示维度固定，无法自定义导出字段，难以适配不同业务场景的统计需求。</font>

**<font style="color:rgba(0, 0, 0, 0.85) !important;">产品价值：</font>**

1. <font style="color:rgba(0, 0, 0, 0.85) !important;">通过预览与下载一体化设计，减少页面跳转和重复操作，例如审计人员可直接在预览界面下载原始发票文件。</font>
2. <font style="color:rgba(0, 0, 0, 0.85) !important;">批量导出功能满足月末数据归档需求，Excel 统计清单帮助快速核对文件完整性（如是否漏传附件）。</font>
3. <font style="color:rgba(0, 0, 0, 0.85) !important;">自定义导出字段功能允许用户仅保留所需信息（如隐藏系统内部字段），避免无效数据干扰。</font>

### 影像文件明细列表主要操作说明
##### 在线预览影像文件
在影像文件列表点击“影像文件编号”蓝色链接，可以打开影像文件预览界面。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747298941152-97646e40-91dc-4f8a-a328-637fe1f2d72b.png)

在预览界面可以点击”下载源文件“将文件下载至本地。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747298980751-3384d347-c4f9-4925-819f-f30abc27bd6a.png)

##### 导出影像文件至本地
在影像文件列表多选影像文件，点击“导出影像文件”按钮，将以压缩包形式导出所选的影像文件至本地。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747299134767-a8bce30c-405c-4fb3-ad69-5ded54cb94ed.png)

##### 导出统计清单至本地
在影像文件列表，点击“导出统计清单”按钮，以EXCEL形式导出全部影像文件信息至本地。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747299378771-10a76b50-421f-499f-9d0d-c6dbee68344a.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747299404354-9da891c4-bc20-4955-add7-52f7dbffbeec.png)

可以点击列表右侧设置隐藏或显示列，控制对应字段在导出统计清单中是否导出 ，以便按需定制导出字段范围。 

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747299422136-4fca9ab9-5139-482e-ba4d-d80fc83b2397.png)
