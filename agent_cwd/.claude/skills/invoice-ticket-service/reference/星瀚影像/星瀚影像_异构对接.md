# 环境准备
## 集团公有云
<font style="color:#117CEE;">路径：基础服务云->安全管理->许可管理->许可分配用户</font>

KBC下单并出库后，在许可分组明细中点击【更新许可】即可在线更新许可

![](https://cdn.nlark.com/yuque/0/2025/png/39256605/1764147693017-53817f02-282e-4aeb-aca1-c5d24fd9d96f.png)

+ 影像私包： 8.0以上版本环境搭建好之后自动带出影像管理模块

# 星瀚影像对接方案
本章节描述星瀚影像的对接方案：包含星瀚标品单据，非标品单据，非星瀚业务系统以及未购买收票产品的情况。

### 方案一：星瀚费用/应付单据（标品）+星瀚收票 <font style="color:#585A5A;">（无实物签收）</font>
+ **产品：**<font style="color:rgb(0, 0, 0);">电子影像（旗舰版）+收票管理（旗舰版）+星瀚应付/费用模块</font>
+ **适用客户：**已购买星瀚费用核算或者应付模块的客户，纸质发票或附件无需签收匹配。
+ **方案简述：**本方案为标品方案，按照本手册步骤二完成配置即可；其中单据底部的附件影像化需在开发平台给单据保存按钮添加微服务。

![画板](https://cdn.nlark.com/yuque/0/2025/jpeg/40561605/1761630388179-276ad6d9-fd75-45f0-934c-41fe2626e9e3.jpeg)

### 方案二：星瀚费用/应付单据（标品）+星瀚收票<font style="color:#585A5A;">（实物签收）</font>
+ **产品：**<font style="color:rgb(0, 0, 0);">电子影像（旗舰版）+收票管理（旗舰版）+星瀚应付/费用模块</font>
+ **适用客户：**已购买星瀚费用核算或者应付模块的客户，纸质发票或附件有专岗签收匹配。
+ **方案简述：**本方案为标品方案，按照星瀚影像初始化手册配置即可；其中单据底部的附件影像化需在开发平台给单据保存按钮添加微服务。

该方案与方案一的区别在于实物签收（匹配模式），星瀚影像标品支持收单机或星瀚影像界面上传影像匹配<font style="color:#2F8EF4;">（配置参照本手册2.3影像方案配置）</font>。

![画板](https://cdn.nlark.com/yuque/0/2025/jpeg/40561605/1761630388027-218353e5-f97f-41b6-bfd7-9d44eb851af8.jpeg)

### 方案三：星瀚其他单据（二开）+星瀚收票<font style="color:#585A5A;">（无实物签收）</font>
+ 产品：<font style="color:rgb(0, 0, 0);">电子影像（旗舰版）+收票管理（旗舰版）+星瀚ERP</font>
+ 适用客户：购买了星瀚非费用/应付模块的其他业务模块，需要单据影像化，且纸质发票附件无需签收。
+ 方案简述：与方案一的区别在于收票采集非标准功能，需单据自行二开调用收票接口实现发票附件上传及封面推送。

![画板](https://cdn.nlark.com/yuque/0/2025/jpeg/40561605/1761630432915-570e3930-6f63-4994-91a1-c8e3a6640d56.jpeg)

### 方案四：星瀚其他单据（二开）+星瀚收票<font style="color:#585A5A;">（实物签收）</font>
+ 产品：<font style="color:rgb(0, 0, 0);">电子影像（旗舰版）+收票管理（旗舰版）+星瀚ERP</font>
+ 适用客户：购买了星瀚非费用/应付模块的其他业务模块，需要单据影像化，且纸质发票附件有专岗签收。
+ 方案简述：与方案二的区别在于收票采集非标准功能，需单据自行二开调用收票接口实现发票附件上传及封面推送。实物签收依然可以用星瀚影像标品，支持收单机或星瀚影像界面上传影像匹配。

![画板](https://cdn.nlark.com/yuque/0/2025/jpeg/40561605/1761630573998-ea186f20-1900-4002-9461-ebaf274440b6.jpeg)

### 方案五：非星瀚系统+星瀚收票<font style="color:#585A5A;">（无实物签收）</font>
+ 产品：<font style="color:rgb(0, 0, 0);">电子影像（旗舰版）+收票管理（旗舰版）</font>
+ 适用客户：未购买使用星瀚ERP的客户，有购买收票产品；纸质发票附件无需签收。
+ 方案简述：
    -  影像采集：异构二开收票接口上传发票/附件与封面
    -  影像查看：异构二开调用3.04影像调阅接口
    -  影像审批：异构调用3.06接口查询影像状态已就绪后，异构系统内审核单据。

![画板](https://cdn.nlark.com/yuque/0/2025/jpeg/40561605/1761631306313-9b0a4ab1-991e-46d1-a1eb-f72a14595cce.jpeg)

### 方案六：非星瀚系统+星瀚收票<font style="color:#585A5A;">（实物签收）</font>
+ 产品：<font style="color:rgb(0, 0, 0);">电子影像（旗舰版）+收票管理（旗舰版）</font>
+ 适用客户：未购买使用星瀚ERP的客户，有购买收票产品；纸质发票附件有专岗签收匹配。
+ 方案简述：与方案五的区别在于实物签收，支持登录星瀚影像系统进行标品的签收匹配，或者异构调用匹配接口进行匹配。

![画板](https://cdn.nlark.com/yuque/0/2025/jpeg/40561605/1761632312470-1a0d6bb6-5110-4ef9-ba11-f3dc96f7ab87.jpeg)

### 方案七：非星瀚系统+无星瀚收票
+ 产品：<font style="color:rgb(0, 0, 0);">电子影像（旗舰版）</font>
+ 适用客户：只购买了星瀚影像，未购买收票产品的客户。
+ 方案简述：
    -  影像采集：调用星瀚影像的采集接口上传提交影像文件；
    -  影像查看：调用3.04影像调阅接口查看影像；
    -  影像审批：调用3.06影像状态查询接口获取影像已就绪状态后，异构系统内单据审核。

![画板](https://cdn.nlark.com/yuque/0/2025/jpeg/40561605/1761633492222-db116677-13e8-400b-acff-2083a27a5951.jpeg)
