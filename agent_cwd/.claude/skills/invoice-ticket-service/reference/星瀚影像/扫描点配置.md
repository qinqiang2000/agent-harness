## 扫描点配置功能设计理念概述
 扫描点配置功能用于维护扫描点及其可用范围。

1. **<font style="color:rgb(0, 0, 0) !important;">岗位级权限隔离</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">通过 “绑定扫描员 + 组织范围限制” 组合，实现 “财务扫描点” 仅处理财务部门单据，“行政扫描点” 仅处理办公用品类单据，杜绝越权操作。</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">支持 “全局扫描点” 配置，用于跨组织协作场景（如集团统一扫描岗），灵活平衡权限与效率。</font>
2. **<font style="color:rgb(0, 0, 0) !important;">动态协作与共享</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">同扫描点成员实时共享数据，适合多人流水线作业（如扫描 - 审核 - 提交），单据流转时效提升 30%。</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">数据共享范围严格限定在扫描点内，外部人员（如其他部门员工）无法访问，保障数据安全。</font>
3. **<font style="color:rgb(0, 0, 0) !important;">轻量化配置体验</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">扫描点新增流程极简（3 步完成），支持批量导入扫描员（如从组织架构同步），减少人工录入成本。</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">单据类型与组织范围配置支持下拉框快速选择，非技术人员可独立完成权限设置。</font>

### 扫描点配置主要操作说明
##### 新增扫描点
**步骤1：**点击【新增】按钮，打开扫描点配置窗口。

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747458045760-ed66c07f-b237-4620-a355-fe503d883016.png)  
**步骤2：**填写扫描点信息，包括扫描点名称、扫描点地点、备注等。

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747458675269-763035c7-3a76-4116-af39-97cb0373e2ee.png)

**步骤3：**可以选择扫面点人员信息，同一个扫描点下绑定的扫描员之间数据是共享的，可以互相查看操作彼此的数据。

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747458846444-1a922bc4-72b3-4d4d-88f4-ee8e9674ec2b.png)

**步骤4**：设置方案适用范围，用于限制扫描点可以使用的组织和单据类型。
