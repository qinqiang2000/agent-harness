## 封面编码规则配置功能设计理念概述
可以通过设置常量和变量段配置封面的编码规则。

**客户痛点**：

1. **<font style="color:rgb(0, 0, 0) !important;">影像分类效率低</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：封面与附件缺乏统一编码规则，人工分辨耗时易错，尤其在批量处理时易导致档案混乱。</font>
2. **<font style="color:rgb(0, 0, 0) !important;">识别精度不足</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：传统编码方式固定，无法动态匹配业务需求（如组织、日期、单据类型变化），影响系统自动分类准确性。</font>
3. **<font style="color:rgb(0, 0, 0) !important;">规则扩展性差</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：新增业务场景时需重新开发编码逻辑，无法通过参数化配置快速调整，灵活性不足。</font>

**产品价值**：

1. **<font style="color:rgb(0, 0, 0) !important;">提升影像处理效率</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">自动分类减少人工标注成本，例如扫描岗采集单据时，系统根据编码规则直接标记 “封面” 与 “附件”，审核效率提升 40%。</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">批量规则管理功能支持一次性配置多套规则（如分公司独立编码），避免重复操作。</font>
2. **<font style="color:rgb(0, 0, 0) !important;">增强数据准确性</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">编码示例实时预览防止格式错误（如年份位数不足导致 “2025” 显示为 “25”），规则校验机制确保变量取值合法（如流水号不超过最大位数）。</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">统一编码标准降低跨部门协作误差，例如财务部与档案室基于同一规则检索影像，文件定位时间缩短 50%。</font>
3. **<font style="color:rgb(0, 0, 0) !important;">适应业务变化</font>**
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">新增业务类型时（如 “合同类” 单据），只需新增编码段（如 “HT-”）并关联变量，无需系统升级，响应时效从 2 周缩短至 1 小时。</font>

### 封面编码规则配置主要操作说明
##### 新增封面编码规则
1. **步骤1：**点击【新增】按钮，进入封面编码规则编辑界面。  

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747304208580-6045ad47-3e2a-4388-9bf1-5f661dd68aff.png)

2. **步骤2：** 在“编码规则名称”字段中录入本规则的名称。  

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747304257728-4e2897b6-fc4c-44db-9a18-67739c9c1641.png)

3. **步骤3：** 在“编码内容”区域设置具体的编码结构：  
+ 设置常量：编码类型选择常量，可以设置分隔符和常量内容；

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747304464776-1e068f43-e77f-456b-bf19-fddeb150a2b7.png)

+ 如需增加更多编码段，点击【新增】按钮，新增一行编码段配置；  

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747304938763-b34e8e26-1fc0-4560-b49b-efb13ff9973d.png)

+ 设置变量： 编码类型选择“变量”，根据需要设置分隔符和变量位数。  

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747305000739-8ae5ebdb-4470-4829-99dc-4b89ad0054d2.png)

完成配置后，可通过页面中的编码示例检查编码结构是否符合预期。  
合理配置封面编码规则，有利于提高封面的识别精度，从而更好地分辨影像是“附件”还是“封面”。
