# <font style="color:rgb(0, 0, 0);">一、订单激活与分配</font>
<font style="color:rgb(0, 0, 0);">在订购了对应产品之后，必须要通过激活邮件的链接操作权益分配，保证该税号在发票云后台存在对应权限，这一步是使用发票云产品的基础。</font>

<font style="color:rgb(17, 124, 238);">PS：分配的时候需谨慎，如税号分配错误，需通过【云之家-智能审批-发票云产品使用企业变更申请】申请权益替换，开票权益替换需付费，请谨慎操作！！！</font>

<font style="color:rgb(0, 0, 0);">如未收到激活邮件，可按照如下步骤处理：</font>

<font style="color:rgb(0, 0, 0);">（1）请先联系销售人员确认订单是否已交付；</font>

<font style="color:rgb(0, 0, 0);">（2）若确认已交付，检查邮箱是否拦截了邮件。</font>

<font style="color:rgb(0, 0, 0);">（3）若以上两步排查后仍未收到邮件，请联系在线客服提供下单企业全称及订单号，重新推送激活邮件。</font>

# 三、特殊情况处理
## 权益税号变更
发票云的权益是按照<font style="color:#DF2A3F;">税号维度</font>划分的，<font style="color:rgb(0, 0, 0);">在实施配置过程中，若分配时税号填写错误，或者由于企业内部调整税号权益需要进行替换，涉及情况：</font>

+ <font style="color:rgb(0, 0, 0);">激活发票云服务时，税号填写错误，需要将错误税号调整为正确税号；</font>
+ <font style="color:rgb(0, 0, 0);">正常使用一段时间后，原税号由于注销或其他不可抗力因素无法再继续使用发票云，需要将税号权益替换至新的关联公司税号；</font>

<font style="color:rgb(0, 0, 0);">涉及以上情况，请联系金蝶内部员工通过</font>**<font style="color:rgb(0, 0, 0);">【云之家-智能审批】</font>**<font style="color:rgb(0, 0, 0);">按照实际情况提交对应的申请表单：</font>

<font style="color:rgb(0, 0, 0);">【发票云开票产品使用企业名称&税号&税盘变更申请】</font>

<font style="color:rgb(0, 0, 0);">【金蝶发票云收票产品使用企业变更流程】</font>

<font style="color:rgb(255, 0, 0);">具体表单填写规范请见表单内部说明，如您没有对应表单的申请权限，请联系当地机构协助提交申请。</font>

## 企业所属租户有误
📃**问题描述：**

分配订单第一步新增企业信息时出现报错：企业【XXXXXXXXXXXXXXXXX】在租户【苏州XXXX有限公司】下已有数据，一个企业只能绑定一个租户!如你确认一定要将该企业绑定在新租户【XXXX有限公司】下，请点击【联系售后】操作。

🎯**解决方案：**

为规范企业变更租户/租户合并流程，防范企业/租户变更导致的业务数据丢失等变更风险，现建立智能审批流程。**<font style="background-color:#E8F7CF;">流程路径：云之家-智能审批-企业变更租户/合并租户申请流程</font>**

参考：[企业变更租户/合并租户](https://jdpiaozone.yuque.com/nbklz3/msbmed/uza66kwt4ielreqf?singleDoc#)
