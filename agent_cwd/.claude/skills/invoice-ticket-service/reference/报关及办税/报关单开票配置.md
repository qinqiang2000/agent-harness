## 报关单开票配置功能设计理念概述
此功能为报关单开票规则的配置项。报关单采集完成后下推开票申请单时会根据此处的配置生成开票申请单。

在国际贸易中，报关单是进出口货物的必备文件之一。经营进出口贸易的企业，需要基于向用于向海关申报货物的详细信息，包括货物的名称、数量、价值、产地、运输方式，结合汇率日期、开票方式等信息，开具增值税发票，并提供给税务部门。

在发票开具过程中，需要完成开票项目输入、开票人民币金额计算、备注填写等一系列内容，操作费时费力，且容易出差错，随着企业业务量增大，开票工作将耗费极大的人力。

星空发票服务直连海关出口报关单数据，客户可采集指定状态、指定日期的报关单数据，采集时即可根据自定义配置的开票规则，获取汇率、计算人民币开票金额、自定义备注内容。报关单数据可以推送为开票申请单并根据自定义的规则可自动处理开票金额、备注、开票内容等信息，下推为开票申请单后提交开票。

**步骤3：**配置报关单开票配置

在采集报关单后，报关单入库或者下推生成开票申请单时相关字段取值的配置。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914221835-fe4ab881-1680-4656-981c-d6005f50f4df.png)

**① 规格型号：**可以选择使用报关单规格型号（<font style="color:rgb(55, 65, 81);">星空发票服务直连海关出口报关单数据，可以使用该报关单数据中的规格型号</font>），或者不填写报关单规格型号。

**② 数量单位：**有三种选其一

+ 法定第一个单位及数量：指的是根据海关规定和相关法律法规，对于所申报货物在计量上首要采用的单位以及对应的数量。例如，对于某些商品，海关规定其法定的首要计量单位是“千克”，那么在报关单中就需要按照“千克”来填报该商品的数量。
+ <font style="color:rgba(0, 0, 0, 0.85);">法定第二单位及数量：是指除了法定第一个单位及数量之外，海关规定的另一个用于计量和申          报货物的单位及相应的数量。例如，对于某些液体货物，法定第一单位可能是“升”，而法定第二单位可能是“千克”。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">成交单位及数量：指的是进出口货物实际交易时所采用的计量单位以及对应的货物数量。</font>

**<font style="color:rgba(0, 0, 0, 0.85);">③ 汇率日期</font>**<font style="color:rgba(0, 0, 0, 0.85);">：指的是在报关时用于确定汇率的具体日期。</font><font style="color:rgb(55, 65, 81);">支持配置是按出口日期汇率、出口当月首个工作日汇率，系统自动判断汇率日期，查询当天的汇率中间价，并根据汇率计算人民币金额，</font><font style="color:rgba(0, 0, 0, 0.85);">有两种选择：</font>

+ <font style="color:rgba(0, 0, 0, 0.85);">按出口日期汇率：指的是在出口报关时，将出口货物的外币计价金额换算为人民币所依据的汇率，是以货物实际出口的日期当天的汇率为准。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">出口当月的首个工作日汇率：指的是在进行出口报关时，用于将外币计价的出口货物金额换算为人民币所依据的汇率是出口所在月份的第一个工作日中国人民银行公布的汇率。</font>

**<font style="color:rgba(0, 0, 0, 0.85);">④ 开票方式：</font>**有两种选择：<font style="color:rgb(55, 65, 81);">支持自定义的开票方式是按人民币FOB价开票、按人民币成交价开票，结合报关单的成交方式，内置多种计算规则，计算最终人民币开票的价税合计。</font>

+ 按人民币 FOB 价开票：发票的开具金额以人民币计价，并且计价基础是货物的 FOB（Free On Board，离岸价）价格。具体来说，这表示在计算发票金额时，只考虑货物在装运港装上船之前的成本和费用，不包括将货物运输到装运港的运费、保险费以及装船后的其他费用。
+ 按人民币成交价开票：指的是以人民币来表示进出口货物实际交易达成的价格，并据此开具发票。这里的“人民币成交价”通常是买卖双方在合同中约定的最终成交价格，它包含了货物的成本、利润以及双方在交易过程中约定的各种费用和附加条件所对应的价值。

**⑤ 开票税务信息自动填充（非配置项，系统默认支持）**：系统根据报关单货物信息，自动匹配已有开票项的税务信息，自动补充税收分类编码、税率、优惠政策，并自动计算税额、不含税金额。

**步骤4**：配置发票备注配置，支持自定义备注需显示报关单信息，例如报关单号、汇率、合同号等，系统将根据配置，自动获取报关单对应信息并填充至发票备注中，无需手动填写。
