<font style="color:rgba(0, 0, 0, 0.85);">用于配置小票码的相关参数，以实现扫码开票的功能</font>

### <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
1. 客户的外部消费者/外部客户：在需要小票码开票的场景中进行消费的消费者，解决了传统开票方式中获取发票不便的痛点，通过扫描小票码可以方便地申请发票，并及时获取电子发票。
2. 客户的内部操作使用人员：财务或相关开票岗位，解决了手动配置小票码的繁琐工作，提高了工作效率，同时能够确保小票码的配置准确无误。
3. 客户部门管理人员：财务部门管理人员，解决了对小票码配置进行管理和监督的痛点，能够保证小票码的正常使用和发票开具的准确性，提升客户满意度。
4. 客户公司管理层：公司高层管理岗位，解决了对公司开票流程优化和客户体验提升的决策需求，通过小票码配置，提高了开票效率，降低了成本，增强了公司的竞争力。

### 小票码配置主要操作说明
##### 新增
**步骤一：**点击“新增”按钮，进入信息填写界面

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748596681692-eb320815-861a-4112-98f5-75eac68d8ddb.png)

**步骤二：**选择要操作的组织，其他信息会对应生成

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748596721553-b9a7f490-1a75-43a6-8be3-5190d1d56bf5.png)

**步骤三：**准确填写”限额“（必须输入正数，且最长字节不超过20个），选择”可开票类型“，并”保持“信息

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748596749135-ccc6da0e-b9b6-4833-bfeb-bdd0eb752cf3.png)

**步骤四：**商家收银/订单系统生成收银小票及开票二维码信息；商家收银或订单系统在打印出开票二维码提供给用户，用户可以通过微信扫码申请发票。

**步骤五：**用户使用微信扫码进入发票服务小程序，填写购方企业类型、抬头、收件邮箱等，提交开票。

其中扫码开票购方企业类型支持选择个人、企业或非企业单位。

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1723207207745-48a490a5-8562-49da-abb9-401afa1f66d4.png)

**步骤六：**交付电子发票到用户预留的邮箱。申请开票成功后，系统会自动交付电子发票到预留的邮箱或手机，可以下载电子发票。

**步骤七：**查看发票或加入微信卡包。发票开具成功后，可以通过二次扫码或者登录发票服务小程序中查看发票，下载发票，重发，加入微信卡包等操作。
