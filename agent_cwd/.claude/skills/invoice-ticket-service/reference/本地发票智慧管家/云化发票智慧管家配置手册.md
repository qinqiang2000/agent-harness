## 初始化配置过程
①打开购买后的产品确认邮件，复制“操作密钥”，点击“操作链接”。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685602399221-f899004b-dc85-4385-8058-7582ea05d45e.png)②输入邮件中的“操作密钥”，点击提交。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685602463110-a6b9ac91-410b-4dd1-bb45-bc3cacd418f3.png)

③在此界面可以添加或编辑租户下企业，如果无需添加租户下企业，直接点击下一步。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685602508593-b07ae53e-2329-42e2-bb85-17ef4794e51f.png)④在可分配的产品后方点击“去分配”。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685602631915-ce553685-eb2a-469f-9568-51b86977a461.png)

⑤选中需要分配权益的企业，点击下一步。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685602673558-03243a1d-8095-4a60-b7b7-8373ca58ea7a.png)

⑥分配完成后，使用微信扫描下方二维码，进入公众号。

## 公众号设置
### 添加电子税局账号
①扫码后，公众号中有如下消息弹窗，点击进入。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1684979174145-4560b85e-11d8-4c7c-9a80-3de40c85b69e.png?x-oss-process=image%2Fresize%2Cw_260%2Climit_0)

②点击进入待配置企业列表后，选择需要配置的企业。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685599651937-9584e26d-ea6d-443e-a307-675a19ef7bf2.png)

③点击新增账号。

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1685599709162-5951d296-b46e-4c10-b739-d3229b3061b5.png)

④进入新增页面，核对地区与税号无误之后，输入用户、密码、角色、登陆类型（默认选择人工登录），点击登录按钮。

登录类型详细说明：

人工登录：<font style="color:rgba(0, 0, 0, 0.85) !important;"></font>需手动验证的常规账号默认选择人工登录，输入用户名密码并点击“登录”后，需要继续完成后续获取验证码等配置，才可新增账号成功。

华盟托管号码：仅限账号是“华盟托管号码”类的账号使用，输入用户名密码并点击“登录”后，等待系统自动处理完成，即可新增账号成功。

~~【已下架】发票云登录App：适用于账号形式是手机号的账号，需要根据文档验证码登录APP使用说明下载APP并进行初始化配置后才可使用，否则无法添加账号。~~

企享云办税小号：仅限账号是“企享云办税小号”类的账号使用，输入用户名密码并点击“登录”后，等待系统自动处理完成，账号即可新增成功。使用此登录类型的，配置详见：[企享云办税小号使用说明](https://jdpiaozone.yuque.com/nbklz3/dn5ehb/qwtvytgyhe2236dm)。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1765434835325-04d19e04-a6a2-488f-ad28-baac440b31ac.png?x-oss-process=image%2Fformat%2Cwebp)

⑤点击“登录”之后，输入正确的验证码，点击“登录”。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1765435810140-8dc61349-39a6-4d5d-824b-3baa327f42ec.png?x-oss-process=image%2Fformat%2Cwebp)

## 业务操作 登录 及 认证 过程效果演示

**"已向用户150xxxxxxxx绑定的微信用户推送服务号消息，请通知该用户在金蝶发票云服务号中完成验证登录"**

### 需要扫脸认证时，会有如下提示，系统会向绑定账号微信的公众号推送一条登录验证信息：
**”已向用户150****1099绑定的微信用户推送服务号消息，请通知该用户在金蝶发票云公众号中完成认证。”**

**tips：扫脸认证是**税局的风控措施，需要使用个人所得税app或者对应地区税务app进行扫脸认证，通过后才能继续开票

**tips:**  扫脸认证**时间间隔**可以在电子税局网页版进行设置（30分钟-24小时），需要注意的是是有法人角色或财务负责人角色才能进行设置

认证过程如下：

![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1694775730790-e433ebc5-74f1-427e-863e-d1aff12b6e83.png)![](https://cdn.nlark.com/yuque/0/2023/png/34387723/1694775797466-8d279cd5-95f3-4783-b760-baa896dab051.png)
