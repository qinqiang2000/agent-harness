# 组件安装
## 组件下载方式
在发票云商户运营平台首页（[https://www.piaozone.com/](https://www.piaozone.com/)），进入<font style="color:red;">下载中心</font>界面，找到金蝶发票管理，点击下载，即可获取金蝶发票管理组件安装包。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465630747-53e93d77-1ff8-49bd-a4f7-2eb7cddf4fe2.png)

 

 

## 组件安装
①．安装包下载下来后，直接鼠标左键双击金蝶发票管理组件安装包，开始安装。

②．第①步操作完成后弹出的窗口中，点击浏览选择自己想要安装进的文件夹路径，然后点击下一步（如果不想选择，可以直接点击下一步，默认将组件安装在C盘），如下图。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465631062-701cdb99-99e7-4739-83c9-db2c529d3397.png)

 

③．自行选择是否创建组件桌面快捷方式图标后，点击下一步即可，具体见下图。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465631449-add57f7c-ecc6-4883-b0b6-5887c1eb5740.png)

 

④．最后，点击【安装】即可完成组件安装，然后在桌面点击金蝶发票管理组件的快捷方式图标即可开始使用组件。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465631884-92551674-345c-4e0b-92e6-345babeeca67.png)

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465632256-bbd10ba3-d03c-4209-a239-8e2903af17a1.png)

 

# 组件登录
      注意：<font style="color:red;">登录前需要先插上税盘，没有税盘无法进行发票下载。</font>

①．在桌面点击金蝶发票管理组件启动后，根据用户自己的税盘信息选择相应的税控设备类型，选择完成后点击下一步。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465632699-fdeae76c-b785-40cd-b967-3fe78d1df25d.png)

②．组件会自动检测出用户电脑上插入的税盘信息，用户需要确认税盘信息是否与自己的相符合。（注：电子发票服务平台(全电平台)试点地区的用户需要确认登录平台的税号是否与当前电脑上税盘的税号是否相符，否则无法下载电子发票服务平台中的发票）

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465632998-29658670-57f6-4605-9936-cc419c19df38.png)

③．输入证书口令（税盘CA密码）后，点击下一步即可登录完成。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465633299-42eda4f2-dabc-4d57-bd4f-c85f84aa3c44.png)

④．组件启动会自动查询订单权益数据：开票和收票权益。如需要正常使用开票和收票功能，请购买开票和收票对应的产品。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465633646-dbbf7917-553e-456d-9e58-ca92db7c7b7a.png)

<font style="color:red;">备注：如上图，只够买收票服务产品，开票服务产品和托管服务产品没有购买，剩余天数显示为</font><font style="color:red;">0</font>

# 系统设置
## 基础信息
获取并展示本地税盘基础信息，不允许修改：企业信息、税号、设备号、证书密码

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465634029-44598468-af3c-410c-904f-968f7e2f702d.png)

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465634435-a11bf32a-63a7-41db-9d9d-75834d0ee263.png)

## 收票设置
如果需要使用税盘收票服务，需要在此模块输入对应的信息：税盘所属城市、勾选平台密码（证书口令）、平台确认密码；如不需要用到收票服务功能，则无需填写。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465634707-2dd6faa6-7427-49d2-a14a-97811d4a3fb2.png)

 

## 网络设置
根据自己公司网络进行自行设置，默认设置为不使用代理，如果公司设置了网络代理，下拉选择代理设置，填写对应的地址、端口、用户名、密码

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465634957-b918a7df-f259-45d2-8b4c-397916aef57e.png)

## 打印设置
①．支持设置发票边距和清单边距。指定一个打印发票的打印机，如果希望将发票左移，就在左移动输入一个值；希望将发票上移，就在上移动输入一个值，单位是毫米，不可输入负数和小数。

②．点击保存，配置立即生效，可直接去ERP中发起打印

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635265-bc442408-cc50-4b30-af66-b83494a37d20.png)

# 发票下载
要在电子发票服务平台下载发票，需要<font style="color:red;">先配置电子发票服务平台的账号密码</font>（账号密码配置见4.1），否则默认是在增值税综合服务平台下载进销项发票。

如果已经配置了电子发票服务平台账号密码后，又想切换回增值税发票综合服务平台下载发票时，只需要<font style="color:red;">删除电子发票服务平台的账号密码再保存即可</font>（具体操作见3.1）。

 

## 电子发票服务平台(全电平台)发票下载
①．在金蝶发票管理组件首页点击右上角的 ![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635575-5b7d4e72-9743-494a-b808-46b04d69acfd.png) 图标，选择全电平台。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635825-d452be16-c610-4cef-b75f-185230b1caa6.png)

②．进入全电平台后，输入税局电子发票服务平台的账号和密码，点击保存。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465636208-70104533-b603-4ac0-9fd2-7e171156b584.png)

③.回到首页，在右上角 ![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635575-5b7d4e72-9743-494a-b808-46b04d69acfd.png) 图标处，选择系统设置。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465634435-a11bf32a-63a7-41db-9d9d-75834d0ee263.png)

④.在系统设置中点击进入发票下载界面，选择需要下载发票类型和开票月份，点击申请下载即可下载电子发票服务平台的全量发票查询中对应的发票。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465636675-840b5d93-49bd-49f0-b45b-412612313bb8.png)

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465637150-5b13f431-634a-4ec2-817f-9c41657c0ae6.png)

⑤.如果需要查看发票是否已下载完成，可以返回到金蝶发票管理组件首页，在右上角 ![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635575-5b7d4e72-9743-494a-b808-46b04d69acfd.png) 图标处，点击发票中心。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465637479-188b9138-6aca-4ff0-ac64-39afb2cf1852.png)

 

⑥.进入发票中心后，在进项管理 - 全票池查询中可以按条件筛选查看发票是否已经下载完成(具体见下图)。

注意：如果发票数量较多，需要等待一定时间才能查看到所有发票信息。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465637782-d5b1f8d8-87d5-434e-b079-f23eaf61177e.png)

 

## 增值税发票综合服务平台发票下载
如果此前已经配置过电子发票服务平台账号密码，是无法在增值税发票综合服务平台下载发票的，需要先在右上角的 ![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635575-5b7d4e72-9743-494a-b808-46b04d69acfd.png) 图标，点击全电平台，将配置的账号密码删除，点击保存后<font style="color:red;">重新启动金蝶发票管理组件</font>。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635825-d452be16-c610-4cef-b75f-185230b1caa6.png)

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465636208-70104533-b603-4ac0-9fd2-7e171156b584.png)

 

如果首次使用金蝶发票组件或者未配置过电子发票服务平台账号密码，默认下载增值税发票综合服务平台中的发票数据，只需按以下步骤操作即可：

①．重启金蝶发票管理组件后进入首页，在右上角 ![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635575-5b7d4e72-9743-494a-b808-46b04d69acfd.png) 图标处，选择系统设置。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465634435-a11bf32a-63a7-41db-9d9d-75834d0ee263.png)

②．系统设置中点击进入发票下载界面，选择需要下载发票类型和开票月份，点击申请下载即可下载增值税发票综合服务平台中对应的发票。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465636675-840b5d93-49bd-49f0-b45b-412612313bb8.png)

③.如果需要查看发票是否已下载完成，可以返回到金蝶发票管理组件首页，在右上角 ![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465635575-5b7d4e72-9743-494a-b808-46b04d69acfd.png) 图标处，点击发票中心。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465637479-188b9138-6aca-4ff0-ac64-39afb2cf1852.png)

 

④.进入发票中心后，在进项管理 - 全票池查询中可以按条件筛选查看发票是否已经下载完成(具体见下图)。

注意：如果发票数量较多，需要等待一定时间才能查看到所有发票信息。

![](https://cdn.nlark.com/yuque/0/2023/png/22553157/1684465637782-d5b1f8d8-87d5-434e-b079-f23eaf61177e.png)
