## 当期抵扣勾选统计功能设计理念概述
当期抵扣勾选统计功能支持直连税局查询当前税款所属期的统计表信息、生成统计表、撤销统计表，并在申报期时确认签名。

当期抵扣勾选统计清单支持展示当前税款所属期统计表所统计的发票信息。

### 当期抵扣勾选统计主要操作说明
##### 当期抵扣勾选统计
**步骤1：**查询统计表。填写核算组织，税务组织，找到需要操作的税号，点击查询统计表，获取<font style="color:rgba(0, 0, 0, 0.85);">企业当前税款所属期的统计表状态</font>。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1746696333543-4e3dbe6b-3e3a-4080-8ef3-0c7ceb4d5abb.png)  
**步骤2：**生成统计表。<font style="color:rgba(0, 0, 0, 0.85);">依据企业的发票勾选等数据，直连税局生成当前税款所属期的统计表。操作后，统计表状态若变为：已生成统计报表，可以进行确认签名。</font>

<font style="color:rgba(0, 0, 0, 0.85);">注意：</font>电子税局（RPA）通道申请生成统计表，税局界面会汇总展示每类发票张数，因此RPA统计表是直接展示税局汇总的发票情况。 乐企通道申请生成统计表，乐企只返回是否生成成功，并未返回哪些发票被汇总到统计表中，因此乐企统计表是查询星空数据汇总的发票情况。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1746696387627-52811f3c-5249-411b-8c7f-d3edd806fe96.png)

**步骤3：**确认签名。确认发票信息无误后，点击确认统计表，输入密码，确认签名。对接电子税局时，不需要密码。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1746696433288-56019b12-6c73-40e4-b636-97fc1484ad79.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1745218512238-b36e5124-fb36-4234-8944-83522edc54cc.png)

<font style="color:rgb(0,0,0);background-color:rgb(255,255,255);">注意：用户在星空做抵扣类操作（</font><font style="color:#8A8F8D;background-color:rgb(255,255,255);">发票抵扣勾选、发票不抵扣勾选、海关缴款书抵扣勾选、海关缴款书不抵扣勾选、增值税代扣代缴完税凭证抵扣勾选、增值税代扣代缴完税凭证不抵扣勾选、撤销勾选、</font><font style="background-color:rgb(255,255,255);">查询统计表、申请统计表、撤销统计表、确认统计表、</font><font style="color:#8A8F8D;background-color:rgb(255,255,255);">农产品发票类型上传确认，发票退税勾选、海关缴款书退税勾选、退税勾选统计、</font><font style="background-color:rgb(255,255,255);">变更税款所属期</font><font style="color:#8A8F8D;background-color:rgb(255,255,255);">、自动抵扣勾选配置保存方案、自动抵扣勾选配置启用方案、批量抵扣方案保存方案</font><font style="color:rgb(0,0,0);background-color:rgb(255,255,255);">）时，若判断为小规模纳税人，则阻断提交任务到税局通道，提示：</font><font style="color:rgb(0,0,0);">小规模纳税人不支持抵扣类勾选，可根据企业在税局的信息，在星空发票服务企业基础资料修改。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1746696452776-40e25f13-75ac-4739-9c61-da1da9513d78.png)

##### 查看当期抵扣勾选发票清单
点击切换至当期抵扣勾选统计清单页签，可以查看完成统计签名后的发票，支持查看发票详细信息，支持对本期确认的发票生成凭证以及导出。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1745218823060-6c36cd73-7cda-4c9e-b06b-7704cfac0bcb.png)

##### 撤销统计表
直连税局撤销当前税款所属期的统计表。撤销成功后，返回申请统计，可以重新查询统计表，也可以继续勾选发票然后再次生成统计。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1746696497190-d35867bd-3b33-403f-9da7-cca426356611.png)

##### 变更税款所属期
（仅乐企通道支持使用，电子税局（RPA）通道不显示此按钮，电子税局（RPA）暂不支持此功能，若有变更属期需求，请上税局操作，再回到星空查询统计表。）

支持切换至上一属期，以便逐期回退至未申报的属期进行抵扣勾选。

变更时需要根据实际情况勾选：【是否保留原勾选记录】。即是否保留变更后上一属期发票的勾选状态，如不保留则上一属期已勾选发票则会变为未勾选状态。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1746696526440-0909dc71-d8ae-4c72-862c-43d229eb8733.png)

##### 更新发票
<font style="color:rgb(0, 0, 0);">仅乐企通道支持使用。</font>

<font style="color:rgb(0, 0, 0);">部分乐企税号出现以下场景：勾选发票后乐企返回成功，但后面电子税局该发票状态为</font><font style="color:rgba(0, 0, 0, 0.85);">“未勾选”</font><font style="color:rgb(0, 0, 0);">，</font><font style="color:rgba(0, 0, 0, 0.85);">此时可通过本功能进行补偿。</font>

<font style="color:rgb(0, 0, 0);">点击“更新发票”，系统将通过乐企查询发票用途信息接口，对当期已勾选或已认证的发票更新勾选状态、将发票抵扣税期保存至星空票池、取</font><font style="color:rgb(0, 0, 0);">乐企用途信息更新全票池的发票用途。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1749539327446-30d54e8b-a502-48fa-b81a-df63abf3856b.png)
