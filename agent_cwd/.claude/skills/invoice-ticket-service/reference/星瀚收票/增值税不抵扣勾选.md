**产品勾选逻辑如下**：

用户在星空发票服务勾选发票，或是税局用票日志列表点击处理失败的抵扣类勾选任务重试时，执行以下全部步骤：

    1. 读取税号的企业基础资料的税局收票通道类型，为乐企或电子税局进行以下步骤。
    2. 进行发票信息和统计表的校验：
        1. 是否小规模纳税人：根据发票的购方税号，读取星空发票服务-基础资料-企业信息的企业性质，如果是小规模纳税人则勾选批次为处理失败，失败描述：小规模纳税人不支持抵扣类勾选，可根据企业在税局的信息，在星空发票服务企业基础资料修改。
        2. 是否可勾选的发票类型：除以下发票类型，进行其他发票类型勾选的，勾选批次为处理失败，失败描述：包含不可勾选的发票类型。
                    1. 电子专用发票
                    2. 纸质专用发票
                    3. 通行费电子发票
                    4. 数电票（增值税专用发票）
                    5. 数电票（航空运输电子客票行程单）
                    6. 数电票（铁路电子客票）
                    7. 机动车销售发票
                    8. 纸质普通发票
                    9. 电子普通发票
                    10. 数电票（普通发票）
                    11. 数电发票（机动车销售统一发票）
                    12. 增值税代扣代缴完税凭证
                    13. 海关进口增值税海关缴款书
        3. 开具金额是否大于0：不含税金额必须大于0，否则勾选批次为处理失败，失败描述：开票金额必须大于0，请下载更新发票。
        4. 是否可勾选的发票状态：只允许勾选发票状态为“正常”和“部分红冲”的发票，否则勾选批次为处理失败，失败描述：只允许勾选发票状态为“正常”和“部分红冲”的发票，请下载更新发票。
        5. 是否异常凭证：只允许勾选发票风险等级为空或正常的发票，否则勾选批次为处理失败，失败描述：只允许勾选发票状态为“正常”的发票，请下载更新发票。
        6. 统计表是否已生成：星空缓存的税号当期统计表状态为已生成，则勾选批次为处理失败，失败描述：当前税款所属期统计表已生成，请撤销统计再勾选，或使用【当期抵扣勾选统计】查询更新统计。
    3. 校验通过，检测登录。
    4. 未登录且是人工登录的账号弹窗提示登录，不登录跳过统计表查询和统计表状态校验。
    5. 登录成功或自动登录的，查询企业统计表。
    6. 查询统计表成功，星空缓存的税号当期统计表状态30分钟。
    7. 校验统计表是否已生成：星空缓存的税号当期统计表状态为已生成，则勾选批次为处理失败，失败描述：当前税款所属期统计表已生成，请撤销统计再勾选，或使用【当期抵扣勾选统计】查询更新统计。
    8. 提交乐企或RPA通道勾选。

勾选任务失败的，在【税局用票日志列表】选择状态“处理失败”的任务重试，需要重复以上流程的校验。

**引入清单勾选规则如下：**

+ 1） 提交清单信息是否完整校验：导入的清单Excel必填信息不完整，在发票引入结果中提示必填字段缺失。
+ 2）导入EXCEL清单含多行重复的发票：多行发票的发票代码和发票号码一致，或缴款书号码相同视为重复）在导入结果提示失败原因“存在重复行，请检查并删除重复行”。
+ 3） 发票是否采集至票池校验：发票在全票池中，根据发票号码、发票代码、海关缴款书号码查询全票池，如果发票未在全票池中，在发票引入结果中提示发票不存在，请先从税局下载或是上传。
+ 4） 发票勾选状态是否合规校验：清单选择撤销勾选的发票，若在全票池中发票状态不是已勾选，则在发票引入结果中提示发票勾选状态不是已勾选，不能撤销勾选，请先从税局下载更新。
+ 5） 发票勾选状态是否合规校验：清单选择勾选的发票，若在全票池中发票状态不是未勾选或预勾选，则在发票引入结果中提示发票勾选状态不是未勾选或预勾选，不能勾选，请先从税局下载更新。
+ 6） 全票池中的发票数据完整性校验：不能满足通道勾选或撤销勾选的必填字段，在发票引入结果中提示发票数据不完整，不能勾选，请先从税局下载更新。
+ 7） 全票池中的发票类型类型校验：不是可勾选的发票类型，在发票引入结果提示<font style="color:rgb(0,0,0);">不可勾选的发票类型。</font>
+ 8） 开票金额大于0校验：上述发票类型，发票校验开票金额（不含税金额），海关缴款书不校验；开票金额不满足要求，发票引入结果中提示<font style="color:rgb(0,0,0);">开票金额必须大于0，请下载更新发票。</font>
+ 9） 发票状态校验：发票抵扣勾选校验发票状态，只能勾选发票状态为正常或部分红冲的发票，缴款书不校验；不满足条件的发票，发票引入结果中提示发票状态异常、提示<font style="color:rgb(0,0,0);">只允许勾选发票状态为“正常”和“部分红冲”的发票，请下载更新发票。</font>
+ 10） 发票风险等级校验：发票风险等级为正常才能勾选，异常发票在发票引入结果中提示<font style="color:rgb(0,0,0);">只允许勾选发票状态为“正常”的发票，请下载更新发票。</font>
+ 11） 小规模纳税人校验：发票核算组织为小规模纳税人生成的异步勾选任务，直接置为处理失败，失败原因：<font style="color:rgb(0,0,0);">小规模纳税人不支持抵扣类勾选，可根据企业在税局的信息，在星空发票服务企业基础资料修改。</font>
+ <font style="color:rgb(0,0,0);">12） 统计表是否已生成校验：星空缓存的税号当期统计表状态为已生成，则勾选批次为处理失败，失败描述：当前税款所属期统计表已生成，请撤销统计再勾选，或使用【当期抵扣勾选统计】查询更新统计</font>
+ <font style="color:rgb(0,0,0);">13）开票日期校验：勾选的发票开票日期或填开日期大于当期税期的月份的最后一天，发票不能勾选，用票台账：提示未到期发票不能勾选。</font>
