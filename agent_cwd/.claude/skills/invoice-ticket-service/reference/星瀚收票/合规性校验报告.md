## 合规性校验报告功能设计理念概述
1. **功能概述**

合规性校验报告，对当前组织一年内的发票校验情况进行统计分析（允许多选组织），并做图表展示。采集发票时，被“严格管控”配置项标记为不合规的发票会被记录在报表中。统计维度包含按年统计的不合规数量、原因分布、票种排行；按月统计的合规性分析以及合规趋势图等。此页面帮助企业多维度监控和分析发票合规性，深入理解发票异常的原因分布，提高整体的财务管理效率和发票的合规水平。

2. **<font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>**

<font style="color:rgba(0, 0, 0, 0.85);">①内部操作使用人员：财务或相关税务岗位，解决了对发票合规性进行校验和统计的痛点，能够通过该操作界面，快速获取发票的校验情况，包括校验发票总量、拦截不合规发票总量等信息，提高了工作效率。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">②部门管理人员：财务部门管理人员，解决了对发票合规性进行监控和管理的痛点，能够通过合规性校验报告，多维度地了解发票的合规情况，包括月度合规情况、不合规原因分布等，及时发现问题并采取相应措施，保证发票的合规性。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">③公司管理层：公司高层管理岗位，解决了对公司发票合规性进行整体把握和决策的痛点，能够通过该报告，了解公司发票合规性的整体情况，为公司的财务管理和决策提供数据支持，降低公司的税务风险。</font>
