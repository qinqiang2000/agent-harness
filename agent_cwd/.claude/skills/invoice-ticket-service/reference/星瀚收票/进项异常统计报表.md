## 进项异常统计报表功能设计理念概述
1. **功能概述**

进项异常统计看板，对当前组织的历史发票进行风险统计分析（允许多选组织），并作图表展示。可以多维度监测风险发票的数量和金额，例如超期未入账发票、已入账未抵扣发票等，并支持进一步查看的风险发票详情。还支持对状态异常及不满足合规性管理指标的发票进行统计。

此看板能够帮助企业监控发票状态，及时发现潜在的财务风险，帮助制定有效的风险管理策略。

2. ******<font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>**

<font style="color:rgba(0, 0, 0, 0.85);">①内部操作使用人员：财务或相关税务岗位，解决了及时查询和统计进项异常发票的痛点，能够通过该操作界面，快速获取超期未入账发票、已入账未抵扣发票等进项异常统计信息，提高了工作效率和数据准确性。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">②部门管理人员：财务部门管理人员，解决了对进项发票进行风险监控和管理的痛点，能够通过进项异常统计报表，多维度地了解进项发票的异常情况，及时发现潜在的财务风险，并采取相应的措施进行处理，保证公司的财务稳定。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">③公司管理层：公司高层管理岗位，解决了对公司财务状况进行整体把握和决策的痛点，能够通过该报表，了解公司进项发票的异常情况及风险分布，为公司的财务管理和决策提供数据支持，制定有效的风险管理策略，降低公司的财务风险。</font>
