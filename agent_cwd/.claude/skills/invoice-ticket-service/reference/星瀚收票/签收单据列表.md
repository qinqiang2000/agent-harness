## 签收单据列表功能设计理念概述
**<font style="color:rgb(0, 0, 0);">用户痛点：</font>**

1. <font style="color:rgba(0, 0, 0, 0.85) !important;">传统方式下，影像采集、调阅流程繁琐，单据状态异常时处理困难，如提交失败、数据不同步等问题缺乏便捷的解决途径。</font>
2. <font style="color:rgba(0, 0, 0, 0.85) !important;">无法在不退扫单据的情况下对影像文件进行修改，且多场景下的影像下载需求（如按类型合并、批量导出）难以满足。</font>
3. <font style="color:rgba(0, 0, 0, 0.85) !important;">单据操作历史和状态流转缺乏可视化记录，问题定位和追溯成本较高。</font>
4. <font style="color:rgba(0, 0, 0, 0.85) !important;">面对大量单据时，缺乏高效的批量重提、同步、下载等功能，影响业务处理效率。</font>

**产品价值：**

1. <font style="color:rgb(0, 0, 0) !important;">灵活的影像处理能力</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">支持按采集渠道、文件格式、类型、介质等 5 种方式合并影像，满足财务、归档等场景的差异化需求。</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">允许在单据已提交 / 审核通过状态下补改影像，且不影响原有匹配关系（发票匹配成功后不可删除 / 替换）。</font>
2. <font style="color:rgb(0, 0, 0) !important;">高效的批量操作体验</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">支持多选单据进行重提、同步、下载，减少重复操作。</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">导出压缩包自动按单据编号分文件夹，文件命名规则清晰（含编号、时间），便于管理。</font>
3. <font style="color:rgb(0, 0, 0) !important;">全流程可视化追溯</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">操作日志详细记录状态流转和操作细节，辅助快速定位系统异常或人为操作问题。</font>
4. <font style="color:rgb(0, 0, 0) !important;">智能化辅助功能</font>
    - <font style="color:rgba(0, 0, 0, 0.85) !important;">影像文件自动分类（发票 / 附件），减少人工标注成本；调阅界面支持 “预览原件” 等个性化设置。</font>

### 签收单据列表主要操作说明
##### 影像采集：为影像单据采集影像
若影像单据为“待采集”状态的，可以点击影像单据编号进入影像采集界面采集影像文件。详细操作可见：[https://jdpiaozone.yuque.com/nbklz3/grvkab/beimwsd7u8dqp5bb](https://jdpiaozone.yuque.com/nbklz3/grvkab/beimwsd7u8dqp5bb)

##### 影像调阅/查看：调阅或查看影像单据详情  
已经采集影像文件的，点击影像单据编号进入影像调阅界面查看单个影像；或勾选多个影像编号，点击“查看影像”按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1766972396813-ae8a05bc-7a16-4fa0-abb4-a7bd3ca56359.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747207749185-60009735-a31a-4994-9c5f-21a56f40d608.png)

影像调阅界面有以下按钮：

①点击“查看签收结果”：打开影像匹配列表，可以查看该批次提交的影像的匹配结果。

②点击“补改影像”：在不退扫单据的情况下，被授权的用户可以通过该按钮对影像文件进行编辑修改。详见第4点。

③点击“试图设置”：用于调整影像调阅界面的展示内容，可以调整是否展示影像信息、是否展示单据汇总信息、预览是否查看原件。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747208680090-60df4522-7c95-4150-8c19-8b9e2ce1f424.png)

④点击”更多“下载影像，普通权限用户可以在影像调阅界面下载影像文件至本地，无需至签收单据列表。

+ 点击“更多-下载当前影像“：只下载当前一份影像文件到本地，不生成压缩包，直接下载文件。
+ 点击“更多-下载当前单据影像“：以压缩包形式下载选中单据的所有影像文件（封面+发票+附件），压缩包名称为影像单据编号+单据影像下载+时间。
+ 点击“更多-下载本批次影像“：以压缩包形式下载本批次调阅的所有单据的影像文件，压缩包名称为：下载当前调阅批次影像+时间。调阅多个影像单据时，压缩包内将依据单据编号拆分形成独立的子文件夹，每个子文件夹中包含对应单据的全部影像文件。
+ 点击”合并影像导出“：以压缩包形式按不同方式合并下载选中单据的所有影像文件，支持以下合并方式。
    - 支持按影像文件采集渠道合并（相同采集渠道的多个文件合并为一个文件）；
    - 按影像文件源文件格式合并（相同文件格式的多个文件合并为一个文件）；
    - 按影像文件分类合并（相同文件类型的多个文件合并为一个文件）;
    - 按影像文件原始介质合并（原始文件存在介质相同的多个文件合并为一个文件（纸、电））；
    - 整单合并导出（将单据下所有影像文件合并为一个文件导出）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747210408242-f2907b71-576d-4f74-8c83-0eb5ddb2a10a.png)

V8.0.4更新：调阅影像时，支持缓存手工调整的图片缩放比例，切换图片时也会保持该比例展示。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1766720693313-4959e3f7-0429-4f49-9157-4b9d1ae09666.png)

##### 手工重提ERP：手工重新提交影像单据至ERP
影像采集完成点击提交后，<font style="color:rgba(0, 0, 0, 0.85) !important;">若因异常导致提交失败</font>，用户可以选中失败的单据点击”手工重提ERP“，重新发起提交流程。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747211007471-1d8ecd4c-5643-4d07-96e5-557bd9a9f0cf.png)

仅状态为【提交ERP失败、影像已就绪】且已知业务系统来源的影像单据才能使用此功能。

支持多选单据进行重提。点击重提后，可以在界面上查看重提结果，重提失败的，需要查看日志。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747211877341-dc6902ce-7710-444b-be12-6a8b408ed66f.png)

##### 同步提单人数据：手工发起同步收票数据至影像系统  
<font style="color:rgba(0, 0, 0, 0.85);">在配置同步模式或匹配模式（电票默认同步）时，可能出现同步收票数据生成影像失败，或单据被驳回但影像没有退回重扫的情况。此时，用户可点击 “同步提单人数据” 按钮，重新从收票同步数据，实现影像数据的更新。</font>

支持多选影像单据点击同步。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747213753068-70462eca-059f-4048-b664-c37fc91d5e80.png)

##### 补改单据：对已提交的影像进行补改
<font style="color:rgba(0, 0, 0, 0.85) !important;">当用户在影像采集完成并提交成功后，于后续业务处理中产生新的影像文件，但需要在不退扫单据的前提下，对影像文件进行修改时，可使用该功能。</font>

<font style="color:rgba(0, 0, 0, 0.85) !important;">点击 “补改单据” 按钮进入影像补改页面，每次仅支持单笔单据操作，不允许多选。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747215729280-d0595479-f02a-4105-854c-5e7311a00ff4.png)

使用影像补改页面的功能编辑影像文件，编辑完成后，点击”保存“，本次修改才可生效。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747216095978-ec0d9aac-bf48-4e1d-b4d5-c67fe16275d4.png)

注意：

+ 只有提交 ERP 成功、审核通过或已入账状态的单据，方可进行补改操作。
+ 补改采集的文件将由系统自动识别分类（发票或附件），若分类有误，可使用“设为发票” 或 “设为附件” 功能手动调整。
+ 单据封面文件仅支持替换，不支持删除；签收匹配成功（已有匹配关系）的发票不可替换、删除。
+ 新补充的文件会自动排列在原有影像文件之后，补改页面无法调整影像顺序，也不支持插入影像操作。
+ 补改上传的文件仅进行识别，不参与签收匹配流程。

##### 下载影像：下载影像文件至本地  
可以使用此功能下载影像文件至本地。支持多选单据下载。

当选中单个单据时，以压缩包形式下载选中单据的所有影像文件（封面+发票+附件），压缩包名称为影像单据导出+时间。

当选中多个单据时，以压缩包形式下载所有选中单据的所有影像文件（封面+发票+附件），压缩包名称为影像单据编号。压缩包内将依据单据编号拆分形成独立的子文件夹，每个子文件夹中包含对应单据的全部影像文件。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747216675742-7ae8462c-519f-45ec-972d-fcd04acdba5a.png)

##### 导出单据列表：导出单据列表数据至本地 
<font style="color:rgb(0, 0, 0);">用于以EXCEL形式下载影像单据列表的数据至本地。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747217419508-6e6ec18e-4bc8-4d12-a88a-ae5e547a84de.png)

可以点击列表右侧设置功能，使用隐藏列来控制此列是否导出。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747217467839-1350fe2c-92fc-471e-bee1-0e04bf53b478.png)

点击单据右侧”详情“，进入单据日志列表，查看当前单据的操作日志。

用户可以通过该页面查看单据的历史操作记录和状态流转情况，便于问题追溯和定位。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1747218036866-ecda2560-c9b7-44cc-a04d-aab612bd3ef6.png)
