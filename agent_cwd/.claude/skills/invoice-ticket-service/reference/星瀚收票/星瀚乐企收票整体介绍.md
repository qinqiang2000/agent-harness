

# <font style="color:rgb(38, 38, 38);">产品概述</font>
## 产品介绍
乐企是<font style="color:rgb(51, 51, 51);">税务机关合规向符合资质的纳税人开放规则和数据的通道，税务机关提供上传、清分、下载等功能，帮助开受方完成发票交付和用票方上传用票数据。</font>

星瀚发票云收票管理支持大型和超大型企业通过乐企平台接口，已对接使用进销项发票归集、进项发票抵扣认证。

企业可在星瀚界面操作已对接了乐企能力接口的功能，实现发票归集，星瀚费用报销、应付等模块可直接查询发票结构化数据；支持在星瀚抵扣认证，星瀚税务云可查询发票抵扣数据生成申报数据；也可通过星瀚收票的外部对接接口嵌入本单位信息化系统中（如：销售、报销、财务等），实现发票自动化归集，便于企业完成发票抵扣等操作，有效提升财税业务办理效率。

## 业务场景
1、税局发票归集：定时归集企业开出或收到的发票；

2、发票和海关缴款书抵扣认证：企业归集到可认证抵扣的发票，税务人员将发票提交税局抵扣，并获取到成功抵扣的数据，准备纳税申报。

# <font style="color:rgb(38, 38, 38);">亮点特性</font>
**乐企快速升级切换**

标品对接乐企沙箱测试，可对星瀚收票应用到的能力和服务进行测试，完成乐企上线要求的测试用例，辅助企业完成乐企上线流程，通过简单配置完成税控通道和电子税局RPA快速切换到乐企通道。



**无缝衔接业务及通道，现有星瀚发票云用票场景，全部无缝衔接乐企通道**

乐企只是通道，缺乏业票或财票协同能力，星瀚收票发票归集、报销入账、抵扣认证兼容不同的通道，升级切换乐企，对现有业务流程无影响。



**进销项自动归集**

全量发票同步设置支持按组织配置使用自动归集功能，支持乐企的全量、增量、用途状态和风险状态归集；企业也会采用该功能下载全票面信息，建立企业发票池，做数据分析，校验收到的发票真伪。



**根据业务条件自动抵扣**

支持按组织自定义抵扣方案，定时任务根据不同的业务条件，如发票报销入账状态，或是预置税负率，自动筛选符合企业抵扣条件的发票提交乐企抵扣勾选。



**风险预警和统计**

从乐企获取发票的最新状态，规避发票被供应商异常红冲、作废、失控、异常，及时发现财税风险；异常发票报销入账合规性校验拦截，形成进项异常统计报表。
