## 预勾选确认勾选功能设计理念概述
### <font style="color:rgba(0, 0, 0, 0.85);">功能概述</font>
此功能与”预勾选“功能连用，支持统计核算组织下发票的预勾选情况，并按照发票抵扣用途分别展示，提交勾选确认可以直连税局进行批量勾选。

### <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户的内部操作使用人员：财务或相关税务岗位，解决了对预勾选发票进行确认勾选的痛点，能够通过该操作界面，方便地查询预勾选信息，并将预勾选发票提交直连税局进行批量勾选，提高了发票抵扣的操作效率。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户部门管理人员：财务部门管理人员，解决了对发票抵扣进行统一管理和监控的痛点，能够通过该功能，清晰地了解核算组织下发票的预勾选情况，按照发票抵扣用途进行分类展示，确保发票抵扣的合规性和准确性。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户公司管理层：公司高层管理岗位，解决了对公司税务成本和合规性进行整体把握和决策的痛点，能够通过预勾选确认勾选功能，合理控制税务成本，降低税务风险，为公司的财务管理提供决策支持。</font>

### 预勾选确认勾选主要操作说明
#### **提交预勾选**
**步骤1：**选择要操作核算组织和税务组织，点击“查询”按钮，同步预勾选信息。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748591654259-6c16d181-41a0-4c05-a35e-9a9bbab54ec9.png)

**步骤2：**点击“提交勾选确认”，将预勾选提交。此处是直连税局进行勾选。若为RPA方式对接电子税局，会弹出登录。若只是提交税局勾选，不会进行勾选确认。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748591670584-7d5c64f7-3236-44ae-88d3-0923c96e6745.png)

<font style="color:rgb(0,0,0);background-color:rgb(255,255,255);">注意：用户在星空做预勾选确认勾选时，若判断为小规模纳税人，则阻断提交任务到税局通道，提示：</font><font style="color:rgb(0,0,0);">小规模纳税人不支持抵扣类勾选，可根据企业在税局的信息，在星空发票服务企业基础资料修改。（</font><font style="color:rgb(0,0,0);background-color:rgb(255,255,255);">考虑到预勾选是对全票池发票做核算组织筛选，预勾选的核算组织可能不是最终抵扣勾选的税号，因此预勾选操作不会校验小规模纳税人，预勾选确认勾选时再进行校验。</font><font style="color:rgb(0,0,0);">）</font>
