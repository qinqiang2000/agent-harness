## 收票助手配置操作路径及所属模块编码
操作路径：【发票服务】→【收票管理】→【基础设置】→【收票助手配置】

所属模块编码：F18018

所属模块名称：发票助手配置

## 收票助手配置功能设计理念概述
支持指定单据类型对星空收票助手的显示界面（PC端、移动端）进行配置，配置可采集类型是否展示。例如设置PC端发票助手不支持使用EXCEL导入。

### 收票助手配置主要操作说明
**步骤1：**新增收票助手配置方案

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593934738-6f810fc5-33fc-4444-be4f-32098fa666a5.png)

  
**步骤2：**配置基础信息及适用单据类型

注：<font style="color:#000000;">如“适用单据类型”中选择“指定单据类型”，需在【发票服务】-->【基础资料】-->【收票资料维护】-->【单据类型管理】中先增加对应的单据类型，注意需取到对应单据的单据编码填写到采集助手方案编码（向对应模块的产品索要，如费用报销单的单据编号为er_dailyreimbursebill，这里不能随意进行编码）</font>

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593949778-10ae8f5b-73a1-4eef-9d49-5242d1f4ed67.png)

**步骤3：**配置PC端报账助手可采集类型

采集类型有国内发票、海外形式发票和附件，可以根据自身收票方式的需要选择展示相应类型的收票方式的入口

+ 展示：对应单据进入发票助手该采集方式显示。
+ 不展示：对应单据进入发票助手该采集方式隐藏。
+ 按权限：按系统权限的配置进行控制。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593963824-76d6d49d-f93e-4951-a6a9-c22c51db3d3a.png)

配置好后可在点击“上传发票和附件”按钮后展示出刚刚选择展示的上传发票类型入口，如下图。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593978736-7088fb0a-ff1a-4508-b734-e062f62f4bb8.png)

**步骤4：**配置移动端可采集类型

发票助手支持云之家（H5码）和微信（小程序码）两种移动端，同一方案下只有一种移动端起作用，具体可参考下面示意图。

云之家可以自定义是否展示相应类型的收票方式入口，例如拍照选择、扫码采集、手动添加、短信取票、邮箱取票、票夹渠道是否展示：

+ 展示：对应单据进入发票助手该采集方式显示。
+ 不展示：对应单据进入发票助手该采集方式隐藏。
+ 按权限：按系统权限的配置进行控制。

微信不支持自定义配置。![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593994778-8f205d38-380b-41d5-b928-d67c68299c0b.png)
