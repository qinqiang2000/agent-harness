## 预勾选功能设计理念概述
### <font style="color:rgba(0, 0, 0, 0.85);">功能概述</font>
预勾选包含统计图和预勾选列表功能。企业可以根据发票业务状态批量选择发票，结合实际需求对增值税专用发票、数电票（增值税专用发票）、数电票（普通发票）、数电票（航空电子客票）、海关缴款书、机动车销售统一发票、旅客运输发票等发票提交预勾选或撤销预勾选。

常用场景：此功能适用于分支机构较多的企业。根据核算组织的维度，由实际业务发生的部门或机构对可以勾选的发票打上预勾选标识，此功能不连税局。预勾选完成后，可以由总部税务人员（共享税管员）一键确认预勾选（使用”预勾选确认勾选“功能，直连税局进行勾选）。此种处理方式将实际税局勾选权限交由总部税务人员统一处理，更加安全高效。

### <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
1. <font style="color:rgba(0, 0, 0, 0.85);">内部操作使用人员：财务或相关税务岗位，解决了对发票进行预勾选和撤销预勾选的操作需求，能够通过列表数据提交、Excel导入提交、扫描仪提交等方式，方便地对发票进行预勾选处理，提高了工作效率。</font>
2. <font style="color:rgba(0, 0, 0, 0.85);">部门管理人员：财务部门管理人员，解决了对分支机构较多的企业进行发票管理的痛点，能够通过预勾选功能，由实际业务发生的部门或机构对发票进行预勾选标识，再由总部税务人员一键确认预勾选，实现了对发票勾选权限的统一管理，更加安全高效。</font>
3. <font style="color:rgba(0, 0, 0, 0.85);">公司管理层：公司高层管理岗位，解决了对公司税务管理进行整体把控和决策的痛点，能够通过预勾选功能，确保发票抵扣的合规性和准确性，优化税务成本，为公司的财务管理提供支持。</font>

### 预勾选主要操作说明
#### 提交预勾选
预勾选可通过三种方式实现：列表数据提交、Excel导入提交、扫描仪提交

##### 列表数据提交
在发票列表可勾选发票，可以编辑发票抵扣用途为抵扣和不抵扣，有效税额默认全额抵扣，您可根据实际情况自行调整有效税额 ；统计图中当期预抵扣发票动态展示当前所有可抵扣的发票，柱状图当期预抵扣进项税额为当前所有可抵扣发票的总税额；点击【列表数据提交】，所有发票状态变成预勾选。

![](https://cdn.nlark.com/yuque/0/2023/png/25809591/1691652805809-179f05aa-c5e5-4229-b102-222de9a8a098.png)

##### Excel导入提交
下载Excel预勾选模板，按照发票分类分别填写发票信息，在【提交预勾选】中选择【Excel导入提交】，选择Excel预勾选文件提交，等待所有发票预勾选完成系统自动下载预勾选文件导入结果。可打开导入结果文件查看各发票具体预勾选情况。

##### 扫描仪提交
首次使用扫描仪录入会弹窗提示安装JsScanner.msi，按照提示安装。

将发票放入扫描仪后点击扫描仪预勾选，扫描完成后会弹窗【扫描仪预勾选结果列表】显示预勾选情况，点击【编辑】可以查看发票信息，点击【完成】后关闭【扫描仪预勾选结果列表】返回首页可查看预勾选发票。

![](https://cdn.nlark.com/yuque/0/2023/png/25809591/1691652806285-c98b8d12-6912-4c2f-ab8d-39474be3074f.png)

#### 撤销预勾选
发票列表勾选预勾选状态的发票，再点击【撤销预勾选】，发票状态变成未勾选。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748590996046-de7a7365-9bbb-484b-b02c-a19b112709ca.png)
