## 进项数据看板功能设计理念概述
1. 功能概述

进项数据看板，对当前组织的进项情况进行统计分析（允许多选组织）。支持分别从开票时间、采集时间、认证时间三方面筛选所属范围内的进项数据，在发票数量、种类、抵扣税额、采集方式等方面展开统计。

2. <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户的内部操作使用人员：财务或相关税务岗位，解决了快速查看进项数据的痛点，能够通过进项数据看板，直观地了解组织的进项发票情况，包括发票数量、种类、抵扣税额等信息，提高了工作效率。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户部门管理人员：财务部门管理人员，解决了对进项数据进行监控和分析的痛点，能够通过看板中的发票使用状态分布、可抵扣税额分布等信息，及时发现问题并采取相应措施，保证进项数据的准确性和合规性。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户公司管理层：公司高层管理岗位，解决了对公司进项情况进行整体把握和决策的痛点，能够通过进项数据看板，了解公司的进项状况，为公司的财务管理和决策提供数据支持，优化资源配置，降低成本。</font>
