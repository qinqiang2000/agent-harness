# 三、ERP配置
## 参数设置
**<font style="color:#DF2A3F;">是否必需：是</font>**

用管理员账号登入星空，在基础管理——公共设置——参数设置——任务共享中心，填入以下信息：

1. 影像系统的接口地址：
    - 公有云演示环境：http://kdimage-demo.piaozone.com/；
    - 公有云正式环境：http://api.piaozone.com/；
    - 私有部署：http://ip:端口/api/（私有部署时设置的服务器IP地址及端口号）
2. 接口实现类：Kingdee.BOS.Workflow.ImageCenter.FPY.ImageInteractPlugIn,Kingdee.BOS.Workflow.ImageCenter.FPY
3. 客户端编码、口令：对应激活邮件中的授权client_id、client_sercret

![](https://cdn.nlark.com/yuque/0/2021/png/21686286/1622542711588-b8208617-1cae-4999-9e0e-dbd19cee2779.png)

## 企业授权配置
**<font style="color:#DF2A3F;">是否必需：是</font>**

功**能说明：**

添加使用影像系统租户下的组织授权，用于影像系统登录、外部获取请求token认证，租户初始化，从收票获取发票数据，数据的新增和删除由影像系统运维人员操作，影像系统用户只有查看的权限。



点击新增，在弹出的窗口中录入：

企业税号、名称、授权客户ID、secret、encrypt_key；

注意<font style="color:#DF2A3F;">信息要和收票的授权保持一致</font>，并且client ID 信息部门一定是存在，否则保存不成功；

录入完成后点击确定，完成企业授权新增
