## 收单机盒号列表功能设计理念概述
收单机盒号列表用于统一展示收单机中收单盒的状态及相关信息（如盒号、状态、收单起止时间、绑定设备号、单据数等）。

 设计目标与价值：

+ <font style="color:rgba(0, 0, 0, 0.85) !important;">集中列表查询替代多系统切换，设备信息获取时间从 10 分钟 / 次缩短至 1 分钟内。</font>
+ <font style="color:rgba(0, 0, 0, 0.85) !important;">批量导入功能将新设备上线周期从 2 天缩短至 1 小时，快速响应业务扩张需求。</font>

### 收单机盒号列表主要操作说明
##### 收单机盒号列表
1. 进入【收单机盒号列表】页面。
2. 根据需要设置查询条件（如收单盒状态、时间区间等）。
3. 在列表中查看收单盒的各类数据，包括盒号、状态、收单开始时间、收单结束时间、绑定设备编号、绑定单据数、数据采集渠道等。  

![](https://cdn.nlark.com/yuque/0/2025/png/35533825/1747462500979-2d0f201f-f28f-44f7-9194-64b3f303da18.png)

#####  通过 Excel 引入收单机盒号数据  
1. 在【收单机盒号列表】页面，点击【引入数据】按钮。
2. 在弹出的引入向导中，点击下载模板按钮，获取标准 Excel 模板。
3. 根据模板要求在本地 Excel 中填写或整理收单盒相关数据。
4. 在引入向导中选择已填写好的 Excel 文件并上传。
5. 根据系统提示完成校验与导入，导入成功后，在列表中即可查看新引入的数据。
