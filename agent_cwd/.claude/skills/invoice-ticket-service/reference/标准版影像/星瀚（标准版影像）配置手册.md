# 影像问题排查指引
[影像](https://jdpiaozone.yuque.com/nbklz3/tadboa/od7shz4aughnlbh6?singleDoc#FzNF)

1. **单据查看影像提示“当前单据无需上传影像”：**

       请联系业务单据的老师排查。

![](https://cdn.nlark.com/yuque/0/2024/png/21778712/1725342642572-480fb4f4-76e7-49a1-a702-4805da4482ff.png)

2. **<font style="color:rgb(15, 17, 21);">若系统提示“无影像”，请按以下步骤排查：</font>**
+ <font style="color:rgb(15, 17, 21);">首先，确认对应单据是否已成功上传影像。</font>
+ <font style="color:rgb(15, 17, 21);">若为“同步”或“匹配”模式，且已通过发票助手上传发票或附件但仍无影像：</font>
    1. <font style="color:rgb(15, 17, 21);">请在【影像映射维护】中查询是否存在对应的影像编号；</font>
    2. <font style="color:rgb(15, 17, 21);">如无影像编号，请联系星瀚共享服务支持人员处理；</font>
    3. <font style="color:rgb(15, 17, 21);">如存在影像编号，请前往影像系统的单据列表中进一步查看当前影像状态。</font>

**<font style="color:rgb(15, 17, 21);">3. 若页面显示存在明显错误：</font>**

+ <font style="color:rgb(15, 17, 21);">请优先参考本文件中的相关指引进行排查；</font>
+ <font style="color:rgb(15, 17, 21);">如问题仍无法解决，请通过KSM系统提交技术支持工单。</font>
