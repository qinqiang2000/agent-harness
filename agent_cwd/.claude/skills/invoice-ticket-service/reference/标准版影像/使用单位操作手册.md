_**<font style="color:#8A8F8D;">乐企联用</font>**__<font style="color:#8A8F8D;">是指企业向依托其信息系统开展交易活动而没有控制关系的其他企业，提供数电发票使用、税费申报等涉税服务。使用单位接入乐企，需直连单位邀请方可接入。</font>_

_<font style="color:#8A8F8D;">申请成为金蝶乐企联用的企业，可享受更稳定的开收票通道（目前仅支持开票），且有无需登录、无需短信验证码、无需实人认证、无需固定IP、性能高等特点；推荐符合底下条件的企业申请：</font>_

_<font style="color:#8A8F8D;">①不满足乐企自用的申请条件</font>_

_<font style="color:#8A8F8D;">②不愿意用RPA</font>_

## 接入条件确认
### 税局**<font style="color:rgb(51, 51, 51);">乐企联用的接入条件</font>**
<font style="color:rgb(51, 51, 51);">使用单位接入乐企平台应当同时符合以下基本条件：</font>

1. <font style="color:rgb(51, 51, 51);">使用单位依托直连单位自有信息系统开展交易或资金收付等活动；</font>
2. <font style="color:rgb(51, 51, 51);">使用单位的纳税信用级别为A级、B级或M级（不参与纳税信用级别评价的除外，如个体工商户、非独立核算分支机构等）；</font>
3. <font style="color:rgb(51, 51, 51);">使用单位被邀请接入时，未被税务机关认定为风险纳税人；</font>
4. <font style="color:rgb(51, 51, 51);">省级税务机关确定的其他条件。</font>

<font style="color:#DF2A3F;">税局原文可参考（此为湖南税局发文供参考，可搜索企业所在地区税局的发文）：</font>

[https://hunan.chinatax.gov.cn/show/20250117785342](https://hunan.chinatax.gov.cn/show/20250117785342)

<font style="color:#585A5A;background-color:#E8F7CF;">根据已发文地区整理出以上规则，理论上全国通用，请先自查是否符合上述标准。</font>

<font style="color:#585A5A;background-color:#E8F7CF;">若不确定是否符合：可以先提供企业名称、税号、企业所在省级税务机关（例如：广东省税务局）至发票云。</font><font style="color:#DF2A3F;background-color:#E8F7CF;">待发票云邀请后，再到电子税局查看能否进入乐企平台。</font><font style="color:#585A5A;background-color:#E8F7CF;">若提示”当前登录纳税人不在乐企白名单中，请联系主管税务机关进行乐企白名单维护“，建议咨询税局是否可以使用。若可以进入乐企平台，可按本文进行配置。</font>

### 使用此乐企联用的限制
<font style="color:#DF2A3F;">注意：企业接入乐企，有特殊票种开票需求的，请与发票云实施确认是否支持。</font>

### ERP环境要求
**1、星空企业版环境要求：**

+ **星空企业版：**7.6及以上版本 <font style="color:#8A8F8D;"> [星空企业版7.6版本开始支持增值税电子专用发票（全电发票）开具]</font>。
+ 配套所购金蝶云模块要求：应收模块+发票管理模块。
+ **金蝶云补丁：**建议打上金蝶云最新的补丁<font style="color:#8A8F8D;">（建议安装Cloud最新的补丁，金税对接的功能是持续完善的，并且Cloud的税收分类编码目前是在补丁中升级，所以建议经常打最新补丁，以便获得较好的使用体验）</font>。
+ 如果在星空的开票模块做过**二开**，请单独咨询实施是否支持。

**2、EAS环境要求：**

+ EAS版本：8.2版本、8.5版及以上<font style="color:#8A8F8D;">（建议购买86版本）</font>
+ 配套所购EAS模块要求：85版及以上<font style="color:#8A8F8D;">【EAS开票管理模块（税务模块）】，此外80、82版本需要购买最新版本税务+同等数量的82/80 bos许可</font>。
+ 如果在eas的开票模块做过二开，请单独咨询实施是否支持。

**3、星瀚环境要求：**

+ 私有化客户发私包处理，支持苍穹平台3.0及以上。私有化客户如需升级，请联系实施人员，提供星瀚部署私包。
+ 原本使用应收--开票单--发票管理中心、应收--开票单--开票申请单开具发票的客户，建议升级至应收--开票申请单的流程开具数电发票。

## 税局端操作内容
进入乐企平台后，可按下述说明进行配置。

### 登录乐企
（1）打开浏览器，输入所属省份电子税务局网址，打开电子税务局。

（2）输入用户相关信息及密码，点击【登录】，身份类型应为法定代表人、财务负责人或办税员。

（3）登入电子税务局后，选择【我要办税】，点击【乐企数字开放平台】进入，仅**单位法定代表人、财务负责人**有执行乐企接入的权限。

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1729490844527-b97bc914-d048-49d5-a7d9-e7f4457f415c.png)

### 使用单位接入确认
使用单位接收到乐企接入邀请后，可登录乐企在【接入管理】->【响应邀请】中确认邀请信息。

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1731048814840-3f7b3e77-bc39-4a97-a1c9-ac4f513b6115.png)

确认邀请时，<font style="color:rgb(0,0,0);">需完成开票人员的添加和开票安全验证时间的配置。建议将时间设置为2099年（</font><font style="color:#DF2A3F;">时间到期前需要及时续期，逾期将影响企业开用票业务，因此建议直接设置为99年</font><font style="color:rgb(0,0,0);">）。</font>

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1733385211428-6e0f1a39-6fc1-4a11-a1e8-9b26cfb2c763.png)

### <font style="color:rgb(51,51,51);">能力接受</font>
现按照2.2节接受邀请时，已同时接受能力。如后续需接受其他能力，可按照此步骤操作。

接入邀请完成后，直连单位发起能力邀请，使用单位可以在此界面接受。

<font style="color:rgb(0,0,0);">（1）【能力中心】-【能力受理管理】，稍等页面数据加载，展示出邀请信息即可。</font>

<font style="color:rgb(0,0,0);">点击对应纳税人操作列的【详情】，展示发起的授权邀请详情。</font>

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1731378283655-f1d3bafd-7daa-4261-95f0-386b3759a35d.png)

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1731378354099-8b7310a7-2eeb-4ae9-afc4-eb4c295cc9c1.png)

（2）<font style="color:rgb(0,0,0);">点击对应能力操作列的【同意】，同意能力授权邀请。</font>

### <font style="color:rgb(51,51,51);">开票员登记新增或解除</font>
现按照2.2节接受邀请时，已同时邀请开票员。如后续需调整开票员（新增或解除开票员），可按照此步骤操作。

（1）开票员新增或解除

法定代表人或财务负责人可在【接入管理】->【开票人员登记及管理】中校验本企业所有开票人员状态。同时可新增登记邀请或解除登记邀请。新增登记邀请时需选择开票员及绑定直连平台。

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1732259872782-0a6a1585-f75f-4e87-af39-93319efdd57f.png)

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1729576252753-0d095283-59f7-4297-96ad-48a2aab40708.png)

（2）企业开票员确认邀请<font style="color:#DF2A3F;">（此步骤已无需处理，乐企平台已取消）</font>

<font style="color:rgb(0, 0, 0);">使用单位法定代表人或财务负责人邀请开票人员后，开票员可在【接入管理】->【开票人员登记及管理】中确认邀请，确认邀请需完成刷脸实人认证。完成认证后，开票员与直连平台正式建立绑定关系。</font>

<font style="color:rgb(0, 0, 0);">注意：此处开票人员看不到乐企平台，无法登录乐企平台的，请确认开票人员在电子税局是否是办税员权限。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1752802354612-7d3c34a7-23a1-4277-b54b-2933ed5e8ce2.png)

### <font style="color:rgb(51,51,51);">设置开票安全验证时间</font>
现按照2.2节接受邀请时，已同时设置开票安全验证时间。如后续需再次设置，可按照此步骤操作。

（1）法定代表人或财务负责人可在【接入管理】->【响应邀请】中查看接入详情。

![](https://cdn.nlark.com/yuque/0/2025/png/647520/1764901513681-77dbc4ec-26f5-4ec4-87dd-7a24611383af.png)

（2）在接入详情页下拉，找到【开票安全验证设置】，点击【设置】配置开票安全验证时间。<font style="color:rgb(0,0,0);">建议将时间设置为2099年（</font><font style="color:#DF2A3F;">时间到期前需要及时续期，逾期将影响企业开用票业务，因此建议直接设置为99年</font><font style="color:rgb(0,0,0);">）。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/647520/1764901814032-40f5ca5b-53f9-406f-9a6f-96e1881d082e.png)

## 【AWS用户】发票云产品操作内容
