## 不动产基础资料功能设计理念概述
### 功能概述
开具不动产销售、不动产经营租赁、建筑服务特殊票种时，可以在此维护相关不动产基础信息，供开票时进行选择，可以减少开票时手工录入的工作量。

### 目标客户以及客户解决核心痛点问题
1. <font style="color:rgba(0, 0, 0, 0.85);">客户的内部操作使用人员：财务或相关开票岗位，解决了在开具不动产销售、租赁、建筑服务等特殊票种时，减少手工录入不动产基础信息的工作量的痛点，能够提高开票效率和准确性。</font>
2. <font style="color:rgba(0, 0, 0, 0.85);">客户部门管理人员：财务部门管理人员，解决了对不动产基础资料进行统一管理和分配的痛点，能够确保不动产信息的准确性和合规性，同时有效控制不动产的分配和使用情况。</font>
3. <font style="color:rgba(0, 0, 0, 0.85);">客户公司管理层：公司高层管理岗位，解决了对不动产基础资料管理进行监督和决策的痛点，能够确保不动产基础资料的管理符合公司的业务需求和税务要求，为公司的运营决策提供支持。</font>

### 不动产基础资料主要操作说明
##### 新增
**步骤1：**点击“新增”按钮，进入不动产信息编辑界面。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365362618-c8047f05-fe62-4bfa-94b7-0d32919be57d.png)

**步骤2：**由创建者来设定“编码”和“名称”。并据实填写不动产类型、地址等必填信息。

①若类型为“不动产销售”，根据需要完善产权证证号、增值税项目编号等选填信息。还可根据需要来完善不动产销售的信息，包括不动产单元代码、网签合同备案编号等。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365386616-ec802d9f-dd35-4c0e-ab27-c7b60e1c7707.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365404433-8c16b0f1-1dec-43ba-8c2f-413e77453fe7.png)

②若类型为“不动产租赁”，还需完善不动产权证号与面积单位。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365424482-44963620-f8c3-4cca-995f-36a5e4d6c611.png)

③若类型为“建筑服务”，还需准确填写土地增值税项目编号。并在第二栏填写建筑项目名称。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365447737-670501f3-88d7-4c8f-ba04-c3fc98600e43.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365614291-fcc54d33-302d-4759-870c-bab7766774d0.png)

##### 分配查询
①（按组织）

在列表中选中组织，点击查询按钮，即可在右侧窗口显示该组织管理的不动产。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365592342-42229461-8a51-4338-a5b8-a97acc0ff40d.png)

②未分配查询

在左侧选中组织，右边即出现其未被分配的不动产，还可以在此窗口为其分配，还可以为其执行取消分配的操作。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365579434-7a741df1-fa6e-46b0-9a02-5e91cc3f7919.png)

③（按数据）

在上方搜索框选择开票项，下方窗口即可查看该不动产为哪些组织所使用。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365563157-b263f289-18f8-4e21-9a33-5cecff87874e.png)
