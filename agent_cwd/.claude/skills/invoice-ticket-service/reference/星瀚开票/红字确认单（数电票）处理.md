    1. 对于红字确认单红冲收购票，选择蓝字发票新增红字确认单时，购销身份选择我是销方，生成的红字红字确认单购销身份为我是购方。②红字确认单-购销方信息栏目中的“购方企业类型”调整为“企业类型”，并增加帮助文本：非收购票，“企业类型”的值为购方企业类型；收购票，“企业类型”的值为销方企业类型。

<font style="color:rgb(0, 0, 0);">不同红冲原因的计算方式：</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748574367311-86125b22-e0ca-4503-b015-a3519a9f0edc.png)

<font style="color:rgb(0, 0, 0);">1）销售退回：可编辑数量</font>

+ <font style="color:rgb(0, 0, 0);">普通征税</font>
    - <font style="color:rgb(0, 0, 0);">数量*不含税单价=不含税金额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额（不四舍五入）*税率=税额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额（四舍五入）+税额（四舍五入）=含税金额</font>
+ <font style="color:rgb(0, 0, 0);">减按计征</font>
    - <font style="color:rgb(0, 0, 0);">减按计征发票不支持选销售退回。</font>

<font style="color:rgb(0, 0, 0);">2）销售折让：可编辑含税、金额不含税金额</font>

+ <font style="color:rgb(0, 0, 0);">普通征税-编辑不含税金额</font>
    - <font style="color:rgb(0, 0, 0);">税额=不含税金额（手动输入，两位小数）*税率</font>
    - <font style="color:rgb(0, 0, 0);">含税金额=不含税金额（四舍五入）+税额（四舍五入）</font>
+ <font style="color:rgb(0, 0, 0);">普通征税-编辑含税金额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额=含税金额（手动输入，两位小数）/（1+税率）</font>
    - <font style="color:rgb(0, 0, 0);">税额=含税金额（四舍五入）-不含税金额（四舍五入）</font>
+ <font style="color:rgb(0, 0, 0);">减按计征-税收分类编码不为3040502020102000000个人出租住房：</font>
    - <font style="color:rgb(0, 0, 0);">编辑含税金额：</font>
        * <font style="color:rgb(0, 0, 0);">税额=含税金额/（1+1.5%）*1.5%    </font>
        * <font style="color:rgb(0, 0, 0);">不含税金额=含税金额-税额</font>
    - <font style="color:rgb(0, 0, 0);">编辑不含税：</font>
        * <font style="color:rgb(0, 0, 0);">税额=不含税金额*1.5% </font>
+ <font style="color:rgb(0, 0, 0);">减按计征-优惠政策为按5%简易征收减按1.5%计征，税收分类编码为3040502020102000000个人出租住房：</font>
    - <font style="color:rgb(0, 0, 0);">编辑含税：</font>
        * <font style="color:rgb(0, 0, 0);">税额=含税金额/（1+5%）*1.5% </font>
    - <font style="color:rgb(0, 0, 0);">编辑不含税：</font>
        * <font style="color:rgb(0, 0, 0);">税额=不含税金额*1.5%/（1+3.5%）</font>

<font style="color:rgb(0, 0, 0);">3）服务中止：数量、含税金额、不含税金额</font>

+ <font style="color:rgb(0, 0, 0);">普通征税-编辑数量</font>
    - <font style="color:rgb(0, 0, 0);">数量*不含税单价=不含税金额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额（不四舍五入）*税率=税额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额（四舍五入）+税额（四舍五入）=含税金额</font>
+ <font style="color:rgb(0, 0, 0);">普通征税-修改不含税金额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额（手动输入，两位小数）*税率=税额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额（手动输入，两位小数）/不含税单价=数量</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额（四舍五入）+税额（四舍五入）=含税金额</font>
+ <font style="color:rgb(0, 0, 0);">普通征税-编辑含税金额</font>
    - <font style="color:rgb(0, 0, 0);">不含税金额=含税金额（手动输入，两位小数）/（1+税率）</font>
    - <font style="color:rgb(0, 0, 0);">税额=含税金额（四舍五入）-不含税金额（四舍五入）</font>
    - <font style="color:rgb(0, 0, 0);">数量=不含税金额（不四舍五入）/不含税单价</font>
+ <font style="color:rgb(0, 0, 0);">减按计征-编辑数量</font>
    - <font style="color:rgb(0, 0, 0);">数量*不含税单价=不含税金额</font>
    - <font style="color:rgb(0, 0, 0);">税额计算-当税收分类编码不为3040502020102000000个人出租住房：</font>
        * <font style="color:rgb(0, 0, 0);">税额=不含税金额（不四舍五入）*1.5%    </font>
        * <font style="color:rgb(0, 0, 0);">含税金额=不含税金额+税额</font>
    - <font style="color:rgb(0, 0, 0);">税额计算-当优惠政策为按5%简易征收减按1.5%计征，税收分类编码为3040502020102000000个人出租住房：</font>
        * <font style="color:rgb(0, 0, 0);">税额=不含税金额（不四舍五入）*1.5%/（1+3.5%）</font>
+ <font style="color:rgb(0, 0, 0);">减按计征-编辑不含税金额、税额：同销售折让</font>

#### 回退红字确认单
1. 回退条件：

①红字确认单回退条件：

+ 只有数据来源为“匹配结果推送”或“单据开票”的数据才可以回退。
+ 只有开票状态为“未开票”或者“开票失败”的数据才允许回退。
+ 只有录入状态为“未录入”或“录入失败”，或录入状态为“已录入”且确认状态为“作废”的数据才允许回退。

以上条件需同时满足才能执行回退或撤回。

②执行回退或撤回逻辑：

+ 录入状态为“未录入”：回退成功则删除确认单；
+ 录入状态为“录入失败”： RPA先根据流水号查询确认为录入失败，按查询的实际状态更新录入状态，再行判断，回退成功则删除确认单（乐企通道暂不支持查询确认判断，忽略此步骤）；
+ 录入状态为“已录入”且确认状态为“作废”：回退成功，不做删除，保留数据。

无论录入状态，回退成功均解除与上游申请单关联关系，反写申请单状态。

2. 操作方式：勾选需要回退的红字确认单记录，点击回退，支持批量回退。

![](https://cdn.nlark.com/yuque/0/2024/png/12881570/1729766841678-ec6ccc43-3265-4f7c-b7c4-63eedf991fca.png)

## 红字确认单（数电票）处理相关配置说明
1. <font style="color:#000000;">配置红字确认单未关联负数开票申请单则不执行回调、配置非业务系统推送</font><font style="color:rgb(0, 0, 0);">（手工新增、税局下载）</font><font style="color:#000000;">的红字确认单不执行回调。</font>
+ <font style="color:rgb(0,0,0);">配置项类型 : sim_invoice_callback_config</font>
+ <font style="color:rgb(0,0,0);">配置项key : red_not_callback</font>
+ <font style="color:rgb(0,0,0);">配置项值 : 1或2或其他</font>

<font style="color:rgb(0,0,0);">配置项值为1时：如果红字确认单未关联负数开票申请单，不查询蓝票所属开票申请单的业务系统，不执行按蓝字开票申请单进行回调。</font>

<font style="color:rgb(0,0,0);">配置项值为2时：业务系统推送的红字确认单，无论是否关联负数开票申请单都回调红票。非业务系统推送的（手工新增和税局下载）的红字确认单，不执行回调。</font>

<font style="color:rgb(0,0,0);">其他：全部回调。</font>
