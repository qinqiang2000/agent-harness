## 自动化设置操作路径及所属模块编码
操作路径：【发票服务】→【基础资料】→【开票参数设置】→【自动化设置】

所属模块编码：F17025

所属模块名称：开票参数设置

## 自动化设置功能设计理念概述
升级开具数电票后，销售方红冲发票存在录入红字确认单、同步、红冲多个操作，当企业数据量较大时，红冲操作复杂。

因此发票服务针对金税四期红冲流程设计了自动化处理规则，目前支持自动更新红字确认单状态、自动将符合状态的数据执行开票。

通过配置即可自动从税局获取红字确认单状态，当红字确认单处于可开票状态时，自动将确认单红冲，并自动回调上游系统。

### 自动化设置主要操作说明
#### 自动化设置配置
**步骤1：**新增自动化设置

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914488359-5e79127d-ac40-4b6a-9ae3-72e44e3a97d2.png)  
**步骤2：**配置基本信息

主要是设置名称

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1741752031172-595677e5-a331-4d20-adb8-8a0d68e914b5.png)

**步骤3：**自动化任务执行设置

##### **①红字确认单自动红冲任务配置**
指定手工新增、单据拆合等多个来源的红字确认单，当状态为无需确认、购销双方已确认时，自动执行红冲操作，红冲成功后，如有上游单据，将发票进行回调。<font style="color:#DF2A3F;">税局下载、手工新增来源的红字确认单，如需关联负数开票申请单后开票，请勿配置自动红冲。</font>配置步骤如下：

+ 选择任务类型为红字确认单-红字确认单自动红冲。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914551164-4687600c-02d1-4dd4-ac2f-818d669468ea.png)

+  设置红字确认单来源，如下图所示可以勾选来源。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1741752108716-26e4664b-11ca-4368-8275-ce7acaeb88f6.png)

+ 设置自动化任务详情：可以选择自动化执行录入日期为最近一个月、最近三个月或最近半年的任务。

##### **② 红字确认单状态自动更新配置**
在指定时间范围内，自动更新当前红字确认单状态。配置步骤如下：

+ 选择任务类型为红字确认单-红字确认单状态自动更新。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914572134-dcca564a-a28d-40e6-84be-b0d91f59293a.png)

+ 设置自动化任务详情：选择申请方（销方或购方），填写录入日期，填写对方纳税人识别号（多个税号用英文逗号“,”分隔，非必填。），系统将会自动根据这些条件去执行自动化任务。   

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1741752162462-3b6eb073-92b3-44fd-95aa-a18542b63d7e.png)

##### **③红字确认单自动确认配置**
当本企业作为购方或者作为销方，对方税号在配置的白名单内（本页面的确认企业信息配置），且红字确认单状态为待本企业确认时，红字确认单将自动确认。配置步骤如下：

+ 选择任务类型为红字确认单-红字确认单自动确认，设置红字确认单来源。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914586373-bb60bf35-5085-44db-b32b-336e8519a3cb.png)

+ 设置自动化任务详情：设置录入日期（有三种：最近24小时、最近48小时、最近72小时），设置可执行身份（销方或购方），选择是否开启“发票服务已配置的税号自动确认”（若开启，在发票服务已配置的所有税号都会根据配置执行确认单自动确认。若只需要对其中某个税号执行自动任务，请见下一点。）

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914597923-ee4f1b90-9458-4f7a-814f-9f5158f90895.png)

+ 确认企业信息配置：若没有打开“发票服务已配置的税号自动确认”，只需要对某个税号执行自动任务。可以点击“增行”，手动填写对方税号和对方名称，将其添加到列表中；或是点击”选择发票服务已配置税号“在已配置的企业中选择添加。（注意：若已经勾选了”发票服务已配置的税号自动确认“的开关，则点击”选择发票服务已配置税号“是无效的。）

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914611135-461ee8ff-f292-4d81-b64b-e0d317252d4b.png)

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914623251-31d60b3f-d092-410b-a734-f2ac219c1637.png)

##### **④ **<font style="color:rgb(33, 33, 33);">自动下载红字确认单</font>
自动下载设置范围内增量红字确认单。配置步骤如下：

+ 选择任务类型为红字确认单-自动下载红字确认单。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748914635638-9a90b7d6-4342-497d-9944-2e2c2f58d7c1.png)

+ 设置自动化任务详情：设置申请方（有两种：我是销方、我是购方），录入日期（最近一天、最近三天、最近七天），对方纳税人识别号（此项非必填多个税号用,分隔。），系统将会自动根据这些条件去执行自动化任务。  

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1741752003396-f4e6f130-f17b-49aa-b848-5a7afa992a83.png)

##### **⑤ **<font style="color:rgb(33, 33, 33);">自动下载发票</font>
可创建下载发票结构化数据的任务。如需使用乐企更新，请确认已开通乐企的归集能力。默认每月五号下载。

填写需要下载的发票内容，保存成功后，自动创建任务，可在开票管理→销项全票池→历史发票导入页面可查看任务。任务完成，发票数据下载完成后，保存至销项发票池。

1、开票日期：可选择上个月的数据。

2、发票种类：支持电子普通发票、电子专用发票、纸质普通发票、纸质专用发票、数电票（普通发票）、数电票（增值税专用发票），选择。

3、可以进入历史数据同步模块查看执行情况，操作路径：[https://jdpiaozone.yuque.com/nbklz3/tadboa/zsaxr3l9ggq2mbgc](https://jdpiaozone.yuque.com/nbklz3/tadboa/zsaxr3l9ggq2mbgc)。

![](https://cdn.nlark.com/yuque/0/2024/png/12881570/1726050426358-30fa178a-fc2a-4877-8d03-1666b452ec8d.png)

##### **⑥ **<font style="color:rgb(33, 33, 33);">自动更新发票用途状态</font>
可创建自动更新发票增值税用途、消费税用途、入账状态任务。如需使用乐企更新，请确认已开通乐企的归集能力。如需使用电子税局更新，执行本任务时该电子税局账号不支持开票，请勿配置同步过多发票。

1、开票日期（待查询用途状态发票的开票日期范围）：可选择当月、上月、最近两个月、最近三个月、最近半年的数据。

+ <font style="color:rgb(0, 0, 0);">当月：查询开票日期为当月的发票，不包含当天。例如当前时间为2025年6月5日，则下载2025年6月1日到2025年6月4日的发票。 </font>
+ <font style="color:rgb(0, 0, 0);">上月：查询开票日期为上月的发票，例如当前时间为2025年6月5日，则下载2025年5月1日到2025年5月31日的发票。 </font>
+ <font style="color:rgb(0, 0, 0);">最近两个月：查询开票日期为当前月份往前算两个月的发票，不包含当天。例如当前时间为2025年6月5日，则下载2025年5月1日到2025年6月4日的发票。 </font>
+ <font style="color:rgb(0, 0, 0);">最近三个月：查询开票日期为当前月份往前算三个月的发票，不包含当天。例如当前时间为2025年6月5日，则下载2025年4月1日到2025年6月4日的发票。 </font>
+ <font style="color:rgb(0, 0, 0);">最近六个月：查询开票日期为当前月份往前算六个月的发票，不包含当天。例如当前时间为2025年6月5日，则下载2025年1月1日到2025年6月4日的发票。</font>

2、发票种类：支持电子普通发票、电子专用发票、纸质普通发票、纸质专用发票、数电票（普通发票）、数电票（增值税专用发票）。

3、查询时间/频次（执行查询任务的时间、频次）：在选择的时间查询发票用途状态，在历史发票查询页生成记录。可选择：5号、10号、15号、20号、26号，默认5号，支持多选。

4、可以进入历史数据同步模块查看执行情况，操作路径：[https://jdpiaozone.yuque.com/nbklz3/tadboa/zsaxr3l9ggq2mbgc](https://jdpiaozone.yuque.com/nbklz3/tadboa/zsaxr3l9ggq2mbgc)

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1750817853432-48dd9c54-be8d-424c-b4a0-d024a0067bc3.png)

#### 自定义自动化任务执行频次、周期
以上6点自动处理配置对应调度任务编码均为：sim_simRedConfirmAutoTaskPlan_SKDP_S

若在此自动化功能中配置了多个任务，则任务按照以下顺序执行：红字确认单更新>红字确认单下载>红字确认单确认>红字确认单开票>下载发票>更新发票用途。

客户可自定义调度计划的频次，默认每天中午12点执行，可自行修改执行次数、执行周期。

路径：【系统服务云】-->【调度中心】-->【调度计划】-->【开票管理】选中”自动红字确认单定时任务计划“

![](https://cdn.nlark.com/yuque/0/2024/png/38481329/1722246541602-fff71b5e-070f-4552-973e-aa6b3d643ebe.png)

点击编码即可进入并进行执行周期配置。
