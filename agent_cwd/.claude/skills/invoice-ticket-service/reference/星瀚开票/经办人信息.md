## 经办人信息功能设计理念概述
### 功能描述
预配置常用的经办人信息供开票时进行选择，可以减少开票时手工录入的工作量，仅数电票适用。

### <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户的内部操作使用人员：财务或相关开票岗位，解决了在开票时重复录入经办人信息的痛点，通过预配置常用经办人信息，减少了手工录入的工作量，提高了开票效率。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户部门管理人员：财务部门管理人员，解决了对经办人信息进行统一管理和维护的痛点，能够确保经办人信息的准确性和完整性，同时有效控制经办人信息的使用情况。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">客户公司管理层：公司高层管理岗位，解决了对经办人信息管理进行监督和决策的痛点，能够确保经办人信息的管理符合公司的内部控制要求，为公司的财务管理提供支持。</font>

### 经办人信息主要操作说明
##### 新增/复制经办人信息
**步骤一：**点击“新增”按钮，填加新的经办人信息。或点击“复制”，在现有信息的基础上进行修改。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366150966-c0c43cd4-24c0-4269-9618-4eb3507604b4.png)

**步骤二**：完善必填的经办人编码与信息等其他信息后，完成保存。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366169541-72a422ea-7201-4dc9-aac3-474024af52f8.png)

##### 启用（禁用）配置
选中经办人，点击“启用（禁用）”按钮，即可开启（或禁止）该经办人信息的使用

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744366193194-9b9da674-df7d-4a3c-ae6e-d5a3ff3fe330.png)
