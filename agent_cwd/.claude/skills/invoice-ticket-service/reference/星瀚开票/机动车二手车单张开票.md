支持手动开具：

+ 机动车销售统一发票
+ 机动车销售统一发票（数电纸票）
+ 二手车销售统一发票（数电纸票）
+ 二手车销售统一发票（数电电票）
+ <font style="background-color:rgb(245, 245, 245);">机动车销售统一发票（数电电票）</font>

### <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
1. 机动车销售统一发票：
    - <font style="color:rgba(0, 0, 0, 0.85);">内部操作使用人员：财务或相关开票岗位，解决了手动开具机动车销售统一发票时信息录入繁琐的痛点，能够快速准确地填写发票信息，提高开票效率。</font>

    - <font style="color:rgba(0, 0, 0, 0.85);">公司管理层：公司高层管理岗位，解决了对机动车开票管理进行决策和把控的痛点，保证公司的税务合规和财务管理的有序进行。</font>

### 机动车/二手车单张开票主要操作说明
#### 、机动车销售统一发票单张开票
**步骤1：**填写基本信息：选择开票组织和发票种类，可以切换开具机动车发票、机动车销售统一发票（数电纸票）

、机动车销售统一发票（数电电票）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753091963218-de4056c0-a8a2-478c-a42a-affd9a2c13c6.png)

**步骤2：**填写买方信息：支持在”购买方名称“的下拉菜单中选择在基础资料维护的开票抬头。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092136334-7299bec3-7f34-44af-a2dc-59d01c1dac84.png)

**步骤3：**填写销售方信息：填写单位住址、电话、开户行、账号。支持在单位住址的下拉菜单中选择在基础资料维护的该企业的抬头地址信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748417189811-eb825f1e-fcde-4f0c-ab7e-7375977b1646.png)

**步骤4：**填写车辆信息：可以手工完善车辆基本信息，或点击车辆信息信息按钮查询（机动车销售统一发票（数电纸票）、机动车销售统一发票（数电电票））。当税局返回的数据为空，但是当前页面已填写值时，不使用税局信息覆盖原信息，税局返回数据不为空，覆盖当前数据。

其中车辆类型可以选择预先在基础资料-车辆信息管理维护的信息，厂牌型号、产地、生产企业名称、税率、税收合并编码、是否享受优惠政策、优惠政策类型将自动同步带出。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092291878-bac9921f-cd8b-41cd-a757-f04bd526b0ca.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092502515-a1c06e35-55b2-41b0-a78a-884121daae17.png)

**步骤5：**填写价税合计（小写），可手动调整税率，其他价税信息将有系统自动计算出。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092391138-661e5d37-6890-4b31-8195-3e07bed0d39e.png)

**步骤6：**点击”开票“按钮，即可完成该机动车发票的开具。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092691301-977861b0-f5bb-4845-ad76-62a9de84f0d8.png)

<font style="color:rgb(33, 33, 33);">若为二手车经销企业-正向开票：可以在买方信息-购买方名称的下拉菜单中选择在基础资料维护的开票抬头；在销售方信息-卖方单位/个人住址的下拉菜单中选择在基础资料维护的该企业的抬头地址信息。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092821801-6338ac62-c402-4dca-96e1-df742c31d22a.png)

<font style="color:rgb(33, 33, 33);">若为二手车经销企业-反向开票：在买方信息-买方单位/个人住址的下拉菜单中选择在基础资料维护的该企业的抬头地址信息；可以在销售方信息-卖方单位/个人的下拉菜单中选择在基础资料维护的开票抬头（列表仅展示性质为个人的抬头数据）。</font>

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092852710-5997e781-fc5a-4e7d-b186-91371a627644.png)

<font style="color:rgb(33, 33, 33);">若为二手车市场-正向开票：可以在买方信息-购买方名称的下拉菜单中选择在基础资料维护的开票抬头；但销售方信息-卖方地址等信息仅支持手工输入。</font>

**步骤3：**填写开票方信息：支持在单位地址的下拉菜单中选择在基础资料维护的该企业的抬头地址信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092875696-fd96249a-c23b-48c2-a205-5d3efecce607.png)

**步骤4：**填写车辆信息：可以手工完善车辆基本信息。

其中车辆类型可以选择预先在基础资料-车辆信息管理维护的信息，厂牌型号、税率、税收合并编码、是否享受优惠政策、优惠政策类型将自动同步带出。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092903055-63be79d6-94ad-44ac-ab04-5e1ec4944046.png)

**步骤5：**填写价税合计（小写），可手动调整税率，其他价税信息将有系统自动计算出。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1753092921324-5dcc0ac3-2986-4968-a9cc-207f66da7f7f.png)

**步骤6：**点击”开票“按钮，即可完成该二手车发票的开具。
