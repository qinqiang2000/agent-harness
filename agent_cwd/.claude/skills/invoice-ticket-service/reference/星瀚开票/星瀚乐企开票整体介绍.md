# <font style="color:rgb(38, 38, 38);">产品概述</font>
## 产品介绍
星瀚直连税局乐企平台，帮助星瀚开票客户（包括公有云、私有云客户）可直连乐企开票。现有星瀚发票云开票场景，全部无缝衔接乐企通道。

1、星瀚发票云无缝链接乐企，严格按税局规范、高度匹配乐企开票要求。

2、提供可视化沙箱环境，快速接入乐企平台。

3、发票云全方位、多票种覆盖企业使用乐企开票诉求。

## 业务场景
使用星瀚开票的客户，直连乐企完成发票开具，星瀚开票所有业务场景皆可兼容直连乐企开票。

# <font style="color:rgb(38, 38, 38);">亮点特性</font>
1、整合乐企零散接口，提供开票一体化服务，帮助企业快速接入

+ 乐企对外接口数量多，仅开票能力就有16个，而且乐企只提供基础开票&用票能力接口，如开票能力含企业风险信息查询、可用税率查询、授信额度校验、预发票处理、发票上传等，仅仅一个普通开票就要调用多个API开票时，需要组合调用多个接口校验后，才能顺利开具发票，企业自行梳理需花费大量时间。
+ 集成信用额度管理、自动赋码、税收分类编码分配等逻辑前置处理，并将申请授信额度、发票号段申请内置为系统自动动作，使企业可专注业/财务处理
+ 提供乐企数据、配置相关的可视化页面，方便企业查看、操作



2、无缝衔接星瀚业务场景

+ 星瀚发票云已对接的财务云、项目云、资管云等上游系统，无需改造，即可接入使用乐企直连开票。
+ 现有强大的开票引擎，包括多单多明细合并、开票项匹配/映射、发票备注、购方抬头等开票元素完全适配乐企开票，并已按照乐企要求精细化改造
+ 星瀚发票云开票客户可直接切换乐企开票，无需额外改造



3、补足乐企缺乏的业财系统的关键功能，如版式文件生成

+ 乐企不提供版式文件生成服务，但是在实际业财税一体的流程中，发票预览是非常重要的环节，并且财政部明确规定，数电票归档必须使用XML文件，综合多方诉求，发票云严格按照税局规范，自主研发PDF、OFD、XML版式文件的生成服务。
+ 金蝶发票云从业、票、财、税多维度快速、专业化集成乐企平台，并生成进销项台账、版式文件及分析报表。

1、乐企：是指国家税务总局向符合条件的企业，通过税务系统与企业自有信息系统直连的方式，提供规则开放、标准统一的全面数字化的电子发票等涉税服务（以下简称“乐企服务”）的平台。
