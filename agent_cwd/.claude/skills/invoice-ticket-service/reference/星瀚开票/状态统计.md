## 状态统计功能设计理念概述
1. **功能概述**

发票统计报表提供多种不同维度的发票数据统计，为主要服务于企业内部运营、管理团队，为企业内部了解已开票数据、分析业务趋势、监测开票状态提供有力依据。

主要提供商品、购方、税率、状态、资料、对账、业务类型等统计维度

状态统计：发票的状态有：正常、作废、红冲；发票类型有蓝票跟红票，结合发票类型和发票状态双重维度统计。

2. **<font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>**

<font style="color:rgba(0, 0, 0, 0.85);">①客户的内部操作使用人员：财务或相关统计岗位，解决了查询和统计发票状态的痛点，能够快速获取不同发票状态下的开票金额、税额等信息，便于掌握发票的使用情况。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">②客户部门管理人员：财务部门管理人员，解决了对发票状态进行监控和管理的痛点，能够通过状态统计报表及时发现发票的作废、红冲等异常情况，确保发票的准确性和合规性。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">③客户公司管理层：公司高层管理岗位，解决了对公司发票状态进行整体把握和决策的痛点，能够通过状态统计报表了解公司的发票管理情况，及时发现问题，制定合理的发票管理制度和策略。</font>

### 状态统计主要操作说明
#### **状态统计查询**
**步骤1：**选择要查询的组织和发票类型，设置查询日期。在已经选择的组织中，可以通过填写销方名称和组织税号继续添加过滤条件。点击“查询”按钮，即可看到该条件下的状态统计详情。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1750822026447-6f126a4d-442e-4593-863f-7441c3742716.png)

**步骤2：**点击发票数量的数字，可以穿透到发票查询页面。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1750822037770-b95f3b52-cc6b-4138-af5c-672134912adf.png)

**步骤3：**点击“引出”按钮，将数据已EXCEL表的形式引出。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1750822054952-fdfff541-f5e3-434c-847d-3b52d9dd1adf.png)
