## 编码管理操作路径
【发票云】-【开票管理】-【海外发票】-【公共数据】-【编码管理】

## 编码管理功能设计理念概述
### 功能定位
【编码管理】用于集中维护海外发票相关的各类“基础资料”及其与外部系统基础资料之间的映射关系。  
支持对标准条目进行新增、编辑、导入/导出，以及维护“外部代码 → 标准代码”的映射。  
主要使用角色包括：实施顾问、主数据管理员、财务/税务人员等。  

### 设计理念与使用价值
+ **统一标准编码**：通过 UOM / Product Codes / Company Codes 等多种编码类型标签，实现不同业务对象编码的统一维护，避免各系统、各流程重复维护。
+ **标准条目 + 映射关系双层结构**：标准条目维护平台内部统一编码，映射关系维护外部系统编码与标准编码的对应，支撑对接 ERP、电商平台等外部系统。
+ **数据质量可视化**：通过“映射覆盖率、完成度、待修复项”等指标，辅助评估当前编码映射的完整性和质量。
+ **支持多范围/多国家场景**：通过“范围”字段，区分不同国家/地区或业务范围下的编码定义。
+ **支持智能匹配**：在映射关系维护中，如未指定标准代码，可由 AI 自动匹配最合适的标准代码，减少人工查找工作量。

### 编码管理主要操作说明
#### 按条件查询标准条目
**操作目标**：根据代码、名称或国家/地区，快速定位目标计量单位标准条目。

**操作步骤：**

1. 确认顶部标签选择正确，下方页签选择 **标准条目**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467163080-994824d9-c937-4ac8-8ad9-4d20bb64cf80.png)

2. 在“搜索代码或名称”输入框中输入关键字（可留空）。
3. 如需按区域筛选，在“选择国家/地区”下拉中选择相应国家/地区或范围。
4. 点击 **【查询】** 按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467177027-0bd96d0f-7fbe-42cb-a4ec-d09e4489543f.png)

5. 系统根据条件刷新下方数据列表。若查询无结果，列表区域将为空，可调整筛选条件重新查询。

#### 新增 / 编辑标准条目
**操作目标**：维护新的基础资料，或调整现有编码的名称、描述等信息。

**新增标准条目：**

1. 在 **标准条目** 页签中，点击工具栏上的 **【添加】** 按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467219990-fff09f92-df31-465d-b57f-cb37ac03ae25.png)

2. 系统右侧弹出“编辑标准条目”（或“新增标准条目”）侧滑弹窗。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467235272-488fa1d7-2657-4871-8639-1e9b9c745fe8.png)

3. 在 **「范围」** 字段中选择适用范围/国家，如 `SG - Singapore`、`DEFAULT` 等。
4. 在 **「代码」** 字段中输入标准编码（必填）。
5. 在 **「名称」** 字段中输入业务名称，例如 `group`。
6. 在 **「描述」** 字段中补充说明（可选，支持多行文本，最多约 200 字符，以页面显示为准）。
7. 确认信息无误后，点击弹窗底部 **【保存】**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467258090-2738b642-9ebd-4abc-96d8-afa7817faf8a.png)

8. 系统保存成功后关闭弹窗，并在列表中新增一条记录。

**编辑标准条目：**

1. 在标准条目列表中找到目标记录。
2. 点击对应行“操作”列中的 **编辑图标**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467275135-75a98963-6707-494e-8887-b4b0e7eb1e94.png)

3. 在弹出的“编辑标准条目”弹窗中，按需调整 **范围 / 代码 / 名称 / 描述**。
4. 点击 **【保存】** 提交修改。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467294752-c8b7bdcc-d5a3-490d-ba81-8ad5e46d910e.png)

5. 系统保存成功后更新列表内容。

#### 导入 / 导出标准条目
**操作目标**：通过文件方式批量导入或导出标准编码。

**导入标准条目：**

1. 在 标准条目页签中，点击工具栏 **【导入】**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467341115-c1c2ee81-995a-4e37-a2b9-4fa449f8d7cf.png)

2. 系统打开导入弹窗，按页面提示选择或下载模板。
3. 按模板填写标准条目数据并上传文件。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467366522-333895d8-df22-451e-94f5-727e1de88e5a.png)

4. 提交导入，等待系统处理并查看结果。

**导出标准条目：**

1. 可先按“搜索代码或名称”“选择国家/地区”等条件过滤需要导出的数据。
2. 点击工具栏 **【导出】**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467381738-1f6554d9-b9e5-4bb9-88ba-97af79d7207c.png)

3. 系统根据当前筛选条件生成导出文件，供用户下载保存。

#### 维护映射关系（新增外部代码映射）
**操作目标**：将外部系统的基础资料与平台内部标准基础资料建立映射关系。

**步骤：**

1. 点击列表上方的 **【映射关系】** 页签。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467434147-a5dc1a0d-8618-46de-a706-8fbf4ac42e4f.png)

2. 点击工具栏 **【添加】**，右侧弹出“添加映射关系”侧滑弹窗。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467451201-5ceae7af-d9c1-44e5-8d15-3929f08838e9.png)

3. 在 **「范围」** 字段中选择适用的范围/国家。该字段同时用于过滤可选的标准代码列表.
4. 在 **「外部代码」** 中输入外部系统提供的编码（必填，最多 64 字符）。
5. 在 **「外部名称」** 中输入外部编码名称（可选，最多 100 字符）。
6. 在 **「描述」** 中补充映射说明（可选，最多 255 字符）。
7. 在 **「标准代码」** 中，可选择一个已存在的标准代码；如暂不选择，可留空。
8. 注意阅读提示：“如果未选择标准代码，AI 将自动匹配最佳的标准代码。” 选择是否需要系统自动匹配。
9. 填写完成后，点击 **【保存】**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467482369-d98add65-4e7a-4642-a9b5-1fc48d82af5c.png)

10. 系统保存映射关系并在“映射关系”列表中显示新记录，同时相关统计指标（映射覆盖率）。

#### 删除标准条目或映射关系（单条 / 批量）
**操作目标**：清理已失效或错误的标准条目及映射关系。

**单条删除：**

1. 在对应列表（标准条目或映射关系）中找到目标记录。
2. 点击“操作”列中的 **删除图标**。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467536581-c37f260d-d86e-4fba-b795-19ea4d6212b6.png)

3. 在弹出的确认提示框中确认删除，确认后，系统删除该记录并刷新列表。

**批量删除：**

1. 在列表左侧勾选需要删除的多条记录。
2. 工具栏上的 **【批量删除(0)】** 按钮中的数字会随勾选数量变化，例如勾选 3 条时显示“批量删除(3)”。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763467576108-b9b25102-4eb5-41d3-912c-0f25ace1b192.png)

3. 点击 **【批量删除】**，在确认弹窗中确认操作，确认后，系统删除该记录并刷新列表。
