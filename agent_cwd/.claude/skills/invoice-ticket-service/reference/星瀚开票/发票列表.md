## 发票列表操作路径
【发票云】-【开票管理】-【海外发票】-【销项发票】-【发票列表】

## 发票列表功能设计理念概述
### 功能定位
 发票列表（企业票池）用于集中查看和管理企业在【海外发票-销项发票】中的所有发票，包括不同开票状态（开票中、开票成功、开票失败）及发票类型（如 Commercial Invoice、Credit Note）。  
主要使用人群包括：财务/税务人员、应收会计（AR Accountant）等。  

### 设计理念与使用价值
+ **统一企业票池视图**：按发票维度集中展示价税合计、税额、开票状态等关键信息，避免在多个系统或国家实例中反复查询。
+ **状态卡片快速监控**：通过顶部状态概览卡片实时看到全部发票数量及成功/失败分布，支持一键切换状态筛选。
+ **一站式发票处理**：在列表中即可查看发票详情、下载文件、发送邮件交付、提交红冲等，减少跨页面操作。
+ **全流程追溯**：通过“数据变更历史”查看开票、交付等关键状态的变更记录，支撑审计与问题排查。
+ **支持多渠道交付**：结合邮箱交付等能力，根据企业配置将已开票发票发送给客户。

### 发票列表主要操作说明
#### 按条件查询发票
**操作目标**：根据时间范围或发票号码，快速定位目标发票。

**操作步骤**：

1. 在“创建时间”中选择开始日期与结束日期。
2. 如需按号码查询，在“输入发票号码搜索”输入完整或部分发票号。
3. 点击【查询】按钮。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465604907-92fe7526-0010-48cd-8233-409b421d3386.png)

**结果说明**：列表将根据输入条件刷新，只显示符合条件的发票记录；点击【重置】可恢复默认筛选。

#### 按开票状态查看票池情况
**操作目标**：快速查看开票成功/失败/待开票等不同状态下的发票数量和明细。

**操作步骤**：

1. 在页面顶部状态概览区域，点击对应卡片，如【开票成功】或【开票失败】。
2. 确认所选卡片高亮，列表自动切换显示对应状态的发票。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465619294-a525a44c-cc7e-4d1d-96c7-4501aff83c14.png)

**结果说明**：

+ 【开票成功】下可进行下载、交付、红冲等操作；
+ 【开票失败】下会显示“失败原因”，便于分析问题。

#### 查看发票详情并下载文件
**操作目标**：查看单张发票的明细信息并下载 PDF/XML 文件。

**操作步骤**：

1. 在发票列表中找到目标发票，点击行级“查看发票”图标。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465664117-8b73656a-8b7a-468f-a568-b0906def4dac.png)

2. 在弹出的预览窗口中，左侧检查发票 PDF 内容，右侧“发票信息”确认基础信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465723130-b80861f2-5912-4808-b7c7-19559d3014da.png)

3. 如需下载，在右侧底部点击【下载PDF文件】或【下载XML文件】至本地。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465739064-6924997a-8c39-4430-9ead-24a03c926f42.png)

#### 通过邮箱交付发票
**操作目标**：将已开票的发票通过邮箱发送给客户或内部收件人。

**操作步骤**：

1. 在发票列表中，定位需要发送的发票（通常为“开票成功”的发票）。
2. 点击该行中的“发送邮件”图标。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465803010-86066b28-7a26-42d4-bdd2-9ea10dd3ceda.png)

3. 在“邮箱交付发票”弹窗中，确认“发票号码、购方名称、发票金额、开票日期”等信息。
4. 在“收件人邮箱 ”输入框中输入邮箱地址，按回车添加；可重复操作以添加多个邮箱。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465834415-1186cb23-f12e-4902-bba7-1f289ccdc696.png)

5. 点击【发送邮件】即可推送邮件。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465887385-b3d93a9f-b1a6-4dd2-9af0-629a22968b54.png)

6. 点击【交付记录】即可查看邮件推送详情。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763465920406-63220367-67c1-468f-a243-c4ccf5e6b852.png)

**结果说明**：

+ 系统会根据配置将发票文件发送到所有收件人邮箱；
+ 实际发送结果可在“交付记录”Tab 中查看；
+ 若邮箱格式不合法或发送失败，系统会提示错误信息。

#### 查看发票数据变更历史
**操作目标**：追踪某张发票从创建到当前的状态流转和交付变更记录。

**操作步骤**：

1. 在发票列表中找到目标发票。
2. 点击该行中的“查看变更历史”图标。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763466029996-c5ae4f44-a738-49e7-89c8-ea1d82ea0704.png)

3. 在“数据变更历史”弹窗中：查看创建、状态变更、交付状态变更等所有记录。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763466082425-24f52b4c-4ce0-4dbf-b0c6-552153b1407a.png)

**结果说明**：可以看到每次变更的时间、变更类型、状态标签以及操作人信息，用于审计和问题追踪。

#### 提交发票红冲
**操作目标**：当发票开具有误时，对已开票发票发起红冲处理。

**操作步骤**：

1. 在发票列表中，定位需要红冲的“开票成功”发票。
2. 点击该行中的“提交红冲”图标。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763466120321-3a4b1574-4014-410e-bb86-b9acab13eadc.png)

3. 在“提交红冲”弹窗中，确认上方显示的 Invoice No、Supplier、Amount、Date 等信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763466214297-98d7c766-89e3-4f1e-975e-4bae879a8c34.png)

4. 在“*红冲理由”文本框中填写红冲原因（必填）。
5. 点击【确定】提交。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763466237567-ca12f441-6fab-45a7-a2c2-f1e9ffde6244.png)
