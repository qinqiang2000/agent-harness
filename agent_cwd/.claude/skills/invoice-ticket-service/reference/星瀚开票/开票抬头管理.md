## 开票抬头管理功能设计理念概述
### <font style="color:#000000;">功能概述</font>
<font style="color:#000000;">发票抬头管理模块，用于维护购方企业信息，在手动开票、导入单据、接口推单等开票场景可根据规则匹配到此模块维护的客户信息。  </font>

### <font style="color:#000000;"></font><font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
<font style="color:rgba(0, 0, 0, 0.85);">1、客户的内部操作使用人员：财务或相关开票岗位，解决了在开票时快速选择和匹配购方企业信息的痛点，减少了手工录入的工作量，提高了开票效率和准确性。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">2、客户部门管理人员：财务部门管理人员，解决了对开票抬头进行统一管理和分配的痛点，能够确保开票抬头信息的准确性和合规性，同时有效控制开票抬头的使用和分配情况。</font>  
<font style="color:rgba(0, 0, 0, 0.85);">3、客户公司管理层：公司高层管理岗位，解决了对开票抬头管理进行监督和决策的痛点，能够确保开票抬头管理符合公司的业务需求和税务要求，为公司的财务管理和决策提供支持</font><font style="color:rgba(0, 0, 0, 0.85);">。</font>

### 开票抬头管理主要操作说明
##### 新增开票抬头
**步骤一：**点击“新增”按钮，进入抬头信息编辑界面

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365672002-c65c1dcb-354d-4f8c-aaf5-c021edb12215.png)

**步骤二：**完善客户的基本信息，填写编码、购方名称等必填信息，根据需要填写税号、代码等选填信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365687508-eae91721-38a4-4f51-a809-bdf41b2df27f.png)

**步骤三：**点击“购方抬头信息”下方的加号继续完善，填写购方的地址及电话、开户银行及账号、联系手机（开票时可作为收票手机）、公司邮箱（收票时可作为收票邮箱）等信息。并设定优先级，进行开票业务时，相关人员会检索抬头，出现条件均符合的项目按照优先级高低进行匹配。“是否默认”，设为默认时系统将自动匹配该抬头。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365711984-f90ff58e-fee9-4693-90e0-85c22f8ab0f8.png)

此处匹配项面板可选字段为开票申请单的单据字段，因此此处的匹配功能适用于开票申请单开票，单张开票不适用。

当上游业务系统有部分开票抬头数据，需要在下游配置部分数据，可以根据匹配条件抓取配置好的开票抬头补全上游缺失的数据。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365957413-d62a177d-8526-47ec-9167-e9b8c103e6ee.png)

映射匹配设置详见开票其他信息设置:[开票其他信息设置 (yuque.com)](https://jdpiaozone.yuque.com/nbklz3/tadboa/ti9hl6nx4knqfhl2)

**步骤四：**可在下方继续添加添加客户信息。点击“增行”，将来自<font style="color:rgb(0, 0, 0);">系统服务云--基础资料中的客户信息</font>导入进来。在开票时，即可根据客户信息匹配带出该条开票抬头。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365734001-712201d2-8919-43e6-b5b6-420deb2a0f4a.png)



##### 分配
①分配开票抬头

选中要执行分配的发票抬头，点击“分配”按钮，再于左侧选中组织，为其分配可使用的发票抬头

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365759567-81ffbb21-d56c-4951-9e95-a688eb79220b.png)

②分配查询（按组织）

在列表中选中组织，点击查询按钮，即可在右侧窗口显示该组织使用的开票抬头

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365774951-15f68496-11bb-475c-9be4-0308749ecfae.png)

③未分配查询

在左侧选中组织，右边即出现其未被分配的开票项目，还可以在此窗口为其分配，还可以为其执行取消分配的操作

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365807297-96e080d2-9552-42a4-8004-908c58094249.png)

④分配查询（按数据）

在上方搜索框选择开票抬头，下方窗口即可查看该项目为哪些组织所使用

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1744365933697-13b5a216-b46a-4117-ba78-bbab531f516e.png)
