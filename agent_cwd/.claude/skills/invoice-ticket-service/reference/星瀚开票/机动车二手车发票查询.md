## 机动车/二手车发票查询功能设计理念概述
### 功能概述
已开具成功的机动车销售统一发票、二手车发票（数电发票），包含批量开具、单张开具。支持红冲、作废、打印机动车销售统一发票及二手车发票（数电发票）。

### <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">内部操作使用人员：财务或销售相关岗位，解决了快速查询机动车/二手车发票信息的痛点，能够准确掌握发票的开具情况、状态等信息，提高工作效率。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">部门管理人员：财务部门或销售部门管理人员，解决了对机动车/二手车发票进行管理和监控的痛点，能够及时发现发票异常情况，保证发票管理的合规性和准确性。</font>
+ <font style="color:rgba(0, 0, 0, 0.85);">公司管理层：公司高层管理岗位，解决了对公司机动车销售业务进行整体把握和决策的痛点，能够通过机动车/二手车发票查询数据，了解销售情况和财务状况，为公司的战略决策提供支持。</font>

### 机动车/二手车发票查询主要操作说明
#### 查看发票详情
进入发票查询页面后，点击“单据编号”，即可查看发票详情。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748498944136-c1da6c0e-b514-416e-a040-746054146a39.png)

#### 发票打印
**步骤1：**选中数据，点击”打印“按钮，选择打印机/电子税局账号即可打印。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748499940532-14c709ea-fd76-4e5e-8706-619f3b79e1fa.png)

#### 发票作废
**步骤1**：选中数据，点击”作废“按钮，打开弹窗展示待作废发票信息、填写作废原因后，点击“确认作废”即可作废。

注意只能作废当月发票，跨月发票请进行红冲操作。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748500302327-a8519807-6493-4b61-8471-c9ffd7f41ebb.png)

#### 发票红冲
**步骤1：**点击“红冲”按钮，打开批量红冲页面查看待冲蓝票信息。红冲按钮校验：仅允许红冲正常的发票，发票状态为红冲、作废的不允许红冲。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748500725967-43ba3ab1-4359-4ab9-903e-3516a829d309.png)

**步骤2**：选择电子税局账号及红冲原因，确认无误后点击“确定”，进入红字确认单页面。如当前蓝票未生成红字确认单，则按照原蓝票信息生成红字确认单详情；如当前蓝票已生成过红字确认单，打开之前生成的红字确认单。生成的红字确认单默认已审核。

电子税局账号：默认选中默认账号，允许手工修改。

红冲原因：机动车支持选择开票有误、销售折让、销货退回；二手车支持选择开票有误、销货退回。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748500863170-13b83742-91e9-4236-8cb3-b44836c8eab8.png)

**步骤3**：在红字确认单页面，确认无误后，点击“保存”、“提交”、“录入”将红字确认单录入至税局。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748502043761-4730ecd2-ab46-4797-a541-08fda3477ec7.png)

录入后，点击“红冲”开具红票。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748502274446-0a3db689-bc2d-4a69-b528-26b88957f136.png)

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748502297731-05f8e6e9-b91f-4760-9ff1-8bafa1ba50a6.png)

可以在“开票管理>发票逆向处理>红字确认单（数电票）”页面查看所有红字确认单信息。

#### 批量引入数据
**步骤1：**点击“引入数据”按钮，选中本次要对数据执行的操作，更新已有数据或添加新数据。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748586681334-6f1b4307-f9cc-4638-88e1-7d1172a0f689.png)

**步骤2：**点击“下载模板”，并按照格式填写。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748586698583-5d23ddf4-cd47-4156-9387-e73579492d87.png)

**步骤3：**点击上传填写好的模板。要求数据模板与单据实体匹配，且数据模板在第一个工作表。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748586715830-93dcc6ee-7590-4776-a6d7-7dac2dd95f78.png)

**步骤4：**点击“开始引入”按钮，将数据引入到机动车销售批量开票。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748586731501-0d796604-34e8-49dc-9e37-2d2c8ac63d7f.png)

#### 导出数据
点击“导出”即可将发票数据以EXCEL格式引出至本地。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1748499195592-aa09d3ef-5585-45e2-bf74-2ba388b1d812.png)

#### 下载发票
机动车/二手车发票查询页面，支持下载二手车与机动车的数电电票，且支持选择下载格式。

当发票种类为机动车数电电票、二手车数电电票时，支持下载，点击后选择发票格式后进行下载。当发票种类为税控发票、数电纸票点击下载，提示：“发票种类名称”不支持发票文件下载，仅机动车发票（数电电票）、二手车发票（数电电票）支持。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1755834214085-56b7e167-2d03-4d56-bfec-ff8d9d0b0e2a.png)

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1755834226486-d4ab9217-1cc0-4ff4-b8b9-be98f88257ac.png)
