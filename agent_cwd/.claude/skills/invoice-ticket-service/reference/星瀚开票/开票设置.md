## 开票设置操作路径
【发票云】-【开票管理】-【海外发票】-【系统设置】-【开票设置】

## 开票设置功能设计理念概述
### 功能定位
 【开票设置】用于集中配置企业在全球电子发票场景下的开票基础能力，包括：

+ 各类业务字段的**编号规则**；
+ PDF 的**版式模板**；
+ 用于推送邮件的**邮箱通道**和**邮件模板**；
+ 各国家/地区的**舍入规则方案**（舍舍入方式、小数位等）。

该功能主要面向实施顾问、系统管理员、财务/税务管理员等角色。

### 设计理念与使用价值
+ **统一管理多公司配置**：通过顶部「全部类型」「编号规则」「邮箱通道」「邮件模板」「PDF 模板」统计卡片，直观查看各类配置在所有公司的覆盖率，便于评估配置覆盖情况。
+ **拆分开票链路关键要素**：将「编号生成」「发送通道」「邮件内容」「PDF 版式」「舍入规则方案」拆分为可独立维护的配置项，降低变更影响范围。
+ **支持多方案复用**：「编号生成」「发送通道」「邮件内容」「PDF 版式」「舍入规则方案」通过分配功能可被多家企业复用，减少重复配置工作量。
+ **提供国家默认舍入方案**：在舍入规则中为不同国家/地区配置默认方案，支持差异化的舍入方式、小数位与容差，方便快速启用。

### 开票设置主要操作说明
#### 通过顶部卡片查看各配置项的整体分配情况
**操作目标**

快速了解当前租户下，各类开票配置（编号规则、邮箱通道、邮件模板、PDF 模板）在公司维度的**整体覆盖率/分配情况**，判断是否还有公司未完成配置。

**操作步骤**

1. 在页面上半部分查看 5 个汇总卡片（示例）：
    - **全部类型 (83%) 5/6 家公司**
    - **编号规则 (100%) 6/6 家公司**
    - **邮箱通道 (100%) 6/6 家公司**
    - **邮件模板 (100%) 6/6 家公司**
    - **PDF 模板 (83%) 5/6 家公司**

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763444508497-18815d28-6a7f-41f7-8c08-4058247417d2.png)

2. 根据卡片含义判断整体分配情况：
    - **编号规则 (100%) 6/6 家公司**：表示当前租户的 6 家公司中，都已经配置并使用了至少一条编号规则；
    - **PDF 模板 (83%) 5/6 家公司**：表示仍有 1 家公司尚未配置或未分配 PDF 模板，需要在【PDF 模板】TAB 中继续维护；
    - **邮箱通道 / 邮件模板** 卡片同理。
3. 若某一类卡片比例不足 100%，可点击对应 TAB（例如【PDF 模板】），结合列表中的【已应用】列和“企业分配”弹窗，进一步查看是**哪些公司尚未分配**该配置，并按“3.3.7：为规则/模板/通道分配企业”完成后续分配。
4. 完成配置或调整分配后，点击页面右侧【刷新】按钮，顶部卡片会同步更新为最新覆盖率。

#### 维护编号规则（以“发票号码”为例）
**操作目标**

为发票配置统一的自动编号规则，例如“日期 + 流水号 + 固定后缀”，确保不同公司、不同场景下的发票号格式规范一致。

**操作步骤**

1. 在页面中点击 TAB【编号规则】，进入编号规则列表。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437214237-81d84afb-beef-4926-a8be-712754283ef9.png)

2. 新增规则：点击右上角【新建配置】；  
或编辑现有规则：在目标行点击“铅笔图标”。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437227627-991e7931-6f1e-4503-805b-f1fc92daf356.png)

3. 在【编辑编号规则】侧边栏中：
    1. 填写【规则名称】，如“发票号码”；
    2. 选择【应用类型】，如“发票”；
    3. 根据需要打开【启用状态】开关。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437252480-9840261e-7efd-4f8c-a91f-ef49d137fc0a.png)

4. 在【规则字段】区域配置编号结构：
    1. 点击【+ 添加字段】，在弹窗中选择【字段类型】为“日期”，设置【日期格式】，必要时配置【分隔符】，点击【确定】；
    2. 再次点击【+ 添加字段】，选择【字段类型】为“流水号”，设置【流水号宽度】【起始值】【步长】【重置周期】，可设置【分隔符】；
    3. 如需固定后缀，再次【添加字段】，选择【字段类型】为“常量”，在【常量值】中输入如 `INV`，设置合适的【分隔符】。
    4. 具体字段说明可见3.1节

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437302805-88d625ee-e484-443c-8072-64e18acf328d.png)

5. 如需调整字段顺序，在字段行使用“上移/下移”图标调整。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437327863-81ee2b9a-a471-4c14-9f11-705f2e2a88c8.png)

6. 查看底部【完整预览】，确认示例编号格式符合预期。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437338585-1b7fdc32-dc62-4236-ba9e-48a697b01e7a.png)

7. 点击右上角【更新】（或【创建】）提交规则配置。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437371338-d0ade0f2-0b88-451f-a05f-917283ca000d.png)

8. 配置完成后，可通过“分配”弹窗将该配置分配给具体企业。详见3.3.7。

**结果说明**

+ 启用且已分配企业的编号规则，将在对应单据开立时参与编号生成。
+ 修改已有规则（尤其是“流水号起始值、重置周期”）可能影响后续编号连续性，需提前评估对业务及对接系统的影响。

#### 配置邮箱通道
**操作目标**

配置可用的邮箱通道，使系统能够通过 SMTP 等方式自动发送带有发票 PDF/XML 附件的推送邮件给客户。

**操作步骤**

1. 在【开票设置】页面点击 TAB【邮箱通道】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437460206-3da87679-281c-4c4e-a69a-832f0c2f926a.png)

2. 点击【新建配置】，或在现有方案行点击“铅笔图标”编辑。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437479106-93b420c5-def3-4ed2-b508-aa3257ab5dbd.png)

3. 在侧边栏中，填写：
    1. 【方案名称】：如 `xxxqqemail`；
    2. 【发件地址】：如 `400000002@qq.com`；
    3. 【发件显示名】：如 `xxxxxx`；
    4. 【回复地址】：如需要设置专门用于接收客户回复的邮箱；
    5. 【邮件通道】：选择 `SMTP`；
    6. 【SMTP 主机】：如 `sxxx.qq.com`；
    7. 【端口】：如 666；
    8. 【TLS 模式】：如 `STARTTLS`；
    9. 【用户名】：一般为发件邮箱地址；
    10. 【密码 / API Key】：填写邮箱授权码或接口密钥。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437572388-0fb3fe93-7be5-4892-a451-638ba9f97ea9.png)

4. 根据业务需要，开启或关闭：
    - 【附加 PDF】：是否在邮件中附加发票 PDF；
    - 【附加 XML】：是否附加发票 XML 文件。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437615898-3d6dca61-ce24-4a23-93a9-d8d635fec013.png)

5. 如需立即验证连通性，点击【保存并测试】；只需保存则点击【保存】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437646426-5080ff3d-a551-461b-b762-14a1245132b6.png)

6. 配置完成后，可通过“分配”弹窗将该配置分配给具体企业。详见3.3.7。

**结果说明**

+ 测试通过且通道已分配给某企业时，该企业在发票交付流程中可使用此通道推送邮件及附件。

#### 维护发票交付邮件模板
**操作目标**

维护交付邮件的主题与正文，通过变量占位符嵌入发票号、金额、开票日期等信息，配合附件中的 PDF/XML 完成对客户的发票交付。

**操作步骤**

1. 在【开票设置】页面点击 TAB【邮件模板】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437752368-bc03fb49-ee26-47c4-87d4-4f040cd7a867.png)

2. 新增模板：点击【新建邮件模板】；  
或编辑模板：在目标模板行点击“铅笔图标”。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437765223-942c9e18-84e9-4ff8-bdd8-0ca9318841d2.png)

3. 在【编辑邮件模板】侧边栏中：
    1. 填写【模板名称】。
    2. 在【邮件主题】中输入主题文本，可包含变量占位符。
    3. 在【HTML 内容】区域粘贴或编辑完整邮件 HTML 代码，包含正文、样式和变量占位符。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437859972-8b488a2b-07f4-4292-bc13-6405a94f442a.png)

4. 根据需要打开或关闭【启用状态】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437875664-f4dc6f53-7711-4305-8c04-02f94b36da3e.png)

5. 如需查看呈现效果，点击右上角【预览】。
6. 确认无误后点击【保存】。配置完成后，可通过“分配”弹窗将该配置分配给具体企业。详见3.3.7。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437912968-c3b2b870-7f8b-4454-a158-248e234177f7.png)

**结果说明**

+ 启用且被企业/流程引用的模板，在推送邮件时会与对应邮箱通道、PDF 模板组合使用，生成实际发送给客户的邮件。
+ 模板修改后会影响后续邮件内容，历史已发送邮件不受影响。

#### 维护 PDF 模板
**操作目标**

设计和维护用于生成发票 PDF 文件的版式模板，控制发票抬头、行项目、税额汇总等信息在 PDF 中的布局和展示，为打印输出及发票邮件附件提供统一格式。

**操作步骤**

1. 在【开票设置】页面点击 TAB【PDF模板】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437966380-942845a9-452b-4f8c-8817-bd25ae10130a.png)

2. 点击【新建PDF模板】，或在目标模板行点击“铅笔图标”进入 **Edit PDF Template** 页面。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763437979554-7aef7204-8231-41b4-85e0-7d796513a5e6.png)

3. 在顶部设置：
    1. 【Template Name】：输入模板名称，如“英语测试pdf模板”；
    2. 【Language】：选择模板展示语言，如 `English`；
    3. 打开【Enable Status】开关，表示模板可用。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438020436-c753b450-b733-48a3-86b2-bd92a1e28ce7.png)

4. 在左侧【Widgets】中，从 Business Components、Basic Components、Layout Components 中拖拽需要的组件到模板画布：
    - Business Components：如 **Invoice Header**、**Invoice Details**、**Invoice Table**、**Invoice Total**、**Tax Code** 等，用于快速搭建发票业务结构；
    - Basic Components：如 **Spacer**、**Heading Block**、**Text Block**、**Divider**，用于控制间距和标题样式；
    - Layout Components：如 **Single Column** 等，用于调整版面布局。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438063794-712e4ffc-c653-49cb-b3c0-20cb24420ddb.png)

5. 在画布中选择相应组件，配合右侧【Insert Variable】面板插入实际发票字段（如客户名称、发票号、金额等）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438117803-46d75b45-17b6-4bca-8501-b4b8d5fb0a6d.png)

6. 反复调整布局、样式及变量，直至满足业务和合规要求。
7. 点击右上角【Save】保存模板。如需将模板 HTML 导出做备份或二次开发，可点击【Export HTML】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438130489-7a477117-2f8b-4014-a11d-e64bcc3426e7.png)

8. 配置完成后，可通过“分配”弹窗将该配置分配给具体企业。详见3.3.7。

**结果说明**

+ 启用并分配给企业的 PDF 模板，将用于生成该企业发票的 PDF 文件，供预览、下载和作为发票交付邮件的附件。
+ 调整模板可能影响客户看到的发票版式，应经过测试环境验证后再在生产环境启用。

#### 查看舍入规则默认方案
**操作目标**

查看系统为各国家/地区内置的舍入规则，了解舍入方式、小数位和容差等配置，为本地化团队确认金额处理逻辑提供参考。

**操作步骤**

1. 在【开票设置】页面点击 TAB【舍入规则】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438312732-a80430b0-0839-45ce-9b04-da68fb82fef4.png)

2. 在页面中确保子标签页【默认方案】处于选中状态。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438327100-ca50389b-d934-44c1-8322-613751eee3dd.png)

3. 在方案列表中找到目标方案，如 **France Default Rule**。
4. 在目标行“操作”列点击“眼睛图标”，打开【舍入方案详情】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438353913-94b2f9bb-79fb-4f0b-b780-e8c394e76f04.png)

5. 在详情弹窗中查看：
    - 【使用国家/地区】、【方案说明】；
    - 【舍入层级】、【舍入步长】、【舍入方式】、【小数位保留数】；
    - 【容差】以及【状态】【类型】【更新时间】等系统信息。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763438415274-43235d2f-795a-4529-91f7-2ed51ee4f94f.png)

6. 查看完毕后，点击右上角关闭图标退出。

**结果说明**

+ 默认方案通常为系统预置，不能直接修改或删除；如需按国家调整舍入规则，可在【自定义方案】中维护并在业务流程中指定使用。

#### 为各配置项分配企业
**操作目标**

控制哪些企业使用某一编号规则、邮箱通道、邮件模板或 PDF 模板。

**操作步骤**

1. 在【开票设置】页面，根据需要选择 TAB：
    - 【编号规则】 / 【邮箱通道】 / 【邮件模板】 / 【PDF模板】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763444335593-a5550e33-b44d-4540-b798-48380f1cadf5.png)

2. 在列表中找到目标记录，在其“操作”列点击“向右箭头图标”，打开“企业分配”弹窗。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763444353733-d4c9af1b-2af2-4ca4-95ac-6ef7b53077b2.png)

3. 在左侧【待分配企业】列表中：
    1. 如需要筛选，输入【搜索企业名称】关键字并点击【查询】；
    2. 勾选单个企业或使用【全选当前页】/【全选所有】批量勾选。
4. 点击中间【分配】按钮，将选中企业移动到右侧【已分配企业】列表。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763444406052-84cf3a60-432c-4e42-8d67-866003458eed.png)

5. 如需取消某企业的使用，在右侧勾选后点击【移除】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763444425741-f4bb9cc5-3f37-4467-88f3-508deb53ee13.png)

6. 调整完成后点击右下角【保存】。如放弃本次调整，点击【取消】关闭弹窗。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1763444436344-61382ba4-29df-489c-845c-ef9580f45057.png)

**结果说明**

+ 保存后，已分配列表中的企业会在相关业务流程中使用该规则/模板/通道。
