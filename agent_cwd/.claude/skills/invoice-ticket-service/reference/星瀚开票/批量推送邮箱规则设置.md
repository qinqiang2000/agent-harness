## 批量推送邮箱规则设置功能设计理念概述
### 功能定位
**发票批量推送方案设置**用于配置系统在开票后通过邮件批量推送发票的规则。  
通过对方案编码、优先级、邮件推送方式、分组规则以及匹配条件的统一配置，实现发票邮件自动/定时批量推送的集中管理。

适用角色主要包括：实施顾问、系统管理员、财务/税务管理员等需要维护发票邮件推送策略的用户。

### 设计理念与使用价值
+ **灵活控制推送时机与分组方式**：通过“发票邮件推送方式”“发票邮件推送批量分组规则”，支持自动推送或定时推送，并可按邮箱、购方、单据编号等维度分组发送。
+ **支持条件匹配**：通过“匹配项面板”定义字段、条件、值及逻辑，用于筛选需要应用当前推送方案的发票范围。
+ **优先级控制冲突方案**：当存在多条推送方案时，可通过“优先级”控制生效顺序。

### 批量推送邮箱规则设置主要操作说明
#### 新增或编辑发票批量推送方案
**操作目标**：为业务组织新增一条发票邮件批量推送方案，并配置推送方式和匹配条件。

**操作步骤**：

1. 在列表页点击【新增】按钮，打开配置详情页。或在列表点击蓝色编码，打开配置详情页。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1764818433593-14dee0f3-f27a-4c87-be65-2829cf001b25.png)

2. 在“基本信息”区域：
    1. **编码***：按企业编码规范输入方案编码，如“001”。
    2. **名称***：输入方案名称，如“推送方案01”。
    3. 如需区分优先级，在**优先级**中选择数值（0为优先级最高，9为优先级最低）。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1764817720345-c6eae896-22c0-48be-b535-7b45664a3b97.png)

3. 在“批量推送设置”区域：
    1. **发票邮件推送方式**支持选择以下推送方式：
        * 选择“开票后自动推送”：发票开具完成后系统自动发送邮件；

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1764817743624-52aab3f5-7c01-45b7-94b2-95c43db7cef5.png)

        * 选择“开票后定时推送”：系统根据调度计划定时批量发送邮件（调度计划编号为：<font style="color:#000000;">sim_invoicemailtiming_SKDJ_S</font>）。支持配置**发票邮件推送批量分组规则**，根据实际业务需求勾选“邮箱”“购方”“单据编号”（支持多选）。发送邮件时会按照配置取交集进行发送，即：只有在所有被勾选维度上取值都相同的发票才会被合并发送邮件。V8.0.1更新，若发票对应多个邮箱地址，系统会先将这些邮箱拆分为“单个邮箱”，然后以拆分后的每一个单独邮箱为单位进行分组。 

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1764817756127-ea1fb791-aac0-434e-b202-1b21c2e7ddba.png)

4. 在匹配项面板”区域：
    1. 点击【+ 添加条件】，新增匹配条件，配置购方名称条件下的匹配策略。
    2. 只有满足匹配条件的发票才会应用当前推送方案.

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1764818252199-aff0ced9-1bfe-4cbd-8c32-85ca6f07ce75.png)

5. 确认信息填写无误后，点击页面左上角【保存】，系统保存当前推送方案。

**结果说明**：

+ 保存成功后，新方案会出现在列表页，对应“使用状态”为可用。

#### 禁用/启用或删除发票批量推送方案
**操作目标**：当某推送方案不再使用时，将其推送方案禁用或删除。

**操作步骤**：

1. 进入【发票批量推送方案设置】列表页。
2. 在列表中勾选需要处理的方案记录。
+ 若仅需暂停使用，点击工具栏【禁用】；
+ 若确认不再使用且无需保留，点击【删除】。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1764823271504-ef8226e9-3f13-4121-8271-7dab800d858a.png)
