# 摘要
升级数电发票后，红冲规则发生较大变化，发票云针对数电票红冲开发了一系列功能，本文档主要说明：

1、税局针对数电发票红冲的基本规则。

2、阐述发票云支持的红冲流程，分别适配哪些红冲场景。

# 名词说明
1、电子税局：税局为所有企业开放的开收数电票的平台，发票云可通过RPA技术连接电子税局，完成开收票。

2、乐企平台：<font style="color:rgb(51, 51, 51);">是指国家税务总局向符合条件的企业，通过税务系统与企业自有信息系统直连的方式，提供规则开放、标准统一的全面数字化的电子发票等涉税服务（以下简称“乐企”）的平台。企业需单独申请资质。税局发文：</font>[https://shenzhen.chinatax.gov.cn/sztax/xxgk/tzgg/202305/99455b27c3744ac8ba20766caa9b2045.shtml](https://shenzhen.chinatax.gov.cn/sztax/xxgk/tzgg/202305/99455b27c3744ac8ba20766caa9b2045.shtml)

3、红字确认单：数电普票、数电专票在开具红字发票时，都必须申请红字确认单，详见 3.税局规则说明。

4、回调：调用发票云开票接口，开票完成后，发票云通过回调接口异步把开票数据返回调用方，回调接口由调用方按发票云的接口文档提供。

# 税局规则说明（必看）
1、红字确认单申请时，可选择开票有误、销货退回、服务终止、销售折让四种红冲原因，其中开票有误仅允许全额红冲，销货退回、服务终止、销售折让可部分红冲。

2、已进行销货退回/服务中止/销售折让开具红字发票的部分冲红的，允许更换申请方再次申请红字确认单，但申请原因只能选择销货退回/服务中止/销售折让。

3、数电票在购买方将蓝字发票勾选或入账后，销售方才可以发起部分红冲；当购买方未勾选未入账时，仅能由销售方发起全额红冲。

4、全额、部分红冲时，单价、数量、金额可修改信息如下，其中，单价严格控制不可修改：

一张原蓝票，第一次使用销售折让进行部分红冲，红票上的单价和数量为空，第二次使用其他红冲原因进行红冲时，红票的单价=剩余可红冲金额/剩余可红冲数量，即红票单价不等于原蓝票单价。

5、电子税局申请红字确认单时，不支持填写备注，开具红票时，不支持展示发票备注。税控发票使用数电票红冲时，同理。

6**、**⭐**数电票红冲时，部分红冲必须将红冲明细匹配原蓝票明细才可红冲，全额红冲默认按原蓝票所有明细红冲。**

7、电子税局、乐企申请红字确认单、开具红票时，不支持输入备注。RPA备注长度为200个字符（1个中文、1个英文均记为1个字符），乐企开票备注长度为450个字符。

8、**电子税局申请红字确认单后，确认单为无需确认、购方申请销方确认后、销方申请购方确认后，税局将自动开票，此时在红票仍仅保存在税局，需要在发票云对应开票页面点击开票，才可将发票同步至发票云**，例如：在负数开票申请单关联红字确认单，确认单状态为销方申请待购方确认，购方确认后税局自动开票，此时，仍需在发票云点击开票，发票才会从税局同步至发票云，并关联负数开票申请单。乐企平台申请红字确认单后，乐企平台不自动开票，在发票云触发后才进行开票。

9、**数电票可红冲税控发票，税盘缴销后，才支持使用数电票红冲税控发票，可先将税盘开的发票同步至发票云，再在发票云红字确认单页面，选择税盘开具的发票，录入红字确认单，即可进行红冲。税局红冲规则如下：**

各票种之间的红冲规则为“新冲旧、电冲纸”，具体如下：

（1）数电票可以对数电票、数电纸票（电子发票服务平台开具的纸票）、税控发票（前提为税控档案已注销）进行红冲；

（2）数电纸票可以对数电纸票、税控发票（前提为税控档案已注销）进行红冲，不允许对数电票进行红冲；

（3）税控发票仅允许对税控发票进行红冲，不允许对数电票、数电纸票进行红冲。

# 星瀚发票云总体规则说明（必看）
1、**通过数电票红冲发票时，原蓝票必须在发票云存在**，发票云进行校验、取数。将原蓝票导入发票云可采用以下方式之一：

（1）通过发票数据库迁移历史数据，建议原蓝票数据量巨大的企业采用该方式，如需由发票云迁移，将产生一定的实施费用。

（2）通过电子税局同步（仅能通过rpa同步、乐企只支持同步上线乐企之后开具的销项发票），建议票量50张/天以内的客户采用该方式，系统内直接操作即可。

（3）在电子税局全量导出Excel，在发票查询页导入原蓝票信息。

2、如确认单在发票云不存在，需先在下载至发票云。

3、**电子税局申请红字确认单后，确认单为无需确认、购方申请销方确认后、销方申请购方确认后，税局将自动开票，此时在红票仍仅保存在税局，需要在发票云对应开票页面点击开票，才可将发票同步至发票云**，例如：在负数开票申请单关联红字确认单，确认单状态为销方申请待购方确认，购方确认后税局自动开票，此时，仍需在发票云点击开票，发票才会从税局同步至发票云，并关联负数开票申请单。乐企平台申请红字确认单后，乐企平台不自动开票，在发票云触发后才进行开票。

###### 在发票云手动申请
**一、规则说明**

1、可支持部分红冲和全额红冲，需要部分红冲，在申请红字确认单时，填写需红冲的内容即可。

2、需关联至原开票申请单时，才支持红冲后回写开票申请单。

3、购方如果也在星瀚发票云，同样可以申请，申请后由销方确认，销方将确认单关联至开票申请单，即可开票。



**二、操作指南**

步骤一：进入红字确认单页面，点击新增，可以选择原蓝票，申请红字确认单。
