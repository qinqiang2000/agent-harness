## 数电配置功能设计理念概述
### 功能概述
使用RPA方式开具数电票；RPA方式登录数电平台进行发票下载、勾选/取消勾选等，均需进行数电配置。

**<font style="color:#DF2A3F;">具体星空升级数电初始化配置详见：</font>**[**https://www.yuque.com/piaozone/implement/ddua6cr8cm9qm1dr**](https://www.yuque.com/piaozone/implement/ddua6cr8cm9qm1dr)

**<font style="color:#DF2A3F;">本文重点围绕数电配置页面的功能展开介绍。</font>**

### <font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>
1. <font style="color:rgba(0, 0, 0, 0.85);">客户的内部操作使用人员：财务或相关开票岗位，解决了在使用RPA方式开具数电票以及进行发票下载、勾选等操作时，需要进行数电配置的痛点，能够提高开票和发票管理的效率。</font>
2. <font style="color:rgba(0, 0, 0, 0.85);">客户部门管理人员：财务部门管理人员，解决了对数电配置进行统一管理和监控的痛点，能够确保数电配置的准确性和合规性，同时有效控制账号的使用状态和权限。</font>
3. <font style="color:rgba(0, 0, 0, 0.85);">客户公司管理层：公司高层管理岗位，解决了对数电配置管理进行监督和决策的痛点，能够确保数电配置符合公司的业务需求和税务要求，为公司的财务管理提供支持。</font>

##### <font style="color:rgb(0,0,0);">添加或修改电子税局账号（含批量添加）</font>
###### 方式一：单个添加账号或修改
**步骤1：**在数电账号配置列表，点击蓝色的企业名称，进入数电配置详情页。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1761130209827-3651fce2-aa9a-43e4-9884-2e575a57af10.png)

**步骤2**：点击“新增账号”按钮，在弹窗界面添加电子税局账号；点击“修改账号”，维护列表已有的电子税局账号。

填写税号所属地、电子税局账号及密码、角色、登录类型、是否默认账号、该账号的默认业务、开票使用用户后保存即可。（字段含义，可见3.2节说明）

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1761131432621-81a890a1-b836-4ba0-986f-c181f4b9920e.png)

若登陆类型选择为“小号登录”，仅适用于“企享云办税小号”类型账号，具体填写要求如下：

电子税局账号：填写办税小号，企享云办理的手机号。

登录类型：企享云办税小号。

企享云appKey：由企享云提供（下单后，企享云会发送邮件给预留邮箱）。

企享云appSecret：由企享云提供（下单后，企享云会发送邮件给预留邮箱）。

![](https://cdn.nlark.com/yuque/0/2025/png/47254879/1753239566827-5df39f83-570c-4f78-a4b0-8e4699a4fcde.png)

**步骤3**：点击确定，完成账号维护。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1761130557780-04700846-07e9-45c0-981d-3f04342df97d.png)

###### 方式二：批量添加账号
更多按钮下拉有“导入数据”按钮和“查看导入结果”按钮。

使用“导入数据”按钮：下载导入模板，填写模板并导入后，即可添加电子税局账号或更新原有电子税局账号。

使用“查看导入结果”按钮：查看批量导入的结果。

![](https://cdn.nlark.com/yuque/0/2025/png/50437264/1751335997534-28dfd43a-76bf-4a7b-8b0b-902ee8851ee9.png?x-oss-process=image%2Fformat%2Cwebp)

<font style="color:#DF2A3F;">新增数电账号后，若需对帐号进行删除，需到公众号操作，详见</font>[<font style="color:#117CEE;">https://jdpiaozone.yuque.com/nbklz3/dn5ehb/gycekfybq6fzgxth</font>](https://jdpiaozone.yuque.com/nbklz3/dn5ehb/gycekfybq6fzgxth)

登录状态和公众号保持一致，统一以公众号为准。

1、星空PC端的账号的使用状态为启用时，若登录状态为未登录，认证状态显示为未认证；

2、星空PC端的账号的使用状态为禁用时，根据后端接口返回的值显示登录状态和认证状态，操作列为空（不显示操作按钮）

3、星空PC端的账号的电子税局权限没有开票权限或者没有收票+开票权限时，根据后端接口返回的值显示登录状态和认证状态，操作列为空（不显示操作按钮）。

4、当接口查不到账号的登录状态和认证状态时，PC端登录状态和认证状态才显示为空。

##### <font style="color:rgb(0,0,0);">修改所属租户</font>
用户可以使用此功能修改企业在星空中的所属租户信息。

通过列表的【clientid】字段，判断企业当前所属租户。

注意：修改租户时，系统会校验企业在权益平台是否归属于目标租户，若企业不在目标租户下，则不允许修改。星空中企业与租户的归属关系，需与权益平台保持一致。

**步骤1**：勾选需要调整租户的企业，点击”修改所属租户“，打开维护租户授权信息列表。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1761207490558-eb5b311a-e469-4ce8-9877-7db70b43ca9c.png)

**步骤2**：选中正确的租户，点击”保存“，即可将该企业归至新租户下。

![](https://cdn.nlark.com/yuque/0/2025/png/38481329/1761207588392-f2e73e70-fc89-4e7b-8fe4-d1e8bfb603e4.png)

##### 将税务云数电信息同步至发票服务
“电子税局账号同步方式”参数，支持将税务云的税局登录配置一键同步至发票服务生成数电配置。

![](https://cdn.nlark.com/yuque/0/2025/png/50437264/1750661355491-b4cbebeb-d535-46c6-bc6b-ebfba3d345d4.png?x-oss-process=image%2Fformat%2Cwebp)

该参数位于系统参数-发票服务-基础资料节点，控制税务云税局登录配置数据是否同步至星空数电配置。

1.当设置为“税局登录配置同步到数电配置”时，系统将通过后台微服务自动将税务云数据同步至星空数电配置。2.若发票服务对应税号还没有生成数电配置单据，会提示：同步失败，请先到“发票服务->基础资料->数电配置”菜单，点击“升级数电票”按钮生成对应税号的数电配置。

3.新增账号时电子税局权限默认给开票管理+收票管理，更新账号时税局账号权限以发票服务数电配置上的为准。

## 数电配置相关配置说明
1. 星空测试开收数电票流程：[https://www.yuque.com/piaozone/implement/ddua6cr8cm9qm1dr#ff9Bu](https://www.yuque.com/piaozone/implement/ddua6cr8cm9qm1dr#ff9Bu)

    - 配置项类型：elePaperInvoice
    - 配置项key：税号
    - 配置项值：0/1 （值为1的时候表示启用数电纸质发票，值为0的时候表示不启用）

    - 配置项类型：all_ele_mock_invoice
    - 配置项key：客户对应模拟开票的销方税号
    - 配置项值：不为空则启用，删除该配置则开真实票
    - （配置成功后见单张开票页面选择数电票种，授信额度为999999999.99则表示配置成功）
