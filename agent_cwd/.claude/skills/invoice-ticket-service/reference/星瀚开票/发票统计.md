## 发票统计功能设计理念概述
1. **功能概述**

此功能支持从多个维度对发票数据进行统计，例如统计不同票种的发票金额、税额、发票数量等。并支持查看发票统计明细。

2. ******<font style="color:rgba(0, 0, 0, 0.85);">目标客户以及客户解决核心痛点问题</font>**
    - <font style="color:rgba(0, 0, 0, 0.85);">内部操作使用人员：财务或相关税务岗位，解决了对发票数据进行统计和查询的痛点，能够通过该操作界面，快速选择统计维度和限制条件，查看发票的金额合计、不含税金额、票面税额等统计信息，提高了工作效率和数据准确性。</font>
    - <font style="color:rgba(0, 0, 0, 0.85);">部门管理人员：财务部门管理人员，解决了对发票数据进行分析和管理的痛点，能够通过发票统计功能，多维度地了解发票的情况，如不同票种的发票数量、金额等，为财务决策提供数据支持，保证公司的财务管理的准确性和合规性。</font>
    - <font style="color:rgba(0, 0, 0, 0.85);">公司管理层：公司高层管理岗位，解决了对公司财务状况进行整体把握和决策的痛点，能够通过发票统计数据，了解公司的发票收支情况、税务负担等，为公司的战略规划和决策提供重要依据，优化公司的财务资源配置。</font>

### 发票统计主要操作说明
#### 发票统计查询
**步骤1：**选择统计维度，并可以点击“更多”按钮，增加其他限制条件。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593335669-9d6de056-b301-476e-8df8-94025ebbf459.png)

**步骤2：**选择好统计维度和限制条件之后，还可将其作为方案进行保存。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593347312-9ebd47df-4700-4f3d-bbdb-a59b632e5ba8.png)

**步骤3：**点击票据分数的数字，可查看发票统计明细。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593360580-917b0015-55f0-4569-b0cc-203e8e3bbad8.png)

**步骤4：**点击“引出”按钮，可将统计信息导出到EXCEL表。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748593375087-5af5dc2a-bdf6-4753-9d24-6168246e5be8.png)
