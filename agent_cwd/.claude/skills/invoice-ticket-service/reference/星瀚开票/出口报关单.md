## 出口报关单功能设计理念概述
### 用户痛点
企业包含出口业务时，针对出口业务，税局要求开具出口发票，作为出口退税的凭据。但由于企业的出口报关数据在海关单一窗口，开票时需手动输入购方、开票商品、人民币金额等信息，操作复杂，容易出错。

### 解决问题的方案
直连海关拉取出口报关单数据，并自动根据人民银行的汇率中间价计算对应人民币金额，保存为可开票的出口报关单，并调用开票项基础资料自动补全税率、免税优惠政策信息，下推报关单开票开票。

###  报关单开票主要操作说明
##### 1、报关单采集
点击”报关单采集“，输入报关单申报时间段，勾选结关状态，<font style="color:rgb(55, 65, 81);">采集时默认采集已结关数据，也可采集全部（包含已结关、未结关等其他状态）状态报关单，由于已结关后数据变更较少，建议采集已结关数据作为开票数据源，</font>输入出口报关单号。

##### ![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748413904318-751d1a17-9821-4e6f-bace-8a36d4a396ab.png)  
2、采集查询
点击”采集查询“。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748413923241-62c916b4-cab0-474a-a71a-cd610d7218bc.png)

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748413943488-d9c206bf-7156-4add-adcc-641ae65143c1.png)

##### 3、下推报关单开票
选中要下推报关单开票的报关单，并点击”下推报关单开票“，点击“确定”即可下推。

![](https://cdn.nlark.com/yuque/0/2025/png/56650036/1748413966498-2b71d48d-b5ba-4976-8d7f-0ae00e485653.png)

V7.0.12更新：报关单开票时，部分特殊商品出口仍需要缴纳3%税率，需要按照出口金额计算不含金额、税额，因此优化推送公式。

| 单据体下推： |  |
| :--- | :--- |
| 源单（报关单） | 目标单（开票申请单） |
| 商品编号 | 商品编码 |
| 商品名称 | 商品名称 |
| 规格型号 | 按配置判断是否推送为规格型号 |
| 税收分类编码 | 根据开票项匹配 |
| 税率 | 根据开票项匹配 |
| 优惠政策 | 根据开票项匹配 |
| 开票单价（人民币） | 含税单价 |
| - | 不含税单价 = 含税单价 /（1 + 税率） |
| 根据配置判断，使用成交、法定第一、法定第二数量 | 数量 |
| 根据配置判断，使用成交、法定第一、法定第二单位 | 单位 |
| 开票总价（人民币） | 含税金额 |
| - | 不含税金额 = 含税金额 /（1 + 税率） |
| - | 税额 = 含税金额 - 不含税金额 |


出口报关单下推为报关单开票，天然复用原报关单开票所有的开票场景，兼容金三、金四多开票通道，报关单开票可作为统一管理所有来源的数据池。同时，通过报关单下推的数据，标记为报关单下推，方便追溯数据来源。
